# Notes

这个知识库现在只保留 6 个核心区域：

- `Inbox`：快速捕捉，先收进来再整理
- `Daily`：日记、碎片记录、当天进展
- `Learning`：长期学习内容，按 `CS / Backend / Python / Agent / AI / Finance` 分类
- `Projects`：正在推进或已经归档的项目
- `Resources`：不适合放进知识正文的参考资料和迁移记录
- `Templates`：常用模板

## 建议使用方式

- 不确定放哪时，先放 `Inbox`
- 能复用的学习内容放 `Learning`
- 有明确目标和阶段的内容放 `Projects`
- 外部资料、迁移清单、参考链接放 `Resources`
