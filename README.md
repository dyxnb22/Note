# Notes

这个知识库现在只保留 6 个核心区域：

- `Inbox`：快速捕捉，先收进来再整理
- `Daily`：日记、碎片记录、当天进展
- `Learning`：长期学习内容，按 `CS / Backend / Python / Agent / AI / Finance` 分类
- `Projects`：正在推进或已经归档的项目
- `Resources`：尚未沉淀为长期知识、但仍需引用或核对的外部资料
- `Templates`：常用模板

## 建议使用方式

- 不确定放哪时，先放 `Inbox`
- 能复用的学习内容放 `Learning`
- 有明确目标和阶段的内容放 `Projects`
- 尚待核对的外部资料和参考链接放 `Resources`；完成沉淀后删除临时审计与迁移记录
