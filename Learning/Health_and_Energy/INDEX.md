# 健康与精力索引

这条主线用于建立可持续的身体与状态底盘，不追求短期硬扛，也不替代医疗诊断。

## 学习路径

```text
观察整体状态
  → 优先修复睡眠与恢复
  → 建立活动和饮食底盘
  → 管理专注、压力与高峰期节奏
```

## 实践顺序

1. 连续记录一至两周：睡眠时长与规律、白天精力、活动、饮食节律和压力。
2. 找最上游且最可改变的一个变量，只做一个低摩擦调整。
3. 比较趋势而不是单日感受；无改善时重新检查假设。
4. 忙碌期使用最低可持续版本，不用一次性补偿制造新的负担。

## 安全边界

本目录提供一般健康教育。持续两周以上的严重或困扰症状、明显的睡眠或食欲变化、持续疼痛、无法完成日常任务，应向医生或合格心理健康专业人员求助。出现自伤、伤人或其他紧急风险时，应立即联系所在地紧急服务。
