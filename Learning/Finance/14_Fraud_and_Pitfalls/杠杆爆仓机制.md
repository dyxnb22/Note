# 杠杆爆仓机制

## 1. 主题定位

这篇讲杠杆如何放大风险、爆仓的触发机制，以及为什么即使"基本面正确"也可能被杠杆消灭。理解爆仓机制是风险管理中最重要的防御课题之一。

学它之前最好先懂：杠杆与保证金、波动率与最大回撤、风险预算。

学完后可以继续学：常见投资骗局（高杠杆是骗局的常见要素）。

---

## 2. 核心概念

**杠杆（Leverage）**
用借来的钱扩大投资规模。杠杆倍数 = 总投资额 / 自有资本。

```
2 倍杠杆：100 元本金 + 100 元借款 = 200 元投资
若资产涨 10%：收益 = 20 元（自有本金收益率 20%）
若资产跌 10%：亏损 = 20 元（自有本金亏损率 20%）
若资产跌 50%：亏损 = 100 元（本金归零，爆仓）
```

**强制平仓（Margin Call / Forced Liquidation）**
当账户净值低于维持保证金要求时，经纪商强制卖出持仓，无论市场时机好坏。

**爆仓（Blowup）**
自有资本完全亏尽甚至产生负债的极端情景。杠杆投资者可能在"正确的论点"上因无法承受短期波动而被强制平仓，最终亏损本金的全部甚至更多。

---

## 3. 核心机制

**爆仓的触发路径**

```
初始状态：本金 100 元，2 倍杠杆，持仓 200 元
    ↓
价格下跌 20%（持仓从 200 跌至 160 元）
    ↓
账户净值 = 160 - 100（借款）= 60 元（本金亏 40%）
    ↓
继续下跌 10%（持仓从 160 跌至 144 元）
    ↓
账户净值 = 144 - 100 = 44 元（本金亏 56%）
    ↓
若维持保证金要求为净值/持仓 > 30%，则 44/144 = 30.5%，临近警戒线
    ↓
再跌 1%（持仓 ≈ 142.5）：净值/持仓 = 42.5/142.5 = 29.8% < 30%
    ↓
触发强制平仓（无论此时是否是最低点）
```

**"死亡螺旋"的形成**

```
资产价格下跌
    ↓
强制平仓触发（需要卖出资产）
    ↓
更多抛售压力 → 价格进一步下跌
    ↓
其他使用同类杠杆的投资者触发强制平仓
    ↓
流动性危机：买方消失，卖压激增
    ↓
价格崩溃（可能远超基本面支撑的价格）
```

这是 2008 年金融危机、2015 年中国杠杆牛市崩溃、多次量化基金危机的共同底层机制。

**杠杆与最大回撤的关系**

```
无杠杆：最大回撤 30% → 需要涨 43% 才能回本
2 倍杠杆：资产下跌 50% → 账户本金归零（爆仓）
3 倍杠杆：资产下跌 33% → 账户本金归零（爆仓）
```

杠杆倍数越高，能承受的最大下跌幅度越小：

```
安全回撤上限 ≈ 1 / 杠杆倍数

无杠杆：100%（不会爆仓）
2 倍：50%
5 倍：20%
10 倍：10%
```

**流动性与杠杆的相互作用**

大多数爆仓发生在流动性危机中——正是在最需要冷静持有的时候，被迫在最低点卖出。高流动性资产（大盘股）相对容易平仓（损失"只是"亏损），低流动性资产（小盘股、私募资产）强制平仓可能导致额外的冲击成本，实际亏损比账面数字更大。

## 4. 分析框架

**杠杆风险评估框架**

在使用杠杆前，必须回答以下问题：

1. **最大回撤容忍度**：我能承受多大的回撤不被强制平仓？
2. **历史最大回撤**：这类资产的历史最大回撤是多少？
3. **安全杠杆倍数**：在历史最坏情景下，能承受的最高杠杆是多少？
4. **融资稳定性**：融资方是否会在我最需要持有时要求提前还款？
5. **流动性**：如果被迫平仓，能否在不影响价格的情况下卖出？

```
安全杠杆上限 = 账户净值回撤容忍度 / 资产预期最大回撤

例：能接受本金亏 50%，历史资产最大回撤 30%：
安全杠杆 = 50% / 30% ≈ 1.67 倍
```

---

## 5. 关键指标或关键变量

**净值杠杆比率**
= 总持仓市值 / 账户净值

实时监控，接近或超过融资约定的维持比率时需要及时补充资金或减仓。

**融资成本 vs. 预期收益**
```
净收益 = 投资收益率 × 总持仓 - 融资利率 × 借款额

如果投资收益率 < 融资利率，杠杆会从正向变为负向
```
在利率上升环境中，原来"划算"的杠杆策略可能因为融资成本上升而变得不经济。

**压力测试：最坏情景模拟**
在 -30%、-50% 的资产下跌情景下，账户净值是多少？是否触发强制平仓？保留多大的缓冲空间？

## 6. 实际例子

**例 1：中国 2015 年杠杆牛市崩溃**
2015 年上半年，大量散户通过配资（1:2 甚至 1:5 杠杆）参与股市。6 月上证综指从 5100 点开始下跌，强制平仓触发 → 抛售加剧下跌 → 更多强制平仓。政府出手救市前，许多 5 倍杠杆投资者在 20% 的跌幅中就已爆仓本金归零。

**例 2：LTCM 长期资本管理公司（1998 年）**
拥有多位诺贝尔奖得主的对冲基金，平均杠杆超过 25 倍。1998 年俄罗斯债务违约引发市场波动，原本相关性很低的资产同时大跌（相关性在危机中收敛为 1）。25 倍杠杆意味着资产下跌 4% 本金即归零，实际亏损超过 40 亿美元，美联储不得不协调银行团体救助以防系统性风险。

**例 3："正确"论点被杠杆消灭**
某投资者用 3 倍杠杆做多某只股票，基本面研究非常扎实，长期来看确实低估。但市场短期情绪极度悲观，股价在三个月内继续下跌 35%（即使基本面没有变化），触发强制平仓，本金全部亏损。18 个月后，该股票涨了 200%——但这位投资者已经出局。**杠杆可以让一个正确的判断变成亏损出局。**

## 7. 常见误区

**误区 1：我能判断市场，所以可以用高杠杆**
即使判断方向正确，短期的随机波动也可能触发强制平仓，让你在最低点出局。杠杆的风险在于**时机**，而非**方向**——你可能对，但不够早。

**误区 2：融资利率低，杠杆不贵**
融资利率是显性成本；强制平仓风险、流动性风险、情绪压力是隐性成本。即使融资利率 3%，被强制在最低点平仓的一次亏损，可能超过多年的利息节省。

**误区 3：历史最大回撤是杠杆上限的依据**
历史最大回撤是已知的过去，未来可能出现更大的回撤。审慎做法是用历史最大回撤的 1.5-2 倍作为杠杆计算的分母（留安全余量）。

**误区 4：只要不爆仓就是安全的**
高杠杆下即使没有爆仓，剧烈的净值波动会带来极大的心理压力，往往导致在恢复前就主动减仓。情绪压力下的被迫减仓与强制平仓在结果上没有本质区别。

## 9. 最小可执行练习

**练习 1（爆仓计算）**
本金 10 万元，使用 3 倍杠杆（借款 20 万），总持仓 30 万。
① 资产下跌多少时本金归零（理论爆仓点）？
② 经纪商维持保证金要求为净值/持仓 = 25%，触发强制平仓的资产跌幅是多少？

**练习 2（安全杠杆评估）**
你想投资某 A 股中小盘股票组合，历史最大回撤约 60%（2015 年熊市）。你可以接受本金最多亏 40%（维持精神状态和流动性需求）。
① 根据历史回撤，最大安全杠杆是多少？
② 如果留 50% 的安全余量（用 1.5 倍历史回撤计算），安全杠杆是多少？

**练习 3（融资成本测算）**
年融资利率 5%，使用 2 倍杠杆（50% 本金，50% 融资）。
① 融资成本占总投资的年化比例是多少？
② 你的投资需要达到多少年化收益才能覆盖融资成本并维持正的净收益？
③ 如果利率涨到 8%，临界收益率变为多少？

## 10. 一页总结

**爆仓机制**：杠杆放大亏损 → 净值低于维持保证金 → 强制平仓 → 在最低点出局。

**死亡螺旋**：强制平仓 → 更多抛售 → 价格进一步下跌 → 更多强制平仓（系统性风险来源）。

**安全杠杆上限**：账户净值回撤容忍度 / 资产预期最大回撤；实际使用应再留余量。

**核心危险**：杠杆可以让一个方向正确但时机不对的投资变成本金归零；融资的稳定性和流动性同样重要。

**核心原则**：永远假设最坏情景；不用无法承受其波动的杠杆；留足缓冲以承受随机波动。
