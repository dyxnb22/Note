# 伪量化识别

## 1. 主题定位

这篇讲如何识别"伪量化"产品和策略——那些借用量化的外衣，实际上依赖数据挖掘偏差、误导性回测、或直接欺骗的投资策略。是量化知识在防御层面的应用。

学它之前最好先懂：量化投资是什么、回测设计与偏差、因子投资框架。

学完后可以继续学：常见投资骗局（伪量化是骗局的一种常见形式）。

---

## 2. 核心概念

**伪量化（Pseudo-Quant）**
使用量化语言和工具的外观（算法、模型、回测图表），但实质上缺乏真实 Alpha 来源、存在严重的回测偏差，或使用欺骗性手段误导投资者的策略或产品。

**真量化 vs. 伪量化的本质区别**

| 维度 | 真量化 | 伪量化 |
|------|--------|--------|
| 逻辑来源 | 先有假说，再验证数据 | 先挖数据，再构造解释 |
| 回测样本 | 样本外验证 + 实盘跟踪 | 仅样本内，无样本外 |
| 成本假设 | 保守，贴近实际 | 极低或未说明 |
| 参数数量 | 少，逻辑简洁 | 多，高度过拟合 |
| 实盘表现 | 与回测衰减合理（30-50%）| 实盘远低于回测或不披露 |
| 策略透明度 | 核心逻辑可解释 | 黑箱，"算法保密" |

## 3. 核心机制

**伪量化的常见手法**

**手法 1：只展示样本内回测**
```
红旗：没有样本外验证，只展示一条漂亮的历史净值曲线
识别：问：回测数据是否参与了策略开发？样本外期间（未参与优化）的表现如何？
```

**手法 2：回测成本假设极低**
```
红旗：高换手率策略使用 0.01%（万一）甚至 0 的交易成本假设
识别：问换手率、问成本假设；自己估算真实年化成本并重新计算净 Alpha
```

**手法 3：选择性展示时间段**
```
红旗：回测恰好始于某个特别有利的起点（如 2013 年，跳过 2008 危机）
识别：要求展示更长时间段的回测，包含至少一个完整熊市
```

**手法 4：参数过度拟合（曲线拟合）**
```
红旗：策略有 10 个以上参数，每个参数都说是"经过优化得出的最优值"
识别：改变任一参数 ±20%，结果是否急剧变化？若是，策略是过拟合产物
```

**手法 5：幸存者偏差数据**
```
红旗：回测数据库不包含退市股票，只含目前存续的股票
识别：问：数据库是否含历史退市股？对价值类策略影响尤其大
```

**手法 6："黑箱"掩护**
```
红旗："具体算法是核心机密，无法披露"——真正有价值的量化策略，逻辑可以解释，具体参数才是机密
识别：策略逻辑（价值、动量、质量等）应该能解释；"连逻辑都保密"通常是没有合理逻辑
```

**手法 7：误导性业绩归因**
```
红旗：用绝对收益率宣传而非相对基准的超额收益（2020-2021 年牛市中，任何持股都能赚钱）
识别：要求看同期的 Alpha（相对基准超额），而非绝对收益
```

## 4. 分析框架

**评估一个量化产品的尽职调查清单**

**回测质量**
- [ ] 是否有样本外验证？样本外夏普 / 样本内夏普 > 0.5？
- [ ] 回测起始时间是否覆盖至少一个完整的经济周期（含 2008 年或 2015 年）？
- [ ] 交易成本假设是否合理（可以独立估算）？
- [ ] 数据是否包含退市股票？是否有前视偏差？

**策略逻辑**
- [ ] 能否用非技术语言解释策略的核心逻辑？
- [ ] 策略有经济学/行为学解释，还是纯粹的数据发现？
- [ ] 参数数量是否合理（< 5 个核心参数优于 20 个）？

**实盘历史**
- [ ] 有没有实盘记录？实盘与回测的相关性如何？
- [ ] 实盘期间的 Alpha 相对于同期市场基准是多少？
- [ ] 规模变化与 Alpha 变化的关系（是否有容量问题）？

**风险与透明度**
- [ ] 是否清楚说明了策略的风险因子暴露？
- [ ] 最大回撤在回测和实盘中分别是多少？
- [ ] 是否有独立的第三方业绩审计？

## 5. 关键指标或关键变量

**回测 → 实盘衰减比**
= 实盘年化 Alpha / 回测年化 Alpha

这个比值只能作为描述，不能用固定区间自动判断真假：分母接近零时比值不稳定，短实盘样本会受市场状态和估计误差支配，策略规模、成本与风险暴露也可能同时变化。应同时比较置信区间、市场环境、实际成交、容量和逐期归因。

**信息比率（ICIR）的时序稳定性**
分期 ICIR 用于检查信号是否只在特定状态生效。状态依赖不等于虚假，但必须在策略假设、仓位和风险预算中明确，不能用全期平均值掩盖。

**参数空间的平滑性**
检查核心参数邻域而非只看最优点。邻域范围应由参数含义和交易频率决定，不机械套用固定百分比；狭窄尖峰通常提示过拟合或数据问题。

## 6. 实际例子

**例 1：回测与实盘的断层**
某量化私募展示回测：2013-2021 年年化 35%，最大回撤 8%，夏普 3.2，看起来完美。实盘（2022 年起）：年化 8%，最大回撤 25%，夏普 0.4。原因分析：① 回测覆盖了 2013-2021 的特定市场结构 ② 回测使用了退市后才知道的股票列表 ③ 成本假设万分之一（实际远高于此）。

**例 2：参数优化导致的过拟合**
某策略核心参数是"均线交叉周期"。研究员测试了从 3 天到 120 天的所有整数组合（118 个参数值），发现 47 天和 48 天的组合夏普最高（2.8）。但当改变为 46 天或 49 天时，夏普降至 0.6。47-48 天没有任何经济学意义，这是典型的曲线拟合——只在历史数据上"碰巧"有效。

**例 3："量化"名义下的主观择时**
某产品宣称"AI 驱动量化策略"，实际上是用机器学习模型做宏观择时（满仓/空仓切换），2019-2020 年因为恰好在牛市期间重仓而表现亮眼。但深入看：① 模型每月只切换一次，不是真正的量化高频信号 ② 模型参数每季度人工调整 ③ 2022 年回撤 35%（与指数相当）。本质是加了算法包装的主观择时，与传统量化完全不同。

## 7. 常见误区

**误区 1：有回测图表就是量化**
美观的净值曲线图 + 数学公式 ≠ 有效的量化策略。任何人都可以用历史数据拟合出一条漂亮的曲线，关键是样本外的表现。

**误区 2：夏普比率很高 = 策略很好**
样本内高夏普率是必要条件但完全不充分。在充分数据挖掘下，任何夏普目标都可以在样本内达到。只有样本外夏普才有意义。

**误区 3：规模大 = 策略真实**
大规模量化产品也可能依赖回测偏差建立声誉，再用规模背书自身。规模是营销能力的体现，不是策略有效性的保证。

**误区 4：团队学历背景保证质量**
顶尖学校、量化背景是必要条件，不是充分条件。量化骗局或误导性产品中，从业者的学历背景通常很好——这是建立可信度的必要前提，但与策略是否真实有效无关。

## 9. 最小可执行练习

**练习 1（回测质量审查）**
如果有人给你看一个量化策略的回测，设计 5 个必须回答的问题，使你能判断这个回测是否可信。

**练习 2（成本重新估算）**
某策略声称年化收益 30%，年换手率 600%，交易成本 0.01%（万一）。
重新用合理成本（双边 0.3%）估算：净 Alpha 剩多少？结论是否改变？

**练习 3（参数敏感性自测）**
选一个你认为可能有效的简单规则（如"买入近 60 日跌幅超过 20% 的股票"）。
① 测试参数在 30-90 日、跌幅在 10%-30% 的范围内，结果是否平滑？
② 如果只在某个精确参数组合下有效，说明什么问题？

---

## 10. 一页总结

**伪量化本质**：借用量化工具和语言，实质是数据挖掘偏差 + 误导性回测 + 不透明包装。

**七大手法**：仅样本内回测、成本假设极低、选择有利时间段、参数过拟合、幸存者偏差数据、黑箱掩护、误导性业绩归因。

**核心验证**：样本外 Sharpe；实盘记录（不是回测）；成本假设可独立验证；参数敏感性平滑；逻辑可解释。

**核心认知**：美观的回测曲线是必要条件，远不是充分条件；"算法保密"通常是没有合理逻辑的委婉说法。

---

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 伪量化 | 借用量化外观但缺乏真实 Alpha 来源的策略 |
| 曲线拟合（Curve-fitting）| 参数过度优化以拟合历史数据，样本外失效 |
| 样本外验证 | 在未参与策略开发的独立数据段上验证策略有效性 |
| 回测衰减比 | 实盘 Alpha / 回测 Alpha，衡量真实策略转化率 |
| 参数敏感性 | 关键参数小幅变化对结果的影响程度 |
| 黑箱策略 | 连核心逻辑都无法解释的"算法保密"策略，是重要红旗 |
