# 基本面分析框架

## 1. 主题定位

这篇讲的是基本面分析的整体框架：如何从宏观到行业到公司逐层分析，判断一家公司值不值得投资以及当前估值是否合理。

它解决的问题是：基本面分析不是随机读财报，而是有系统的分析流程。什么应该先看，什么应该后看？分析的输出是什么？

在整个知识体系中，这是 `03_Stocks` 的核心分析框架，整合了后续的财报分析、护城河、估值等子模块。

学它之前最好先懂：供给与需求、股东权益与公司价值、经济周期。

学完后可以继续学：财报分析、护城河与商业模式、关键财务指标、DCF 估值（用框架的输出做估值）。

---

## 2. 核心概念

**基本面分析（Fundamental Analysis）**
通过分析公司的经营状况、财务数据、竞争优势、行业环境和宏观背景，估算其内在价值，并与市场价格比较，做出投资判断。

核心逻辑：长期来看，股票价格最终反映公司盈利能力。如果能准确判断一家公司的长期盈利能力，就能找到价格与价值之间的偏差机会。

与技术分析的区别：技术分析只关注价格和交易量数据；基本面分析关注价格背后的公司和行业。

**自上而下（Top-Down）vs. 自下而上（Bottom-Up）**

自上而下：宏观经济 → 行业 → 公司（先筛选好行业，再选公司）
自下而上：直接从公司出发，找出优秀公司（不论当前宏观是否"好"）

实践中，两种方法往往结合使用。

## 3. 核心机制

**基本面分析的三层结构**

**第一层：宏观分析（Macro Analysis）**
- 当前经济周期所处阶段？
- 利率和通胀走势如何？（影响折现率和消费）
- 汇率如何？（影响跨国公司）
- 政策环境：财政/货币政策方向？

宏观分析的目标：识别宏观顺风或逆风，判断对哪类行业和公司有利

**第二层：行业分析（Industry Analysis）**
- 行业处于什么阶段？（成长期、成熟期、衰退期）
- 行业的竞争格局如何？（波特五力模型）
- 行业的驱动因素是什么？
- 监管风险？

**波特五力模型（用于判断行业吸引力）：**
1. 现有竞争者的竞争强度
2. 新进入者的威胁（进入壁垒高低）
3. 替代品的威胁
4. 供应商的议价能力
5. 买方（客户）的议价能力

五力越弱，行业盈利能力越强（竞争越少，利润越高）。

**第三层：公司分析（Company Analysis）**
- 商业模式：公司如何创造价值、如何收费？
- 护城河：竞争优势来自哪里？是否可持续？
- 管理层：能力、激励、诚信？
- 财务状况：历史盈利质量、增长、资产负债健康度？
- 估值：当前价格隐含的预期合理吗？

## 4. 分析框架

**完整的基本面分析步骤（实操清单）：**

```
Step 1：宏观定位
  - 经济扩张还是收缩？
  - 利率上升还是下降？
  - 哪些行业在宏观背景下顺风 / 逆风？

Step 2：行业筛选
  - 行业增长驱动是什么？是否可持续？
  - 竞争格局：集中还是分散？
  - 进入壁垒：高还是低？
  - 行业盈利能力趋势？

Step 3：公司定位
  - 在行业中的位置：龙头还是跟随者？
  - 核心竞争优势是什么？
  - 管理层质量如何？

Step 4：财务分析
  - 历史增长率（营收、净利润、自由现金流）
  - 盈利质量（现金流 vs. 利润是否一致）
  - 资产负债健康度（负债率、流动性）
  - ROE 和 ROIC 趋势

Step 5：估值
  - 当前 PE、PB、EV/EBITDA 是否合理？
  - 用 DCF 测算内在价值区间
  - 与历史估值和同业比较

Step 6：催化剂与风险
  - 推动价格向价值回归的催化剂是什么？
  - 主要下行风险？
  - 是否有"不知道自己不知道"的潜在风险（黑天鹅）？
```

## 5. 关键指标或关键变量

**ROE / ROIC（资本回报率）**
反映什么：公司用资本产生回报的能力，是衡量竞争优势最重要的财务指标。
容易误判：ROE 可以被杠杆拉高，ROIC（投入资本回报率）更能反映真实的运营效率，尤其对重资产企业重要。

**自由现金流（FCF）**
反映什么：经营活动产生的现金减去资本支出，是公司真实可支配的现金，比净利润更难造假。
容易误判：部分资本密集行业（如半导体）在扩产期 FCF 为负，不代表公司质量差，需要看 FCF 的趋势和战略背景。

**毛利率和净利率趋势**
反映什么：公司的定价能力和成本控制能力，是判断竞争优势的重要指标。
容易误判：毛利率下降不一定是坏事（如主动降价提升市占率），要结合市场份额变化解读。

**营收增速和盈利增速**
反映什么：公司的成长轨迹，以及增长质量（两者是否一致）。
容易误判：营收增速高但利润不增长（甚至下降），说明公司在赔钱增长，可持续性差。

## 6. 实际例子

**例 1：宏观到行业到公司的链式分析**
宏观背景：利率上升，消费者购买力收紧
→ 行业判断：可选消费（奢侈品、高端餐饮）承压；必需消费（超市、药品）相对稳健
→ 公司选择：在必需消费行业中，选择护城河强、运营效率高的公司
→ 估值：必需消费公司传统 PE 20-25 倍在当前利率下略显偏高，但 PEG（PE/增速）仍在合理区间

**例 2：波特五力分析（白酒行业）**
1. 竞争者：寡头格局，茅台五粮液优势明显，新进入者极难抢市场
2. 新进入者：进入壁垒极高（品牌、渠道、年份窖藏积累，需要几十年）
3. 替代品：白酒有较强文化属性，短期替代品威胁低
4. 供应商：原料粮食成本占比低，供应商议价能力弱
5. 客户：高端白酒面对分散零售客户，议价能力弱

结论：五力都弱 → 行业吸引力极高 → 龙头企业利润率极高（毛利率超过 90%）是合理的。

## 7. 常见误区

**误区 1：基本面分析是关于预测"下一个季度业绩"**
基本面分析的核心是理解公司的长期竞争地位和盈利能力，而不是精确预测短期季报。频繁地基于季度数据做买卖决策，通常是策略方向的错误。

**误区 2：好公司 = 好股票**
公司优秀但股票已被充分定价（估值过高），仍然是差的投资。投资回报来自于"买得合理"，不只是"买得好"。

**误区 3：财务数字一定是真实的**
财务数据是分析的起点，但不是终点。需要检验盈利质量（现金流与利润的一致性）、异常项目、关联交易等，才能判断数字是否可信。

**误区 4：只做行业分析，忽视宏观**
行业顺风时，即使质量一般的公司也会上涨；行业逆风时，即使优秀公司也会承压。宏观和行业背景是分析的起点，不是可以跳过的步骤。

## 9. 最小可执行练习

**练习 1（行业分析）**
选择一个你感兴趣的行业，用波特五力模型做完整分析。最终判断：这个行业吸引力高还是低？龙头企业的盈利率应该高还是低？

**练习 2（自上而下）**
从当前宏观环境出发（利率、增长、通胀），推导出你认为在接下来 12 个月内应该配置的 2-3 个行业，写出完整的推理链。

**练习 3（自下而上）**
选择一家你熟悉的公司（消费者视角），分析它的商业模式：它如何赚钱？主要成本是什么？最大竞争风险是什么？

**练习 4（好公司 vs. 好股票）**
找一个你认为商业模式优秀、护城河强的公司，查看其当前 PE 和历史 PE 区间。当前估值处于历史什么分位？在当前估值下投资，有没有足够的安全边际？

**练习 5（盈利质量检验）**
取一家公司最近 3 年的净利润和经营性现金流。如果净利润增长而现金流未增长甚至下降，这意味着什么？写出可能的原因和你的判断。

## 10. 一页总结

**基本面分析的三层结构**：宏观（周期和政策背景）→ 行业（竞争格局和驱动因子）→ 公司（竞争优势、财务质量、估值）。

**核心工具**：波特五力（判断行业吸引力）；ROE/ROIC（核心盈利效率）；自由现金流（比净利润更可靠）；估值倍数（相对和绝对）。

**关键判断**：好行业 + 好公司 + 合理价格，三者都满足才是理想的投资。

**核心陷阱**：好公司 ≠ 好股票（估值过高则回报差）；财报数字需验证真实性；短期季报预测 ≠ 长期基本面分析。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 基本面分析 | 通过分析公司经营、财务、行业来估算内在价值的方法 |
| 自上而下 | 从宏观 → 行业 → 公司的分析顺序 |
| 自下而上 | 从公司出发，不依赖宏观判断的选股方法 |
| 波特五力 | 分析行业竞争格局的框架（五种竞争力量）|
| ROIC（投入资本回报率）| 净利润（调整后）/ 投入资本，排除杠杆影响的真实效率 |
| 自由现金流（FCF）| 经营现金流减资本支出，公司真实可用现金 |
| PEG（市盈增长比）| PE / 增速，用于成长股的相对估值 |
| 催化剂 | 推动股价从偏离内在价值向内在价值回归的具体事件 |
