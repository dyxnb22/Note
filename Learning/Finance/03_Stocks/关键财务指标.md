# 关键财务指标

## 1. 主题定位

这篇集中讲解股票分析中最常用的财务指标：定义、计算方式、经济含义，以及容易误用的场景。

它解决的问题是：面对一家公司时，应该重点关注哪 10-15 个指标？每个指标回答什么问题，有什么局限？

在整个知识体系中，这是财报分析的延伸，是基本面分析的工具箱，也是后续估值（PE、PB 等）的基础。

学它之前最好先懂：财报分析（理解三张报表的结构）、股东权益与公司价值（ROE、PB 的含义）。

学完后可以继续学：相对估值框架、DCF 估值、护城河分析（用 ROIC 验证护城河）。

---

## 2. 核心概念

财务指标分为四类：

| 类别 | 目的 | 代表指标 |
|------|------|---------|
| 盈利能力 | 公司赚钱的效率 | 毛利率、净利率、ROE、ROIC |
| 成长性 | 公司扩张速度 | 营收增速、净利润增速、EPS 增速 |
| 资产质量 / 偿债能力 | 财务健康度 | 负债率、流动比率、利息覆盖倍数 |
| 估值 | 当前价格是否合理 | PE、PB、PS、EV/EBITDA |

---

## 3. 核心机制

**盈利能力指标**

**毛利率 = (营收 - 营业成本) / 营收**
衡量产品竞争力和直接成本控制。毛利率决定了公司在抵扣所有其他费用前的空间。毛利率 < 竞争对手往往意味着定价能力弱或成本结构劣势。

**净利率 = 净利润 / 营收**
从收入到最终利润的转化率。净利率受销售 + 管理 + 研发 + 财务费用等所有运营摩擦影响。

**ROE（净资产收益率）= 净利润 / 股东权益**
巴菲特最重视的核心指标之一。衡量股东资本的利用效率。持续 ROE > 15% 通常是优质公司的信号，但需用杜邦分解判断来源质量。

**ROIC（投入资本回报率）= NOPAT / 投入资本**
`NOPAT = 息税前利润 × (1 - 税率)`
`投入资本 = 股东权益 + 有息负债 - 现金`
比 ROE 更能反映真实运营效率，排除了财务杠杆的影响。若 ROIC > WACC，公司在创造价值；若 ROIC < WACC，公司在摧毁价值。

**成长性指标**

**营收增速（Revenue Growth）**
顶线增长的驱动力。需区分有机增长（内生）和并购增长（外延）。

**EPS（每股收益）增速**
净利润增速与股本变化的综合结果。如果公司通过大量增发股票提升净利润，EPS 增速可能远低于净利润增速。

**资产质量 / 偿债能力指标**

**资产负债率 = 总负债 / 总资产**
财务杠杆水平。不同行业合理水平差异很大（银行可以 90%+ 而制造业超过 70% 可能危险）。

**流动比率 = 流动资产 / 流动负债**
短期偿债能力。通常 > 1.5-2 认为较安全，但也取决于行业特性（零售行业流动比率通常低）。

**利息覆盖倍数（ICR）= EBIT / 利息费用**
公司盈利是否足以偿还利息。ICR < 2 通常被认为风险较高。

**估值指标（详见估值模块，此处仅定义）**

**PE（市盈率）= 股价 / 每股收益（EPS）**
最常用的估值倍数，衡量为每 1 元盈利支付多少钱。PE 无法用于亏损公司。

**PB（市净率）= 市值 / 账面净资产**
适用于资产密集型行业（银行、房地产）。

**PS（市销率）= 市值 / 营收**
适用于亏损但有增长的公司（如初创期科技公司）。

**EV/EBITDA = 企业价值 / EBITDA**
排除资本结构和税率差异，跨公司比较更公平。

## 4. 分析框架

**分析一家公司时的指标使用优先级：**

```
第一步：判断业务规模和增长
→ 营收规模、营收增速、EPS 增速

第二步：判断盈利质量
→ 毛利率、净利率、FCF/净利润比

第三步：判断资本效率
→ ROE、ROIC vs WACC

第四步：判断财务健康
→ 资产负债率、利息覆盖倍数、流动比率

第五步：判断估值
→ PE、PB、PS、EV/EBITDA（结合增长预期）
```

## 5. 关键指标对照

| 指标 | 公式 | 反映 | 高值含义 | 低值含义 |
|------|------|------|---------|---------|
| 毛利率 | 毛利 / 营收 | 产品竞争力 | 定价强 | 成本压力或竞争强 |
| 净利率 | 净利润 / 营收 | 整体盈利效率 | 运营高效 | 费用高或竞争激烈 |
| ROE | 净利润 / 净资产 | 股权利用效率 | 资本高效 | 需分析来源 |
| ROIC | NOPAT / 投入资本 | 运营效率（排杠杆）| 真实护城河 | 可能在摧毁价值 |
| 资产负债率 | 总负债 / 总资产 | 财务杠杆 | 风险高 | 保守 |
| ICR | EBIT / 利息 | 利息偿还能力 | 安全 | < 2 危险 |
| PE | 股价 / EPS | 盈利的定价 | 预期高 | 低增长 / 被低估 |
| PB | 市值 / 净资产 | 净资产的定价 | 高 ROE 预期 | 低 ROE 或资产质量差 |

## 6. 实际例子

**例 1：毛利率验证定价能力**
茅台毛利率 > 90% vs. 某普通白酒毛利率 40%：同行业但毛利率相差 2 倍以上，清楚说明茅台的品牌溢价和定价权。

**例 2：ROIC vs. ROE 的差异**
某房地产公司：ROE = 20%（看起来不错），但资产负债率 85%，大量用债务放大股权。ROIC = 8%（低于行业平均和资本成本），说明公司靠高杠杆维持 ROE，实际运营效率一般。

**例 3：成长性 vs. 利润率的权衡**
某科技公司：营收增速 50%，但净利率 -10%（亏损），这是"先扩张，后盈利"模式。适合用 PS（市销率）估值，不适合 PE。关键问题是：达到盈利平衡点需要多长时间，届时边际利润率能有多高？

## 7. 常见误区

**误区 1：单一指标决定一切**
没有一个单一指标能完整描述公司质量。PE 低可能是便宜，也可能是低增长；毛利率高可能是护城河，也可能只是行业特性。指标需要组合分析。

**误区 2：跨行业比较估值倍数**
不同行业的合理 PE、PB 差异极大。银行 PE 10 可能是高估，科技公司 PE 30 可能是低估。只有在相同或相似行业内比较才有意义。

**误区 3：只看利润指标，忽视现金流**
净利润和经营现金流的差异往往揭示盈利质量问题。补充分析 FCF / 净利润比是必要的。

**误区 4：历史指标等于未来判断**
所有财务指标都是历史数据，而投资是对未来的判断。历史毛利率、ROE 是参考，核心问题是：这些指标的驱动因素在未来是否还会维持？

## 9. 最小可执行练习

**练习 1（指标计算）**
已知：营收 100 亿，营业成本 40 亿，净利润 15 亿，股东权益 50 亿，总资产 150 亿，总负债 100 亿，利息费用 2 亿，EBIT 20 亿。计算：毛利率、净利率、ROE、资产负债率、ICR、流动比率（流动资产 60 亿，流动负债 40 亿）。

**练习 2（ROIC 分析）**
某公司净利润 10 亿，税率 25%，利息费用 3 亿，股东权益 40 亿，有息负债 30 亿，现金 10 亿。计算 NOPAT、投入资本和 ROIC。若 WACC = 10%，该公司是在创造还是摧毁价值？

**练习 3（横向比较）**
找两家同行业竞争公司（如两家消费品公司），比较它们的毛利率、净利率、ROE、PE。分析：哪家公司盈利能力更强？哪家估值更合理？你会选哪家？

**练习 4（成长与盈利的权衡）**
公司 A：营收增速 20%，净利率 5%，PE = 30
公司 B：营收增速 5%，净利率 15%，PE = 15
PEG = PE / 增速：A 的 PEG = 1.5，B 的 PEG = 3。哪家更"便宜"？这个结论你是否认同？

**练习 5（指标综合判断）**
选择一家公司，整理其最近 3 年所有关键指标，判断：①盈利能力是改善还是恶化？②财务健康度如何？③当前估值是偏高、偏低还是合理？写出你的综合结论和最大不确定性。

## 10. 一页总结

**四类指标**：盈利能力（毛利率、净利率、ROE、ROIC）、成长性（营收增速、EPS 增速）、偿债能力（负债率、流动比率、ICR）、估值（PE、PB、PS、EV/EBITDA）。

**最重要的单一指标**：ROIC——若持续 > WACC，公司在创造价值，说明存在护城河；若 < WACC，规模扩张是在摧毁价值。

**使用原则**：同行业内横向比较 + 历史纵向趋势分析；单一指标 ≠ 结论；财务指标只反映过去，投资判断需预测未来。

**核心陷阱**：跨行业比较估值倍数；ROE 高 = 质量好（需看来源）；净利润 ≠ 真实盈利（需验证现金流）。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 毛利率 | (营收 - 营业成本) / 营收，衡量产品盈利空间 |
| 净利率 | 净利润 / 营收，综合盈利效率 |
| ROE | 净利润 / 股东权益，股权资本的使用效率 |
| ROIC | NOPAT / 投入资本，运营效率（排除财务杠杆影响）|
| EPS（每股收益）| 净利润 / 总股本，每股对应的盈利 |
| PE（市盈率）| 股价 / EPS，最常用估值倍数 |
| PEG | PE / 盈利增速，考虑增长的估值指标 |
| ICR（利息覆盖倍数）| EBIT / 利息费用，衡量还息压力 |
| 流动比率 | 流动资产 / 流动负债，短期偿债能力 |
| WACC | 加权平均资本成本，ROIC 的对比基准 |
