# 财报分析

## 1. 主题定位

这篇讲的是如何读懂上市公司的三张财务报表（利润表、资产负债表、现金流量表），以及如何从财报中提炼有用信息来判断公司质量。

它解决的问题是：财报不只是一堆数字，它是了解公司经营实况的最权威窗口。如何有目的地阅读，而不是被数字淹没？

在整个知识体系中，财报分析是基本面分析的核心技术工具，是关键财务指标的来源，也是识别财务造假的基础。

学它之前最好先懂：基本面分析框架（分析目的）、股东权益与公司价值（资产负债表基础）。

学完后可以继续学：关键财务指标（从财报提炼指标）、护城河与商业模式（用财报验证护城河）、财务造假识别。

---

## 2. 核心概念

**三张财务报表**

**1. 利润表（Income Statement / P&L）**
报告：一段时间内的收入、成本、利润
核心逻辑：营收 - 成本 = 利润

```
营业收入
- 营业成本
= 毛利润
- 运营费用（销售、管理、研发）
= 运营利润（EBIT）
- 财务费用（利息）
+ 其他收益 / 损失
= 税前利润（EBT）
- 所得税
= 净利润
```

**2. 资产负债表（Balance Sheet）**
报告：某一时点的资产、负债和股东权益快照
核心逻辑：资产 = 负债 + 股东权益

左边（资产）：流动资产（现金、应收账款、存货）+ 非流动资产（固定资产、无形资产）
右边（负债 + 权益）：流动负债（应付账款、短期借款）+ 非流动负债（长期债务）+ 股东权益

**3. 现金流量表（Cash Flow Statement）**
报告：一段时间内现金的流入流出
三部分：
- 经营活动现金流（最重要）：主营业务产生的现金
- 投资活动现金流：购买/出售资产
- 融资活动现金流：借债/还债/分红

**利润 vs. 现金流：关键区分**
利润是会计概念（应计制），现金流是真实现金（收付实现制）。公司可以有会计利润但没有现金（应收账款高、回款慢），也可以没有利润但有充裕现金（折旧大、现金流好）。

---

## 3. 核心机制

**为什么现金流比利润更重要**

利润可以通过会计方法调整（应收账款确认时间、折旧方法、存货计价等），但现金不会说谎——要么流进来了，要么没有。

"利润是观点，现金是事实"（Profit is an opinion, cash is a fact）

**三表的联动关系**

```
利润表的净利润
↓（调整非现金项目：折旧、摊销；营运资本变化）
现金流量表的经营现金流

利润表的净利润
↓（如果不分红）
进入资产负债表的留存收益（股东权益增加）

现金流量表的净现金变动
↓
资产负债表的现金余额变化
```

三表相互勾稽，内部一致性是财报真实性的基础验证。

**盈利质量的判断**

高质量盈利：
- 经营现金流 ≈ 净利润（利润都能收回来钱）
- 应收账款增速 ≤ 营收增速（没有靠放账期推销售）
- 存货增速 ≤ 营收增速（没有产成品积压）

低质量盈利信号：
- 净利润高但经营现金流持续为负
- 应收账款激增（可能是虚增收入）
- 存货激增（可能是滞销或存货虚增）

## 4. 分析框架

**读财报的四步流程：**

**第一步：看增长**
- 营收、毛利润、净利润、经营现金流的历史增速
- 趋势是加速还是减速？
- 增长是有机增长还是并购驱动？

**第二步：看盈利质量**
- 经营现金流 / 净利润（应接近 1 或 > 1）
- 应收账款周转天数（是否延长）
- 存货周转天数（是否积压）

**第三步：看资产负债健康度**
- 负债率（资产负债表杠杆水平）
- 流动比率（短期偿债能力）= 流动资产 / 流动负债
- 利息覆盖倍数 = EBIT / 利息费用（还债压力）

**第四步：看资本效率**
- ROE：净利润 / 股东权益
- ROIC：NOPAT / 投入资本
- 资本支出（Capex）/ 折旧（比值大说明在扩张）

## 5. 关键指标或关键变量

**毛利率 = 毛利润 / 营收**
反映什么：产品定价能力和直接成本控制。行业天花板指标。
容易误判：毛利率高不一定最终净利润高，要看费用率（运营效率）。

**净利率 = 净利润 / 营收**
反映什么：最终转化为利润的比例，综合反映盈利效率。
容易误判：净利率低的行业（如零售：2-3%），薄利多销，不代表质量差，要结合周转率看。

**经营现金流 / 净利润**
反映什么：盈利是否有现金支撑，是最重要的盈利质量指标。
容易误判：初创公司或快速扩张期，FCF 可能为负（大量投资），不能机械地用这个指标否定公司。

**应收账款周转天数（DSO）**
反映什么：客户多久还款，间接反映客户质量和销售的"真实性"。
容易误判：行业惯例不同，不同行业的 DSO 差异很大，需在同行业内比较。

**资产负债率 = 总负债 / 总资产**
反映什么：公司财务杠杆水平。
容易误判：适度杠杆可以提高 ROE，过高杠杆增加危机风险。不同行业的合理负债率差异很大（银行行业负债率天然高）。

## 6. 实际例子

**例 1：利润高但现金流差的警示**
某公司净利润 5 亿，但经营现金流仅 1 亿，应收账款从去年的 10 亿增加到 20 亿。

这说明：公司确认了大量收入（增加了应收账款），但钱没有收回来。可能原因：激进的收入确认政策、下游客户资质差、甚至是虚假收入。这类公司需要高度警惕，即使利润报告"好看"。

**例 2：看折旧和 Capex 的关系**
某制造公司折旧每年 2 亿，Capex 每年 8 亿（4 倍折旧）。

这说明公司在大量扩张产能（正常情况下投资驱动增长），FCF 可能暂时为负，但如果增长战略正确，这是合理的。若 Capex < 折旧（只修不增），说明公司"吃老本"，未来产能可能萎缩。

**例 3：三表勾稽验证**
某公司利润表净利润 3 亿，但现金流量表经营现金流 -1 亿，资产负债表应收账款增加 4 亿。

勾稽关系：净利润 3 亿 + 应收账款增加（-4 亿，现金未收到）≈ 经营现金流 -1 亿。三表一致，说明财报数字内部自洽。但应收账款激增本身是需要深究的问题。

## 7. 常见误区

**误区 1：净利润是最重要的财务指标**
净利润受会计政策影响大，现金流量表的经营现金流更难造假，是判断真实盈利能力的更可靠指标。

**误区 2：资产负债率越低越好**
适度负债可以放大股权回报（ROE）。零负债的公司可能在放弃本可利用的廉价资本；关键是负债质量和还款能力，不是绝对水平。

**误区 3：研发费用增加是坏事（增加成本）**
研发投入是未来竞争力的来源。会计上计入费用（降低利润），但经济上是资产（未来盈利的基础）。用净利润评价重研发公司会低估其长期价值，需看研发效率（研发 / 收入比、新产品增速）。

**误区 4：季度财报比年报更重要**
季度数据有大量噪声（季节性、一次性项目）。除非出现重大趋势改变，长期投资者应更关注年报和多年趋势，而非短期季报的波动。

## 9. 最小可执行练习

**练习 1（三表阅读）**
下载任意一家你感兴趣的 A 股上市公司最近一年年报，找到利润表、资产负债表、现金流量表，回答：营收增速是多少？毛利率是多少？经营现金流与净利润比是多少？

**练习 2（盈利质量判断）**
某公司 3 年数据：净利润分别 2 亿、3 亿、4 亿；经营现金流分别 3 亿、2 亿、1 亿；应收账款分别 5 亿、8 亿、14 亿。评估盈利质量，写出你的判断和依据。

**练习 3（三表勾稽）**
某公司净利润 10 亿，折旧 2 亿，应收账款增加 3 亿，应付账款增加 1 亿，存货增加 2 亿。估算经营现金流（粗略）。

**练习 4（杜邦分解）**
某公司 ROE = 18%，净利率 6%，资产周转率 1.5，财务杠杆 2。验证 ROE 的拆解。如果净利率从 6% 降到 4%，在其他不变的情况下，ROE 变成多少？

**练习 5（资本效率判断）**
A 公司：Capex 10 亿，折旧 8 亿，营收增速 15%
B 公司：Capex 3 亿，折旧 8 亿，营收增速 3%
哪家公司的资本投入战略更健康？你会选择哪个标准判断这个问题？

## 10. 一页总结

**三张表**：利润表（一段时期的经营结果）、资产负债表（某时点的资产负债快照）、现金流量表（现金流向）。

**三表关系**：净利润 → 留存收益（资产负债表）；利润调整后 → 经营现金流；现金净变动 → 资产负债表现金余额。

**盈利质量核心**：经营现金流 / 净利润（越接近 1 越好）；应收账款增速 vs. 营收增速（同步是健康）。

**分析四步**：增长（趋势）→ 盈利质量（现金流）→ 资产负债健康（偿债能力）→ 资本效率（ROE/ROIC）。

**核心陷阱**：利润可以被操纵，现金流更可靠；季报噪声大，看年度趋势；研发和扩张投入拉低净利润，但未必是坏事。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 利润表（P&L）| 报告一段时期内收入、成本、利润的报表 |
| 资产负债表（Balance Sheet）| 报告某时点资产、负债、股东权益的报表 |
| 现金流量表（CFS）| 报告一段时期内现金流入流出的报表 |
| 应计制 | 收入和成本按权责发生，而非现金收付来确认 |
| EBIT | 息税前利润（Earnings Before Interest and Tax）|
| EBITDA | 息税折旧摊销前利润，常用于估值分析 |
| 经营现金流（OCF）| 主营业务产生的现金，最重要的现金流指标 |
| 自由现金流（FCF）| 经营现金流 - 资本支出，真实可分配给股东的现金 |
| DSO（应收账款周转天数）| 应收账款 / 日均营收，衡量回款速度 |
| 流动比率 | 流动资产 / 流动负债，衡量短期偿债能力 |
