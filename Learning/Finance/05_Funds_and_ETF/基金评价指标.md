# 基金评价指标

## 1. 主题定位

这篇讲的是评价基金表现的核心量化指标，以及各指标的含义、局限和使用边界。

学它之前最好先懂：复利与对数收益率、风险与收益的基本关系、夏普比率（需了解基本概念）。

学完后可以继续学：夏普比率与绩效指标（更深入的量化分析）、主动与被动投资（评价语境）。

---

## 2. 核心概念与指标

**绝对收益指标**

- **年化收益率**：最基础的指标，需用几何年化（不是算术平均）。
- **累计净值**：从 1 元起步的总增值倍数，直观反映长期复利效果。

**风险调整后收益指标**

**夏普比率（Sharpe Ratio）**
= (年化收益率 - 无风险利率) / 年化波动率

衡量每承担一单位风险获得的超额收益。夏普比率越高，单位风险回报越好。

**索提诺比率（Sortino Ratio）**
= (年化收益率 - 无风险利率) / 下行标准差

只计算下行波动（对投资者有害的部分），不惩罚上行波动。比夏普比率更能反映真实风险。

**卡尔玛比率（Calmar Ratio）**
= 年化收益率 / 最大回撤（绝对值）

衡量每承担一单位最大回撤所获得的年化收益，对以控制最大损失为目标的策略更相关。

**风险指标**

- **最大回撤（Maximum Drawdown）**：从历史最高点到最低点的最大跌幅，反映极端下行风险。
- **波动率（Volatility）**：收益率标准差，衡量收益的波动幅度。
- **VaR（在险价值）**：在给定置信水平下（如 95%），最大可能损失额。

**相对表现指标**

- **Alpha**：相对于基准指数的超额收益（扣除 Beta 后）
- **Beta**：对基准指数的敏感度
- **信息比率（IR）= Alpha / 主动风险（Tracking Error）**：每承担一单位主动风险获得的超额收益，是评价主动基金经理的核心指标。

---

## 3. 核心机制

**为什么不能只看收益率？**

高收益可能伴随极高风险（如高杠杆策略）。风险调整后的收益（夏普、索提诺等）才能真正反映基金的"性价比"。

**不同指标适用的评价场景**

| 场景 | 推荐指标 |
|------|---------|
| 评价被动指数基金 | 跟踪误差、跟踪差、费率 |
| 评价主动基金对市场 | Alpha、Beta、信息比率 |
| 评价绝对收益策略 | 夏普比率、卡尔玛比率、最大回撤 |
| 评价风险控制能力 | 最大回撤、索提诺比率、VaR |
| 比较不同风险水平的基金 | 夏普比率（同一无风险利率环境下）|

**样本期的重要性**

所有指标都依赖历史数据，样本期越短，结论越不可靠：
- 3 年数据通常不足以区分技能和运气
- 理想情况需要 10 年以上（包含至少一次完整的经济周期和市场下跌）
- 还需要排除幸存者偏差

---

## 4. 分析框架

**评价一只基金的系统流程：**

1. **收益评估**：年化收益率（几何均值）、相对基准的超额收益
2. **风险评估**：年化波动率、最大回撤、下行偏差
3. **风险调整后评估**：夏普比率、索提诺比率、卡尔玛比率
4. **相对表现**：Alpha、Beta（系数 vs. 风格是否符合描述）、信息比率
5. **成本评估**：总费率、跟踪差（被动基金）
6. **时间序列稳定性**：不同时期表现是否一致？还是只有某一段时间超额？

---

## 5. 关键指标或关键变量

**夏普比率**
反映什么：每单位波动（总风险）的超额收益。
容易误判：夏普高不一定安全，可能来自"卖波动率"策略（平时稳定，极端时大幅亏损）。需结合最大回撤看。

**最大回撤**
反映什么：历史上最差的"从高到低"的损失，是投资者最能感受到的痛苦。
容易误判：最大回撤依赖历史，未来回撤可能超过历史最大值，不能用历史最大回撤作为绝对上限。

**信息比率（IR）**
反映什么：主动基金经理每承担一单位超出指数的主动风险，获得多少超额收益。
容易误判：IR 需要足够长的时间序列（3 年以上），且需要确认样本期代表性。

## 6. 实际例子

**例 1：两只基金的比较**
基金 A：年化 15%，波动率 20%，无风险利率 3%，最大回撤 -35%
基金 B：年化 12%，波动率 10%，无风险利率 3%，最大回撤 -15%

夏普比率：A = (15-3)/20 = 0.6；B = (12-3)/10 = 0.9
卡尔玛比率：A = 15/35 = 0.43；B = 12/15 = 0.8

结论：B 的收益绝对值低，但风险调整后收益更好。对于风险厌恶型投资者，B 更优。

**例 2：高夏普 ≠ 安全（尾部风险）**
某策略夏普比率 2.0（历史 5 年），最大回撤仅 5%。但仔细分析发现：策略是"卖出极端尾部期权"（收取少量期权费，但在极端市场崩溃时会承受巨额损失）。2020 年 3 月，该策略一个月亏损 80%，历史夏普和最大回撤完全失去参考价值。

---

## 7. 常见误区

**误区 1：夏普越高越好，不看其他指标**
夏普高可能来自低波动的"稳定"策略，但这类策略的极端风险（尾部）可能很大，正常情况下无法体现。

**误区 2：短期高收益说明基金经理能力强**
3 年甚至 5 年的超额表现，统计上还不足以排除运气因素。需要更长时间和样本量，并且要分析 Alpha 的来源（是风格偏差带来的？还是真正的个股选择能力？）

**误区 3：只看排行榜推荐的基金**
基金排行榜通常是按近期绝对收益排名，既有幸存者偏差，也有"买赢家"的追涨逻辑。历史上，前 1 年最高排名的基金，后 1 年大幅落后的概率很高。

## 9. 最小可执行练习

**练习 1（指标计算）**
基金年化收益 12%，年化波动率 15%，最大回撤 25%，下行偏差 10%，无风险利率 3%。计算夏普比率、索提诺比率、卡尔玛比率。

**练习 2（指标综合判断）**
查找两只你感兴趣的主动股票基金，收集近 5 年的夏普比率、最大回撤、年化收益、Alpha。综合这四个指标，判断哪只基金表现更好，并说明理由。

**练习 3（样本期影响）**
某基金近 3 年年化收益 20%，但 2018 年（熊市）该基金下跌 30%。如果你只看近 3 年数据（2019-2022 年牛市），夏普比率看起来很高。加入 2018 年的数据后，夏普比率和最大回撤如何变化？这说明什么？

---

## 10. 一页总结

**收益指标**：年化收益（几何均值）、累计净值。
**风险指标**：波动率、最大回撤、下行偏差。
**风险调整**：夏普（总波动）、索提诺（下行风险）、卡尔玛（最大回撤）。
**主动管理评价**：Alpha、Beta、信息比率（IR）。

**核心原则**：不能只看收益；需要足够长的时间（10 年+）；指标要综合使用，不依赖单一数字；警惕尾部风险被高夏普掩盖。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 夏普比率 | (年化收益 - 无风险利率) / 年化波动率 |
| 索提诺比率 | (年化收益 - 无风险利率) / 下行标准差 |
| 卡尔玛比率 | 年化收益 / 最大回撤，衡量单位回撤换取的收益 |
| 最大回撤 | 历史最高点到最低点的最大跌幅 |
| Alpha | 扣除 Beta 后的超额收益，代表主动管理附加价值 |
| 信息比率（IR）| Alpha / Tracking Error，主动基金的核心评价指标 |
| VaR | 给定置信水平下的最大可能损失 |
| 下行偏差 | 只计算负偏差的标准差，是索提诺的分母 |
