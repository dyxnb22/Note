# 量化投资是什么

## 1. 主题定位

这篇是 `11_Quant` 模块的入口，建立对量化投资的整体认知：定义、流程、与主观投资的区别、以及量化的核心优势和局限。

学它之前最好先懂：基本面分析框架（理解"主观投资"是什么，才能对比"量化"）、概率直觉与统计基础（量化的数学基础）。

学完后可以继续学：因子投资框架、回测原则、Alpha与Beta，以及 11_Quant 的全部内容。

---

## 2. 核心概念

**量化投资（Quantitative Investing）**
用数学模型、统计方法和计算机程序，系统化地从历史数据中发现规律，并以规则化方式执行投资决策的方法。

**量化投资的核心特征**

| 特征 | 说明 |
|------|------|
| 系统化 | 规则明确，去除主观判断 |
| 数据驱动 | 结论来自数据分析，而非感觉 |
| 可重复 | 相同规则在相同数据下产生相同结果 |
| 高频执行 | 可以同时管理数百至数千只股票 |
| 回测验证 | 在历史数据上验证策略的有效性 |

**量化 vs. 主观投资的核心区别**

| 维度 | 主观投资 | 量化投资 |
|------|---------|---------|
| 决策来源 | 人的判断和经验 | 数学模型和规则 |
| 覆盖范围 | 少数深度研究的公司 | 宽泛，可覆盖全市场 |
| 执行速度 | 较慢 | 极快（毫秒级别） |
| 偏差来源 | 行为偏差 | 模型风险、数据问题 |
| 适合场景 | 信息不对称、长期价值 | 统计规律、大规模分散 |

## 3. 核心机制

**量化投资的完整链条**

```
数据（原始输入）
    ↓
因子（信号提取）
    ↓
Alpha 模型（预测收益）
    ↓
风险模型（控制风险）
    ↓
组合优化（确定权重）
    ↓
交易执行（订单管理）
    ↓
绩效归因（验证和改进）
```

每个环节都有专门的知识体系，各环节的质量共同决定策略的最终表现。

**量化投资的三种主要类型**

**1. 统计套利（Statistical Arbitrage）**
利用资产之间的历史统计关系（均值回归），当偏差出现时做多低估、做空高估，等待收敛。高频，风险低，容量有限。

**2. 因子投资（Factor Investing / Smart Beta）**
系统性地暴露于历史上有超额收益的因子（价值、动量、质量等），在大量股票上分散实施。中低频，可规模化。

**3. 机器学习型量化**
用机器学习方法（神经网络、随机森林等）从数据中发现复杂非线性关系。高数据需求，过拟合风险更大。

## 4. 分析框架

**评估一个量化策略的框架：**

1. **逻辑合理性**：这个信号有没有经济学/行为学的解释？还是纯粹的统计发现？（无逻辑支撑的 = 更可能是数据挖掘）
2. **样本外表现**：在不用于建模的时期是否仍然有效？
3. **容量分析**：策略的有效规模是多少？（太小则规模化困难）
4. **成本分析**：扣除交易成本、冲击成本后，Alpha 是否仍然存在？
5. **衰减分析**：Alpha 是否在随时间衰减？衰减速度是否超过预期？

---

## 5. 关键现象与观察点

**量化策略的"拥挤"问题**
当大量量化资金追逐同一因子 / 信号时，因子溢价会被侵蚀，且当所有人同时退出（去杠杆），会出现剧烈波动（"量化危机"）。2007 年 8 月、2015 年 8 月的量化基金集体异常损失都与"因子拥挤"有关。

**量化与主观的界限模糊**
现代投资实践中，量化工具（因子筛选、系统化执行）被主观投资者广泛采用；主观判断（对宏观情景的判断）也被部分量化策略纳入。两者的界限日益模糊。

`量化视角：量化投资的核心竞争优势不是"机器比人聪明"，而是"系统化执行消除了行为偏差"和"大规模处理数据的能力"。这两点是个人投资者也可以部分借鉴的。`

## 6. 实际例子

**例 1：简单因子策略的实现**
"买入 A 股中 PB 最低的 20% 股票，每季度调仓"——这是最简单的价值因子策略。规则完全明确，任何人按相同规则都会得到相同组合，这是"量化"的本质。

**例 2：统计套利的直觉**
两家高度相关的银行股（如工商银行和建设银行），历史上价格比率（A/B）在 1.05 附近波动。某天比率变成 1.15（A 相对贵了）→ 做空 A、做多 B →等待比率回归到 1.05 → 平仓获利。这个策略完全基于历史统计关系，不依赖对两家银行的基本面判断。

## 7. 常见误区

**误区 1：量化投资是黑箱，普通人不能理解**
因子投资等基础量化策略逻辑完全透明（低 PB = 价值因子），任何人都可以理解和实施。机器学习类复杂模型确实更难解释，但基础量化方法的逻辑并不神秘。

**误区 2：量化投资一定比主观投资更好**
量化和主观各有适用场景。量化在大规模分散、规律性信号上有优势；主观投资在信息优势、复杂判断、小流动性市场上可能更有效。

**误区 3：回测效果好 = 实盘会好**
回测在历史数据上"拟合"，但实盘环境不同（数据偏差、市场结构变化、策略拥挤）。历史好的回测是必要条件但远不是充分条件。

## 9. 最小可执行练习

**练习 1（量化 vs. 主观）**
对同一只股票，分别用主观分析（基本面、护城河）和简单量化信号（PB、PE、动量）做分析。两种方法的结论一致吗？各自依赖什么假设？

**练习 2（简单因子策略设计）**
设计一个最简单的量化策略：选取因子（如 PE 最低 20%）、调仓频率（月/季）、选股数量（20 只 / 50 只）。写出完整规则，确保规则明确到任何人按规则都能得到相同结果。

**练习 3（数据驱动 vs. 直觉）**
测试一个你直觉认为有效的投资规律（如"股价创 52 周新高后继续上涨"），尝试在历史数据中验证它（即使是简单的 Excel 分析）。结果是否与你的直觉一致？

## 10. 一页总结

**量化投资本质**：用数学和数据系统化替代人的主观判断，规则化执行投资决策。

**核心链条**：数据 → 因子 → Alpha 模型 → 风险模型 → 组合优化 → 交易执行 → 归因。

**三种主要类型**：统计套利（高频、低容量）+ 因子投资（中低频、可规模化）+ 机器学习（高数据、高过拟合风险）。

**核心优势**：消除行为偏差 + 大规模处理数据。

**核心局限**：历史不等于未来；回测偏差；拥挤风险；交易成本侵蚀 Alpha。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 量化投资 | 用数学模型和规则系统化执行投资决策的方法 |
| 因子 | 对股票收益有系统性预测能力的特征变量（如 PE、动量）|
| 统计套利 | 利用资产历史统计关系，在偏差时做多低估、做空高估 |
| Alpha 模型 | 预测个股超额收益的量化模型 |
| 风险模型 | 估计组合风险来源和规模的量化工具 |
| 组合优化 | 在收益预测和风险约束下，确定每只股票最优权重的数学过程 |
| 回测（Backtesting）| 在历史数据上模拟策略表现的验证过程 |
