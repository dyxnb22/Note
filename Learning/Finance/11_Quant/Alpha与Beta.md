# Alpha 与 Beta

## 1. 主题定位

这篇讲投资收益的分解框架：Beta 是市场暴露带来的系统性收益，Alpha 是超越市场的超额收益。理解两者的区别是评估任何投资策略的基础。

学它之前最好先懂：风险与收益的基本关系、期望方差与协方差、分散化与相关性。

学完后可以继续学：因子投资框架（多因子模型是 Alpha/Beta 分解的延伸）、夏普比率与绩效指标（信息比率基于 Alpha 构建）、回测设计与偏差。

---

## 2. 核心概念

**Beta（系统性风险暴露）**
投资组合收益随市场整体波动的敏感程度。Beta = 1 表示与市场同步；Beta = 2 表示市场涨 1%，组合涨 2%（也跌 2%）。

Beta 代表的是**已知、可以低成本复制**的市场风险溢价，不需要为此额外支付主动管理费用。

**Alpha（超额收益 / 主动价值）**
在控制市场风险（Beta）之后，投资组合相对基准的超额收益部分。Alpha 代表管理技能或信息优势，是主动管理真正的"产品"。

**资本资产定价模型（CAPM）**

```
E(Rp) = Rf + β × (Rm - Rf)

其中：
Rp = 组合收益
Rf = 无风险利率
β  = 组合 Beta
Rm = 市场收益
```

收益超过 CAPM 预测的部分 = Alpha（Jensen's Alpha）。

**Alpha vs. Beta 的本质区别**

| | Alpha | Beta |
|---|-------|------|
| 来源 | 技能 / 信息 / 错误定价 | 风险暴露 |
| 是否可以低成本复制 | 不能 | 能（指数基金） |
| 是否应该为此支付高费用 | 是 | 否 |
| 稀缺性 | 稀缺 | 充裕 |
| 规模敏感性 | 高（规模增大 Alpha 下降）| 低 |

---

## 3. 核心机制

**收益分解：从单因子到多因子**

单因子（CAPM）：
```
Rp = Rf + β×(Rm-Rf) + α + ε
 └────────────┘   └┘
      Beta        Alpha
```

多因子（Fama-French 三因子）：
```
Rp = Rf + β_mkt×MKT + β_smb×SMB + β_hml×HML + α + ε
 └─────────────────────────────────────────┘   └┘
                  多个 Beta                   Alpha
```

随着因子模型越来越丰富，原来算作"Alpha"的部分可能被归入某个因子的 Beta（例如动量因子）。Alpha 与 Beta 的边界随定义的因子集而移动。

**Alpha 的两种来源**

1. **信息优势**：获得市场其他参与者没有的信息，或更早获得信息（传统价值投资的护城河分析、调研优势）
2. **行为套利**：利用其他投资者的认知偏差（过度反应/反应不足），在错误定价回归前获利

随着信息获取成本下降（机器学习、替代数据），纯信息优势的 Alpha 越来越难维持；行为套利类 Alpha 更持久，但容量有限。

**Beta 不等于不好**

Beta 本身不是"坏东西"——市场风险溢价（股权风险溢价，历史约 5-7%）是长期最重要的收益来源之一。问题在于：
- 不要为 Beta 支付主动管理费用（用低成本指数基金获取 Beta）
- 主动管理者的费用应该仅对应其真实创造的 Alpha

**Beta 的类型**

| Beta 类型 | 来源 | 典型代表 |
|-----------|------|---------|
| 市场 Beta | 整体股票市场暴露 | 指数基金 |
| 行业 Beta | 特定行业暴露（科技/能源等）| 行业 ETF |
| 因子 Beta | 价值/动量/质量等因子暴露 | Smart Beta ETF |
| 宏观 Beta | 利率/通胀/汇率等宏观因子 | 债券、商品 |

## 4. 分析框架

**评估主动管理者是否真正创造 Alpha 的框架：**

1. **基准对比**：超过市场收益 ≠ Alpha，需要控制同等 Beta 暴露后比较
2. **因子归因**：超额收益能否被已知因子（价值、动量、质量等）解释？能被解释的是 Beta，剩余才是 Alpha
3. **统计显著性**：Alpha 在统计上是否显著（t 值 > 2）？用多少年的数据才够？（通常需要 5-10 年）
4. **费后 Alpha**：扣除管理费、交易成本后，Alpha 是否仍然为正？
5. **规模衰减**：随着管理规模增大，Alpha 是否显著下降？（是则说明 Alpha 来自容量有限的策略）
6. **一致性**：Alpha 在牛市和熊市中是否都存在？只在牛市有 Alpha 可能是承担了隐性风险

## 5. 关键指标或关键变量

**Beta 系数**
衡量对市场（或某因子）的敏感度。计算方法：组合收益与基准收益回归的斜率系数。

**Jensen's Alpha**
回归截距项，即控制 Beta 后的超额收益。Alpha > 0 且统计显著，才是真实的主动价值。

**信息比率（IR）**
= Alpha / 主动风险（Tracking Error）

衡量每单位主动风险获得多少 Alpha。通常 IR > 0.5 认为有价值，> 1.0 认为优秀。

## 6. 实际例子

**例 1：Alpha 被因子解释的情况**
某主动基金经理 10 年年化收益 16%，市场同期 12%，看似 Alpha = 4%。但经因子分析：该经理持续重仓小市值+低 PE 股票（规模因子 + 价值因子），这两个因子在同期年化贡献约 3.5%。真实 Alpha 只有约 0.5%，且不显著。→ 这个经理收取的 1.5% 管理费主要是为 Beta 付费，不合算。

**例 2：Alpha 的规模衰减**
某量化对冲基金管理 1 亿时，年化 Alpha 约 15%；扩张至 50 亿时，Alpha 降至 5%；扩张至 200 亿时，Alpha 接近 0%。原因：策略容量有限，规模越大买卖冲击越大，Alpha 被交易成本侵蚀。→ Alpha 的稀缺性部分体现在容量约束上。

**例 3：牛市中的"假 Alpha"**
某基金在 2019-2021 年科技牛市中年化收益 35%，远超市场（+20%）。但实际上基金持仓集中在高 Beta 科技股（平均 Beta = 1.6），没有真正的选股能力。2022 年科技股回调时，该基金跌幅也远超市场。→ 牛市高收益可能只是高 Beta，不是 Alpha。

## 7. 常见误区

**误区 1：跑赢指数就是有 Alpha**
跑赢指数可能只是承担了更多 Beta（仓位更重、板块押注更集中）。真正的 Alpha 需要在风险调整后、因子中性化后仍然存在。

**误区 2：Alpha 是固定的属性**
Alpha 会随市场参与者增多而衰减（策略被复制）、随规模增大而减小（冲击成本上升）、随时间推移而变化（市场结构改变）。没有"永久性 Alpha"。

**误区 3：Beta 越低越好**
低 Beta 本身不是优点——低 Beta 意味着更少地参与市场上涨。低 Beta + 高 Alpha 才是理想组合。单纯为了降 Beta 而持有大量现金，是机会成本的浪费。

**误区 4：Alpha 和 Beta 是固定不变的**
随着使用的因子模型不同，同一个策略的 Alpha 分解结果会不同。今天的"Alpha"可能随着新因子的发现，明天被重新归类为某种 Beta。

## 9. 最小可执行练习

**练习 1（Alpha/Beta 拆解）**
某基金年化收益 18%，市场 12%，无风险利率 3%，基金 Beta = 1.4。
计算：① CAPM 预测的期望收益 ② Jensen's Alpha ③ 这个 Alpha 是否值得支付 1% 年管理费？

**练习 2（因子归因直觉）**
某"选股高手"的持仓特征：平均 PE 10x（全市场 20x），过去 12 个月涨幅前 20%，小市值为主。
分析：他的超额收益中，哪些部分更可能是因子暴露（Beta），哪些部分更可能是真实 Alpha？

**练习 3（Alpha 衰减思考）**
某策略在规模 5 亿时夏普 1.5，规模 50 亿时夏普 0.6。请解释：① 为什么规模会导致 Alpha 衰减 ② 如何估算该策略的"最优规模"（Alpha 衰减到接近 0 的规模）

---

## 10. 一页总结

**Beta**：市场（或因子）风险暴露带来的系统性收益，可低成本复制，不应为其支付高费用。

**Alpha**：控制所有已知 Beta 后的超额收益，代表技能/信息优势，稀缺且有规模限制。

**核心公式**：收益 = Beta × 市场溢价 + Alpha + 噪声

**评估真实 Alpha**：因子中性化归因 → 费后 Alpha → 统计显著性检验 → 规模稳定性验证。

**关键认知**：Alpha 和 Beta 的边界随因子模型的丰富而移动；今天的 Alpha 可能是明天的 Beta。
