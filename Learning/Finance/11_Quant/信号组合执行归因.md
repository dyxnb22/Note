# 信号组合、执行与归因

## 1. 主题定位

这篇讲量化策略从研究到实盘的"最后一公里"：多信号如何合成单一组合信号、执行层面的关键决策，以及如何通过绩效归因判断策略哪里有效、哪里漏了 Alpha。

学它之前最好先懂：因子投资框架、Alpha与Beta、回测原则、滑点交易成本与容量。

学完后可以继续学：13_Research_Workflow（研究方法与备忘录写作）。

---

## 2. 核心概念

**信号合成（Signal Combination）**
将多个独立因子信号合并为单一的选股/择时得分的过程。合成方法影响策略的多样性和稳定性。

**组合优化（Portfolio Optimization）**
在给定收益预测（Alpha 模型）和风险估计（风险模型）的条件下，求解最优股票权重的数学过程。目标是最大化风险调整后收益，同时满足约束条件（换手率、行业偏离、个股集中度等）。

**绩效归因（Performance Attribution）**
将实盘或回测的总收益分解为各个来源（因子贡献、选股贡献、行业贡献、交易成本损耗等），识别 Alpha 的实际来源。

**实施短缺（Implementation Shortfall）**
决策价格（策略信号产生时的价格）与最终成交价格的差距。是衡量执行质量的核心指标，量化了"想到"和"做到"之间的差距。

---

## 3. 核心机制

**信号合成的主要方法**

**方法一：等权合成**
```
综合得分 = (标准化因子A得分 + 标准化因子B得分 + ...) / N
```
优点：简单透明，不易过拟合；缺点：假设每个因子同等重要。

**方法二：IC 加权合成**
```
综合得分 = Σ(IC_i × 标准化因子i得分)
其中 IC_i 为因子i的历史平均信息系数
```
优点：历史表现好的因子权重更大；缺点：IC 本身有估计误差，可能引入过拟合。

**方法三：机器学习合成**
用模型（随机森林、神经网络等）学习因子与收益的非线性关系。优点：可捕捉因子交互效应；缺点：黑箱、过拟合风险高、解释性差。

**组合优化的标准流程**

```
Alpha 模型输出：每只股票的预期超额收益（α_i）
风险模型输出：协方差矩阵（Σ）
↓
目标函数：最大化 α'w - λ/2 × w'Σw
↓
约束条件：
  - 行业偏离 ≤ X%
  - 个股权重 ≤ Y%
  - 换手率 ≤ Z%
  - 净多头（w_i ≥ 0）
↓
解优化问题 → 最优权重向量 w*
```

**绩效归因的 Brinson 模型（股票选择 + 行业配置）**

```
总超额收益 = 行业配置效应 + 个股选择效应 + 交互效应

行业配置效应 = Σ (w_策略,i - w_基准,i) × (r_基准,i - r_基准)
个股选择效应 = Σ w_基准,i × (r_策略,i - r_基准,i)
```

将总 Alpha 拆分为"押对行业"贡献多少 vs. "在行业内选对股票"贡献多少。

## 4. 分析框架

**实盘策略的 Alpha 损耗链**

```
理论 Gross Alpha（因子回测）
- 前视偏差修正（数据真实可得时的 Alpha 衰减）
- 交易成本（佣金 + 冲击）
- 执行损耗（实施短缺）
- 因子衰减（策略拥挤、信号失效）
= 实盘净 Alpha
```

识别每个环节的损耗量，是持续改进策略的基础。

**月度归因报告的核心内容**

| 归因维度 | 关键问题 |
|---------|---------|
| 因子贡献 | 哪个因子本月贡献了 Alpha？哪个拖累了？ |
| 行业配置 | 超额收益来自行业暴露还是选股？ |
| 交易成本 | 本月实际成本是多少？与预算对比如何？ |
| 执行质量 | 实施短缺是多少？是否可改善？ |
| 风险暴露 | 组合的实际因子暴露是否符合预期？ |

## 5. 关键指标或关键变量

**实施短缺（IS，Implementation Shortfall）**

```
IS = (成交价 - 决策价) / 决策价 × 方向系数
```

IS 越低越好（理想值为 0）。IS 高说明执行效率差，Alpha 在执行过程中漏损。

**IC 稳定性（ICIR）**
= 月度 IC 均值 / 月度 IC 标准差

ICIR 是信号组合质量的核心指标。ICIR > 0.5 通常认为可以实盘部署，> 1.0 认为较好。

**Alpha 传导效率**
= 实盘净 Alpha / 回测 Gross Alpha

通常期望 > 50%（即回测 Alpha 至少一半能在实盘中实现）。传导效率 < 30% 说明执行或成本有严重问题。

## 6. 实际例子

**例 1：信号合成方式的影响**
某量化团队使用价值、动量、质量三个因子：
- 等权合成：三因子同等贡献，综合 ICIR = 0.7
- IC 加权（过去 6 个月）：动量因子 IC 最近最高，权重增加 → ICIR = 0.75，但样本外 ICIR 降至 0.6（因为 IC 加权引入了过拟合）
- 最终选择等权合成，因为稳定性更优。

**例 2：绩效归因揭示问题**
某量化基金某季度跑输基准 3%。归因分析发现：
- 因子选股贡献：+2%（因子有效）
- 行业配置效应：-4%（不经意间重仓了低迷行业）
- 交易成本：-0.8%（成本正常）
- 执行损耗：-0.2%（执行正常）

结论：超额亏损的来源是行业配置，而非因子失效。改进方向：加强行业中性化约束。

**例 3：实施短缺的实际量化**
某策略决策价格（信号产生时收盘价）到下一日开盘的平均价差：大盘股约 0.1%，中盘股约 0.3%，小盘股约 0.8%。如果策略持仓 100 只股票，每月换 30 只，小盘股导致的执行损耗约 0.24%/月（年化约 2.9%），是显性成本的数倍。

## 7. 常见误区

**误区 1：信号越多越好**
信号叠加不是简单的分散化——高度相关的信号叠加只是重复信息，反而增加了研究复杂度和过拟合风险。每增加一个信号，需要问：这个信号在已有信号之外提供了多少独立预测力（增量 IC）？

**误区 2：组合优化一定优于简单规则**
组合优化依赖精确的协方差矩阵估计，而协方差矩阵估计本身误差很大（维度诅咒）。在因子数量不多的情况下，简单等权或行业等权往往比理论优化效果更稳定。

**误区 3：回测夏普高 = 可以直接实盘**
回测夏普没有考虑执行成本和实施短缺。实盘前必须有完整的成本预算和执行模拟，确认净 Alpha 仍然显著。

**误区 4：归因分析只在亏损时做**
盈利时的归因同样重要——需要确认盈利来自预期的信号，而非偶然的行业暴露或市场 Beta，这样才能判断 Alpha 是否可持续。

## 9. 最小可执行练习

**练习 1（信号合成）**
三个因子的月度 IC 数据（最近 12 个月）：
- 价值因子 IC 均值 = 0.04，标准差 = 0.06
- 动量因子 IC 均值 = 0.05，标准差 = 0.04
- 质量因子 IC 均值 = 0.03，标准差 = 0.05

计算：① 各因子 ICIR ② IC 加权合成时各因子权重 ③ 如果因子两两相关系数约 -0.2，等权合成是否合理？

**练习 2（归因分析）**
某量化策略月度超额收益 +1.5%，已知：因子选股贡献 +2.2%，行业配置 -0.5%，交易成本 -0.3%。
问：① 这个结果好不好？② 下个月如何改进？③ 如果行业配置是"意外"的（没有刻意做行业押注），说明了什么问题？

**练习 3（Alpha 传导效率）**
某策略回测 Gross Alpha = 15%，实盘运行 6 个月后发现：
- 实际年化净超额收益 = 6%
- 年化交易成本 = 2.5%
- 年化执行损耗 = 1.5%
- 因子实际贡献 = 10%（低于回测，因为拥挤）

分析每个环节的 Alpha 损耗，计算传导效率，并提出最优先改进的一个环节。

## 10. 一页总结

**信号合成**：等权（简单稳健）vs. IC 加权（历史驱动但过拟合风险高）vs. 机器学习（复杂但解释性差）。

**组合优化**：最大化风险调整收益，关键约束是换手率、行业偏离、集中度。简单规则往往不亚于复杂优化。

**实盘 Alpha 损耗链**：理论 Alpha → 数据修正 → 成本 → 执行损耗 → 因子衰减 → 净 Alpha。

**归因价值**：不只是在亏损时做，盈利时同样需要确认 Alpha 来源的可持续性。

**核心指标**：ICIR（信号稳定性）、实施短缺（执行质量）、Alpha 传导效率（整体漏损）。
