# 金融体系总览

## 1. 主题定位

这篇是 `02_Markets` 模块的起点，提供整个金融体系的鸟瞰图：钱在哪里、怎么流动、谁是参与者、市场如何分类。

它解决的问题是：金融体系的整体架构是什么？储蓄如何变成投资？各类市场之间有什么关系？

在整个知识体系中，这是进入所有具体资产类别（股票、债券、基金、衍生品）之前必须建立的宏观框架。没有这张地图，后续的产品知识容易碎片化。

学它之前最好先懂：货币与信用（货币和信用的基本概念）、风险与收益（基本分析框架）。

学完后可以继续学：金融中介与金融市场、市场结构与流动性，以及各具体资产类别模块。

## 2. 核心概念

**金融体系（Financial System）**
连接资金盈余方（储蓄者）和资金短缺方（投资者/借款人）的一整套机构、市场和工具的集合。

核心功能：
1. 资金融通（将储蓄转化为生产性投资）
2. 风险分配（将风险分配给愿意且能够承担的人）
3. 流动性提供（将不流动的长期投资转化为可交易的证券）
4. 价格发现（通过市场交易形成资产的合理价格）
5. 信息传递（价格信号和研究报告传递信息）

**金融中介（Financial Intermediary）**
站在储蓄者和借款人之间，将分散的资金汇聚起来再贷出的机构。主要包括：银行、保险公司、养老基金、共同基金。

**直接融资 vs. 间接融资**
- 直接融资：企业直接向投资者发行股票或债券筹资（通过资本市场）
- 间接融资：储蓄者把钱存银行，银行贷给企业（通过金融中介）

`中国语境提醒：中国长期以间接融资（银行贷款）为主，直接融资（股票和债券市场）比例相对低于发达国家，但近年来政策方向是提升直接融资比例。`

## 3. 核心机制

**金融体系的资金流动**

```
家庭 / 储蓄者
│  存款、购买基金/股票/债券
↓
金融中介（银行、保险、基金）  ←→  金融市场（股票、债券、外汇、衍生品）
│  贷款、投资
↓
企业 / 政府（借款方 / 融资方）
│  还本付息、分红
↓
回流至储蓄者
```

**金融市场的分类**

按融资期限：
- 货币市场（Money Market）：短期借贷（< 1 年），如国库券、商业票据、银行间拆借
- 资本市场（Capital Market）：长期融资（≥ 1 年），如股票、债券

按是否为首次发行：
- 一级市场（Primary Market）：企业首次发行证券筹资（IPO、债券发行）
- 二级市场（Secondary Market）：已发行证券在投资者之间交易（证券交易所）

按组织形式：
- 交易所市场（Exchange Market）：集中竞价，透明，有监管（如上交所、纽交所）
- 场外市场（OTC Market）：双边协商，灵活，流动性可能较低（如大部分债券交易）

**金融体系的参与者**

| 类型 | 代表机构 | 主要功能 |
|------|----------|----------|
| 商业银行 | 工商银行、美国银行 | 存贷款中介、支付结算 |
| 投资银行 | 高盛、中金公司 | IPO 承销、并购顾问 |
| 保险公司 | 中国人寿、AIG | 风险转移、长期资产管理 |
| 养老基金 | 社保基金、CalPERS | 长期投资、为退休储蓄增值 |
| 共同基金/公募基金 | 易方达、先锋领航 | 集合理财，提供分散化 |
| 对冲基金 | 桥水、城堡投资 | 追求绝对收益、策略多样 |
| 私募股权 | 黑石、KKR | 非上市公司投资和并购 |
| 证券交易所 | 上交所、纽交所 | 提供交易场所和规则 |
| 央行 | 中国人民银行、美联储 | 货币政策、系统性风险维护 |

## 4. 分析框架

**评估金融体系健康度的框架：**

1. 信用中介是否畅通？（银行是否正常放贷）
2. 市场流动性是否充足？（买卖差价、成交量）
3. 风险定价是否合理？（信用利差、股权风险溢价是否反映真实风险）
4. 信息是否充分传递？（信息不对称程度）
5. 系统性风险水平？（机构间连接程度、杠杆水平、集中度）

---

## 5. 关键现象与观察点

**系统性风险（Systemic Risk）**
当金融系统内某个关键节点（如大型银行）失败时，可能通过连锁反应导致整个系统崩溃。2008 年金融危机的核心是系统性风险的爆发。

**金融周期与经济周期的互动**
金融周期（信贷扩张与收缩）与经济周期相互叠加，但不完全同步。金融危机（金融周期顶部崩溃）带来的衰退往往比普通经济衰退更深、更持久。

**金融深度（Financial Depth）**
金融市场规模相对 GDP 的比例。通常认为适度的金融深化（而非过度）对经济增长有促进作用。过度金融化可能带来资源错配和系统性风险。

## 6. 实际例子

**例 1：2008 年金融体系崩溃的路径**
房贷机构（发放次贷）→ 投行（打包成 MBS 和 CDO）→ 全球投资者持有 → 房价下跌 → MBS 价值暴跌 → 银行资产负债表崩溃 → 银行间市场冻结 → 信贷渠道中断 → 实体经济崩溃。

这个案例展示了金融体系各环节如何通过证券化和杠杆高度联结，系统性风险如何快速传播。

**例 2：一级市场 vs. 二级市场**
某公司 IPO（首次公开募股）是一级市场行为：公司发行新股，向公众募集资金，资金流入公司。上市后，投资者之间买卖这只股票，是二级市场行为：资金在投资者之间转移，公司不再受益。很多人误以为"买股票"是在给公司钱，实际上二级市场交易中钱在投资者之间流转，与公司无关。

---

## 7. 常见误区

**误区 1：买股票就是给企业钱**
只有在 IPO（一级市场）时，资金才流入企业。二级市场的股票交易是投资者之间的转让，企业不参与，也不从中获益。

**误区 2：金融市场规模越大越好**
金融市场有适度规模的经济效用，但过度金融化会导致实体经济资源被金融业抽走、系统性风险积累。金融应该服务于实体经济，而非凌驾其上。

**误区 3：金融中介只是"中间商"**
银行不只是把储户的钱搬给借款人，它们通过信用创造实际上扩大了货币供给（见《货币与信用》）。金融中介是货币创造的主体，而不只是资金的搬运工。

## 10. 一页总结

**金融体系功能**：资金融通、风险分配、流动性提供、价格发现、信息传递。

**两种融资方式**：直接融资（企业→资本市场→投资者）；间接融资（储蓄者→银行→企业）。

**市场分类**：货币市场（短期）vs. 资本市场（长期）；一级市场（发行）vs. 二级市场（交易）；交易所 vs. OTC。

**主要参与者**：商业银行（存贷）、投行（承销）、保险/养老（长期）、基金（集合理财）、央行（政策）。

**核心风险**：系统性风险——金融体系内部的连锁崩溃，比单一资产违约危害更大。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 金融体系 | 连接储蓄者和借款人的机构、市场、工具的集合 |
| 金融中介 | 汇聚分散储蓄并贷出的机构（银行、保险等）|
| 直接融资 | 企业直接向投资者发行股票或债券筹资 |
| 间接融资 | 储蓄者通过银行等中介间接资助借款人 |
| 货币市场 | 短期（< 1 年）借贷市场 |
| 资本市场 | 长期融资市场（股票、长期债券）|
| 一级市场 | 证券首次发行，资金流入发行人 |
| 二级市场 | 已发行证券在投资者间交易 |
| 系统性风险 | 金融体系内连锁崩溃的风险，单个节点失败引发整体危机 |
| 场外市场（OTC） | 不在交易所集中的双边协商交易市场 |
