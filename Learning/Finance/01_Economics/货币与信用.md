# 货币与信用

## 1. 主题定位

这篇讲的是货币是什么、信用是什么，以及两者如何共同构成现代金融体系的底层运作机制。

它解决的问题是：钱从哪里来？银行如何创造货币？信用扩张和收缩如何影响整个经济？为什么"印钱"不一定带来通胀，而信贷崩溃却可以引发大萧条？

在整个知识体系中，货币与信用是理解利率机制、经济周期、货币政策、债券市场的前置框架。几乎所有的宏观经济现象，背后都有货币与信用的影子。

学它之前最好先懂：供给与需求的基本概念，时间价值与贴现（理解利率的含义）。

学完后可以继续学：利率机制、通货膨胀、货币政策、经济周期、债券基础、流动性陷阱。

## 2. 核心概念

**货币（Money）**
货币是被广泛接受的交换媒介，同时也是价值储存工具和记账单位。

货币的三大功能：
- 交换媒介：用货币换商品，代替以物易物
- 价值储存：今天的购买力可以延迟到未来使用
- 记账单位：用统一尺度衡量所有商品和服务的价值

直觉：货币本身没有内在价值（纸币本身的纸不值钱），它之所以有价值，是因为所有人都相信它有价值，并且愿意接受它。这是一种基于信任的社会协议。

**信用（Credit）**
信用是借用未来购买力用于今天消费或投资的能力。贷款即信用的创造：借款人今天获得了购买力，承诺未来偿还。

容易混淆的点：货币不等于信用。货币是一种支付手段；信用是一种借贷关系。两者密切相关但不相同——现代经济中大部分"钱"实际上是信用（债务），而不是实物货币。

**货币供应量（Money Supply）**
市场上流通的货币总量。通常有不同的口径：
- M0：流通中的现金（纸币 + 硬币）
- M1：M0 + 活期存款
- M2：M1 + 定期存款、货币市场基金等

`中国语境提醒：中国央行（中国人民银行）使用 M0、M1、M2 口径，其中 M2 是最常被引用的广义货币供应量指标。`

**基础货币（Base Money / High-Powered Money）**
中央银行创造的货币，包括流通中的现金和商业银行存放在央行的准备金。它是整个信用体系的基础。

**信贷（Credit / Lending）**
商业银行向借款人发放贷款，本质上是银行在借款人的账户里记入一笔数字——这笔钱在信贷发生之前并不存在。这就是"银行创造货币"的机制来源。

## 3. 核心机制

**商业银行如何创造货币（信用创造）**

现代银行体系采用"部分准备金制度"：银行只需保留一小部分存款作为准备金，其余可以用于发放贷款。

过程如下：
1. 张三在银行存入 1000 元
2. 银行保留 100 元作为法定准备金（假设准备金率 10%），贷出 900 元给李四
3. 李四将 900 元存入另一家银行
4. 那家银行再保留 90 元，贷出 810 元
5. 如此循环……

最终，1000 元基础货币可以在整个银行体系中创造出约 10000 元的存款（货币乘数 = 1 / 准备金率）。

**关键含义**：大部分流通中的"钱"不是央行印出来的，而是商业银行通过发放贷款创造的。信贷扩张 = 货币增加；信贷收缩 = 货币减少。

**信用周期（Credit Cycle）**

信用有自我强化和自我反转的特性：

扩张期：
- 经济向好 → 借款人信用质量提升 → 银行愿意放贷 → 更多投资和消费 → 经济进一步向好 → 资产价格上涨 → 抵押品价值上升 → 银行更愿意放贷（正反馈）

收缩期：
- 经济放缓 → 借款人违约增加 → 银行收紧放贷 → 投资和消费减少 → 资产价格下跌 → 抵押品缩水 → 银行进一步收紧 → 去杠杆（负反馈）

信用收缩（去杠杆）是历史上最严重的经济危机（如 1929 年大萧条、2008 年金融危机）的核心机制。

**央行的角色：控制基础货币，影响信用成本**

央行通过调节基准利率（影响借贷成本）和调节基础货币（公开市场操作、量化宽松）来影响整个信用体系。但央行只能影响信用的"价格"和"数量"，无法直接控制商业银行的放贷行为。

`量化视角：信用利差（Credit Spread）是量化研究中常用的宏观因子，信用利差扩大往往预示经济下行风险上升。`

## 4. 分析框架

**判断货币与信用环境的框架：**

第一步：判断信用是在扩张还是收缩
- 信贷增速是否加快？（M2 增速、社会融资规模）
- 银行放贷标准是否收紧？
- 信用利差是否扩大？

第二步：判断货币供应量与实体经济是否匹配
- 货币增速是否超过经济增速？（超额货币可能导致通胀或资产泡沫）
- 货币速度（Money Velocity）是否下降？（货币速度下降意味着相同货币量创造的经济活动减少）

第三步：判断信用质量
- 借款人的债务偿还能力如何？
- 债务占收入的比例（债务收入比）是否过高？
- 抵押品价值是否稳定？

第四步：判断政策方向
- 央行是否在收紧还是放松？
- 财政是否在扩张还是收缩？
- 政策对信用扩张的支持力度如何？

## 5. 关键指标或关键变量

**M2 增速**
反映什么：广义货币供应量的增长速度，衡量信用扩张的整体力度。
容易误判：M2 高增速不一定带来通胀——如果货币流通速度同步下降（如存款不被使用），实际通胀压力可能有限。

**社会融资规模（中国特有）**
反映什么：实体经济从金融体系获得的资金总量，比单纯的银行贷款更全面。
容易误判：社融数据有季节性波动（月初月末差异大），需要看同比增速而非单月绝对值。

`中国语境提醒：社会融资规模是中国人民银行使用的统计口径，覆盖银行贷款、债券、股票融资等，比 M2 更全面反映实体融资情况。海外市场没有直接对应指标。`

**信用利差（Credit Spread）**
反映什么：企业债与同期国债的收益率差，衡量市场对信用风险的定价。
容易误判：信用利差反映的是市场情绪和预期，有时会超前或滞后于实际经济状况。

**货币乘数**
反映什么：基础货币扩张为广义货币的倍数。
容易误判：货币乘数在实践中不稳定，当银行不愿放贷（如信用危机时），乘数会大幅下降，央行增加基础货币的效果有限。

**债务收入比（Debt-to-Income Ratio）**
反映什么：借款人债务相对收入的比例，衡量信用风险和还款压力。
容易误判：债务收入比本身不是危机的触发因素，但高比例会放大外部冲击的影响。

## 6. 实际例子

**例 1：2008 年金融危机——信用收缩的极端案例**
2007-2008 年，美国房价开始下跌。以房产作为抵押品的贷款大量违约，银行资产负债表出现损失。银行开始恐慌性地收缩放贷，信用市场冻结。这导致企业无法获得正常经营所需的短期信贷，大量企业倒闭，失业率飙升。

这个例子说明：信用收缩的速度远快于信用扩张，而且它的破坏力是非线性的——不是"少贷了一点钱"，而是整个信用机制的自我强化崩溃。

**例 2：量化宽松（QE）——央行增加基础货币的局限**
2008 年后，美联储通过购买国债和抵押贷款支持证券（MBS）大幅扩大了基础货币规模。但这些钱大部分以超额准备金的形式留在银行体系，并没有通过贷款流入实体经济——信用乘数下降了。因此，QE 对通胀的推动比很多人预期的要小。这说明：央行扩大基础货币 ≠ 社会总信用扩张。

**例 3：日本的"流动性陷阱"**
1990 年代泡沫破裂后，日本长期维持接近零的利率，央行也持续放松货币，但企业和家庭仍然不愿借贷——债务收缩期中，即使钱很便宜，借款人也在努力还债而非新增债务。这种现象被称为"流动性陷阱"，是信用机制失效的典型案例。

## 7. 常见误区

**误区 1：认为"央行印钱"一定导致通胀**
货币供应量增加导致通胀，需要一个前提：这些钱真的在实体经济中流转。如果银行将准备金留存不贷，或借款人还债而非消费，货币不一定转化为通胀压力。通胀的发生取决于货币量与货币流通速度的乘积，不是单纯货币量。

**误区 2：认为信用扩张必然是好事**
信用扩张在扩张期能刺激经济，但它同时积累了未来的还款压力。借得越多，未来去杠杆的压力越大，危机时的冲击也越剧烈。

**误区 3：混淆"钱多"和"流动性充裕"**
市场上"钱多"不等于流动性充裕。流动性指的是资产能以合理价格快速变现的能力，以及参与者愿意提供资金的意愿。信用危机中，即使货币供应量未减少，流动性也可能骤然枯竭——因为没有人愿意放贷。

**误区 4：认为货币是政府"凭空创造"的**
现代货币体系中，大部分货币不是政府印出来的，而是商业银行通过放贷创造的。政府（央行）只直接控制基础货币；广义货币的创造依赖整个银行体系的行为。

**误区 5：用单一指标（如 M2）判断货币环境**
货币环境的判断需要综合多个维度：货币供应量、信用增速、利率水平、信用利差、货币流通速度。单看 M2 可能错过信用质量恶化或流通速度下降等关键信号。

## 10. 一页总结

**货币的本质**：被社会广泛接受的交换媒介，建立在信任基础上，本身没有内在价值。

**信用创造机制**：商业银行通过发放贷款创造货币，大部分流通中的"钱"是信用（债务），而非央行直接发行的货币。

**信用周期**：信用具有自我强化特性——扩张时正反馈加速增长，收缩时负反馈加速崩溃。信用崩溃是历史上最严重经济危机的核心机制。

**央行的能力边界**：央行可以调节基础货币和利率，但无法直接控制商业银行的放贷意愿和实体经济对信用的需求。"印钱"不等于通胀，更不等于经济复苏。

**核心指标**：M2 增速、社融规模、信用利差、货币乘数、债务收入比。
