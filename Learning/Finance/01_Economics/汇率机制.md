# 汇率机制

## 1. 主题定位

这篇讲的是汇率是什么、由什么决定、为什么会波动，以及汇率变动对企业、投资和经济的影响。

它解决的问题是：为什么人民币对美元的汇率会变动？汇率影响什么资产？跨境投资时需要考虑哪些汇率风险？

在整个知识体系中，汇率是全球化投资的基础概念，是理解新兴市场风险、跨境资产配置、大宗商品定价的重要变量。

学它之前最好先懂：供给与需求（汇率是货币的价格）、利率机制（利率差是汇率的重要驱动因素）。

学完后可以继续学：中国市场与海外市场差异概览、固收在资产配置中的作用、跨境资产配置方法。

## 2. 核心概念

**汇率（Exchange Rate）**
两种货币之间的兑换比率。如"1 美元 = 7.2 人民币"，表示用 7.2 元人民币可以换 1 美元。

**直接报价 vs. 间接报价**
- 直接报价：用本国货币报 1 单位外币的价格（如中国：1 美元 = 7.2 元）
- 间接报价：用外币报 1 单位本国货币的价格（如英国：1 英镑 = 1.26 美元）

**升值 vs. 贬值**
- 本币升值：更少本币换 1 单位外币（直接报价数字变小）
- 本币贬值：更多本币换 1 单位外币（直接报价数字变大）

**名义汇率 vs. 实际汇率**
- 名义汇率：市场上观察到的汇率
- 实际汇率：调整两国通胀差异后的汇率，反映实际购买力和国际竞争力

实际汇率下降（实际贬值）→ 本国商品相对便宜 → 出口竞争力上升。

**固定汇率 vs. 浮动汇率**
- 固定汇率：政府维持汇率在特定水平或区间（需要消耗外汇储备）
- 浮动汇率：汇率完全由市场供需决定
- 管理浮动（中国现行制度）：有弹性，但央行会干预以防止过度波动

`中国语境提醒：人民币目前实行"以市场供求为基础、参考一篮子货币调节、有管理的浮动汇率制度"。中国人民银行每日公布中间价，实际汇率在中间价 ±2% 范围内波动。人民币存在一定的资本管制，跨境资本流动受限，这与完全自由浮动货币不同。`

## 3. 核心机制

**汇率的主要决定因素**

**1. 利率差异（最重要的短中期驱动）**
高利率货币对资本有吸引力 → 资本流入 → 该货币升值。

利率平价理论：两国利率差 ≈ 预期汇率变动幅度。高利率货币往往预期未来贬值来抵消利差（未覆盖利率平价），但短期内高利率货币往往仍表现较强。

**2. 通货膨胀差异（购买力平价，长期因素）**
购买力平价（Purchasing Power Parity, PPP）：长期来看，汇率应该使相同商品在不同国家的价格相等。高通胀国家的货币应该贬值，来抵消更快上涨的价格水平。

PPP 对短中期汇率预测能力很差，但对判断货币的长期高估/低估有一定参考价值。

**3. 经常账户余额**
经常账户顺差（出口 > 进口）→ 外国人需要购买本国货币来支付进口 → 本币升值压力。
经常账户逆差 → 本国需要购买外币支付进口 → 本币贬值压力。

**4. 资本账户与资本流动**
外资大量流入股市、债市 → 换购本币 → 本币升值。
外资撤退 → 卖出本币 → 本币贬值。

**汇率对经济和资产的影响**

本币贬值：
- 出口商受益（产品在海外更便宜）
- 进口商受损（进口成本上升）
- 进口通胀压力上升
- 持有外币资产的投资者，以本币计算收益上升

本币升值：方向相反。

## 4. 分析框架

**分析汇率走势的框架：**

短期（数周至数月）：
- 利率差异和货币政策预期（最重要）
- 市场情绪和风险偏好（避险时资金流入美元、日元、瑞郎）
- 短期资本流动

中期（数月至 1-2 年）：
- 经常账户动态
- 货币政策路径（加息周期中货币倾向升值）
- 政治稳定性和主权信用

长期（3 年以上）：
- 购买力平价（通胀差异）
- 经济相对生产力
- 人口结构和储蓄率

**跨境投资的汇率风险评估：**

第一步：识别敞口（持有外币资产的金额和期限）
第二步：判断汇率风险方向（本币升值还是贬值对我有利）
第三步：决定是否对冲（对冲有成本，等于两国利率差）
第四步：评估对冲成本是否合理

## 5. 关键指标或关键变量

**名义有效汇率（NEER）和实际有效汇率（REER）**
反映什么：一国货币相对一篮子贸易伙伴货币的综合强弱，比单一汇率对更全面反映货币竞争力。
容易误判：双边汇率（如人民币对美元）不等于综合竞争力，REER 才是衡量实际竞争力的指标。

**外汇储备规模**
反映什么：政府干预汇率和应对外部冲击的能力。外储大幅减少可能是货币贬值压力的信号。
容易误判：外储减少不一定是危机信号，也可能是央行主动使用储备来干预或企业正常的外债偿还。

**利率差（Interest Rate Differential）**
反映什么：两国政策利率或债券收益率之差，是短期资本流动和汇率走势的重要信号。
容易误判：高利率差吸引套利交易（Carry Trade），但当风险偏好逆转时，套利头寸大规模平仓会引发汇率剧烈波动。

## 6. 实际例子

**例 1：2022 年美元强势周期**
美联储快速加息 → 美国利率显著高于其他主要经济体 → 资本大规模流向美元资产 → 美元指数大幅升值（2022 年升值超过 15%）→ 新兴市场货币普遍贬值，资本外流压力上升，外债负担加重。

这个例子展示了利率差如何驱动短中期资本流动，进而影响汇率。

**例 2：人民币对出口企业的影响**
假设人民币兑美元升值 5%：一家以美元定价出口的企业，相同美元收入换回的人民币减少 5%，利润率受压。若产品价格弹性低，企业可能被迫涨价，影响竞争力。反之，人民币贬值对出口企业是利好。

**例 3：套利交易的崩溃**
2007-2008 年，大量投资者借入低利率日元（日元利率接近 0），转换为高利率货币资产（如澳元、新兴市场货币）套利。金融危机爆发后，风险厌恶上升，投资者急速平仓 → 大量买回日元 → 日元急剧升值（"日元套利交易崩溃"）。这说明套利交易积累时，汇率存在突然逆转的风险。

## 7. 常见误区

**误区 1：汇率贬值一定是坏事**
对本国出口商是利好；对进口商是坏事；对持有外债的企业是负担。汇率变动没有普遍的好坏，取决于谁的视角和具体情境。

**误区 2：购买力平价可以预测短期汇率**
PPP 是长期均衡的参考，在短期（1-2 年内）汇率可以大幅偏离 PPP，资本流动和利率差在短期更重要。不能用"人民币被低估"直接推断短期会升值。

**误区 3：忽视对冲成本**
海外投资者对冲汇率风险需要支付成本（约等于两国利率差）。高利差环境下，对冲成本可能吃掉债券收益的大半，使"高收益"债券实际回报平庸。

**误区 4：认为央行可以无限干预维持汇率**
固定汇率需要持续消耗外汇储备。当市场贬值压力极大时，外储可以被耗尽（如 1997 年亚洲金融危机）。外汇储备不是无限的防火墙。

## 9. 最小可执行练习

**练习 1（汇率方向判断）**
以下事件对人民币汇率（对美元）有什么影响？分析方向和理由：
- 中国出口大幅增加，贸易顺差扩大
- 美联储加息，中国央行维持利率不变
- 中国通胀率远高于美国
- 外资大量流入中国股市

**练习 2（对汇率敏感性分析）**
你持有一只美国科技股 ETF（以美元计价）。如果人民币升值 10%，你以人民币计算的投资收益会如何变化？如果美股本身涨了 15%，净收益是多少？

**练习 3（PPP 判断）**
某发展中国家 A 国，通胀率长期比美国高 5%/年，但名义汇率只微幅贬值 1%/年。从购买力平价角度看，A 国货币是被高估还是低估？长期来看会如何调整？

**练习 4（套利交易风险）**
描述一个外汇套利交易的构建方式（借入低利率货币，投入高利率货币）。这个策略在什么情况下会产生损失？哪类市场事件会触发大规模平仓？

**练习 5（跨境投资分析）**
你考虑投资日本股市 ETF，日本利率接近 0，中国利率 3%，人民币兑日元可能升值。分析：这笔投资的汇率风险来自哪里？对冲成本大概是多少？不对冲时你的最大风险场景是什么？

## 10. 一页总结

**汇率本质**：两种货币的相对价格，由供需决定，受利率差、通胀差、资本流动、央行政策影响。

**核心驱动因素**（重要性从高到低）：
1. 利率差（短中期）
2. 资本账户流动（短中期）
3. 经常账户（中期）
4. 通胀差/购买力平价（长期）

**汇率变动影响**：本币贬值利好出口商、推高进口通胀、增加外币资产本币收益；本币升值反向。

**跨境投资核心关注**：汇率敞口大小 + 对冲成本（≈ 利率差）+ 汇率风险方向。

**核心陷阱**：贬值 ≠ 坏事（视角不同）；PPP 无法预测短期；套利交易集中时汇率可以突然逆转；固定汇率体制在危机时有崩溃风险。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 汇率（Exchange Rate） | 两种货币之间的兑换比率 |
| 名义汇率 | 市场观察到的汇率，未调整通胀 |
| 实际汇率 | 调整两国通胀差异后的汇率，反映实际竞争力 |
| 购买力平价（PPP） | 长期均衡理论：汇率应使相同商品跨国价格相等 |
| 利率平价 | 利率差应等于预期汇率变动，否则存在套利机会 |
| 经常账户 | 记录商品、服务、收入的跨境流动，顺差支持本币 |
| 资本账户 | 记录资本的跨境流动（外资买股票、债券等） |
| 套利交易（Carry Trade）| 借入低利率货币，投资高利率货币资产，赚取利差 |
| NEER / REER | 名义/实际有效汇率：对一篮子货币的综合汇率指标 |
| 管理浮动 | 允许汇率在一定范围内浮动，但央行保留干预权利 |
