# 经济学十大核心概念

## 1. 主题定位

这篇是 `01_Economics` 模块的总纲，提炼出理解整个经济学体系最不可或缺的十个核心概念。

它的目标是：在进入具体经济学主题之前，建立一个思考经济问题的基本语言和直觉框架。

在整个知识体系中，这篇是经济学模块的"地图"，帮助把后续的具体主题（供需、通胀、周期、政策等）定位到更大的框架中。

学它之前：没有特别要求，但有一定生活常识即可。

学完后可以继续学：供给与需求、机会成本与边际思维，以及 01_Economics 的全部内容。

## 2. 核心概念

以下十个概念，是理解经济学推理的基础语言。

---

**概念 1：稀缺性（Scarcity）**

资源是有限的，欲望是无限的。经济学的根本问题是：如何在有限资源中做出选择。

直觉：你的时间、金钱、注意力都是稀缺的。稀缺性迫使做出权衡（取舍）。

**概念 2：机会成本（Opportunity Cost）**

做一个选择，意味着放弃另一个选择的最优方案。放弃的那个最优选择，就是你的机会成本。

直觉：拿出 100 万买房，机会成本是这 100 万如果用于其他投资本可以获得的回报。

→ 详见：`00_Foundations/机会成本与边际思维.md`

**概念 3：边际分析（Marginal Analysis）**

决策不是全有全无，而是每次多做一点是否值得（边际收益 vs. 边际成本）。最优决策在边际收益 = 边际成本处。

直觉：不问"总体好不好"，问"再多一点，值不值"。

**概念 4：激励（Incentives）**

人们对激励做出反应。好的政策或制度设计，需要先理解它创造了什么激励。

直觉：提高销售提成 → 员工工作更努力；提高污染罚款 → 企业减少排放。激励设计不当会带来意外后果（如医院按手术数量付费 → 过度手术）。

**概念 5：贸易与比较优势（Trade and Comparative Advantage）**

即使一方在所有方面都不如另一方，两者仍可以通过专业化和贸易实现共赢——只要各自生产相对擅长的东西。

直觉：比较优势，而非绝对优势，决定了专业化分工的方向。国际贸易创造价值，不是零和博弈。

**概念 6：价格机制（Price Mechanism）**

价格是传递信息和协调行动的机制。价格上涨→告诉生产者"多来"；价格下降→告诉消费者"多买"。没有中央协调，数百万人的行为通过价格信号自发协调。

**概念 7：效率 vs. 公平（Efficiency vs. Equity）**

经济效率（最大化总产出，消除浪费）和公平（合理分配）往往存在权衡。两者没有天然的优先级，是价值判断，不是纯经济问题。

直觉：累进所得税增加公平，但可能降低工作激励（效率代价）；完全自由市场可能效率高，但分配不均。

**概念 8：外部性与市场失灵（Externality and Market Failure）**

当行为的成本或收益由未参与交易的第三方承担，市场无法有效配置资源（负外部性 = 污染；正外部性 = 教育）。这是政府干预市场的主要理由之一。

**概念 9：时间偏好与贴现（Time Preference and Discounting）**

人们通常更偏好"现在"而非"未来"。现在的 100 元比未来的 100 元更有价值（时间价值）。这个时间偏好率是利率的重要组成部分。

→ 详见：`00_Foundations/时间价值与贴现.md`

**概念 10：供给与需求（Supply and Demand）**

市场中所有价格和数量，最终由供给（生产者）和需求（消费者）共同决定。这是经济学分析的核心框架，适用于商品、服务、劳动力、资本、货币等所有市场。

→ 详见：`01_Economics/供给与需求.md`

## 3. 核心机制

**为什么这十个概念如此重要？**

这十个概念不是孤立的知识点，而是互相支撑的推理体系：

- 稀缺性 → 机会成本 → 边际分析（如何做选择）
- 激励 → 价格机制（如何协调行动）
- 外部性 → 市场失灵（市场何时需要修正）
- 时间偏好 → 利率 → 资产定价（跨期资源配置）
- 比较优势 → 贸易 → 国际市场分析

几乎任何经济现象，都可以用这十个概念的某种组合来解释和分析。

## 4. 分析框架

**用经济学思维分析任何问题的通用流程：**

1. **谁在做决策？他们的激励是什么？**（激励分析）
2. **资源是稀缺的，权衡是什么？**（机会成本）
3. **额外多做一点，是否值得？**（边际分析）
4. **价格在传递什么信号？**（价格机制）
5. **是否存在外部性导致市场失灵？**（市场效率）
6. **时间因素如何影响决策？**（时间偏好）
7. **效率和公平如何权衡？**（政策判断）

---

## 5. 关键现象与观察点

**激励设计失败的典型现象**
当激励与目标错位时，会产生意外的负面效果（如业绩考核导致短视行为、贷款员佣金制导致次贷危机）。识别激励是分析经济政策和企业行为的第一步。

**价格管制的代价**
当政府强制设定低于市场均衡的价格（如限价）时，必然出现短缺（需求量 > 供给量）；设定高于均衡的价格（如价格下限），出现过剩。这是价格机制被扭曲的直接后果。

**外部性的广泛存在**
碳排放（负外部性，全球变暖代价由所有人承担）、教育（正外部性，受益社会整体）、疫苗（正外部性）。外部性是理解大量市场失灵和政府政策的钥匙。

## 6. 实际例子

**例 1：比较优势与分工**
中国在劳动密集型制造业有比较优势，美国在高科技和金融有比较优势。即使美国在所有行业都比中国效率更高（假设），双方专业化分工和贸易仍可使双方受益——这是比较优势原理的核心。

**例 2：激励扭曲——医疗系统**
按诊疗次数付费的医疗体系，给医生创造了多开检查、多次复诊的激励，即使并非患者最优选择。激励设计直接决定行为结果，独立于人的道德水平。

**例 3：外部性——碳排放市场**
碳排放税或碳交易市场，是将碳排放这一负外部性"内部化"到企业成本中的政策工具。通过价格机制（碳价），将市场无法自行计价的社会成本纳入企业决策。

## 7. 常见误区

**误区 1：经济学认为人是纯理性的**
标准经济模型确实假设"理性人"，但这只是简化的起点。行为经济学已经大量记录了人类的系统性偏差。现代经济学已经大量吸收了心理学的发现。

**误区 2：自由市场一定优于政府干预**
自由市场在没有外部性、信息对称的理想条件下效率最高。但现实中有大量外部性和信息不对称，政府干预在这些领域有正当性。问题不是"市场 vs. 政府"，而是"什么情境下各自更优"。

**误区 3：GDP 增长等于社会福利提升**
GDP 是产出的测量，不直接衡量分配公平、环境质量、时间压力、幸福感。GDP 高增长可以同时伴随收入不平等加剧和环境恶化。

**误区 4：经济学结论是普遍适用的**
经济学结论往往依赖假设（如竞争市场、理性人、信息对称）。当现实不满足这些假设时，结论需要修正。不存在"放之四海而皆准"的经济学结论。

## 9. 最小可执行练习

**练习 1（概念应用）**
选择你身边一个经济现象（如某商品涨价、某政策出台），用十大概念中的 3 个来分析它。

**练习 2（激励识别）**
分析以下制度各自创造了什么激励，可能带来什么意外后果：
- 销售人员底薪制 vs. 纯提成制
- 学生按排名获得奖学金
- 医院按诊疗次数获得报销

**练习 3（外部性识别）**
列出 3 个正外部性和 3 个负外部性的例子，对每个例子说明：市场是否能自行解决？如果不能，需要什么干预？

**练习 4（机会成本计算）**
你现在可以用 3 年时间去读一个 MBA（学费 30 万 + 放弃薪资 30 万/年），也可以继续工作。分析读 MBA 的真实经济成本（会计成本 + 机会成本），以及什么情况下读 MBA 是合算的。

**练习 5（效率 vs. 公平）**
富人税（对高收入征收更高税率）能增加公平，但可能减少工作激励（效率损失）。你认为在判断一个社会应该采用多高的边际税率时，哪些因素最重要？

## 10. 一页总结

**十大核心概念速查：**

| 概念 | 核心问题 |
|------|----------|
| 稀缺性 | 资源有限，欲望无限，必须选择 |
| 机会成本 | 任何选择都有代价：放弃的最优替代 |
| 边际分析 | 多做一点，值不值 |
| 激励 | 人对激励做出反应，设计激励即设计行为 |
| 比较优势 | 专业化分工，各做相对擅长的，创造共赢 |
| 价格机制 | 价格是稀缺信息的信号，协调亿万人的行为 |
| 效率 vs. 公平 | 两者存在权衡，无纯经济答案，需价值判断 |
| 外部性 | 行为成本/收益由第三方承担，市场失灵的根源 |
| 时间偏好 | 现在的钱比未来更值钱，利率的来源 |
| 供给与需求 | 一切价格和数量的决定机制 |

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 稀缺性 | 资源有限、欲望无限的经济学基本事实 |
| 机会成本 | 做出选择而放弃的下一最优方案的价值 |
| 边际分析 | 评估额外一单位投入带来的额外收益与成本 |
| 激励 | 影响人做出特定行为的奖罚机制 |
| 比较优势 | 生产某商品的机会成本相对于他人更低的优势 |
| 外部性 | 行为对未参与交易的第三方产生的成本或收益 |
| 时间偏好 | 人们偏好现在消费胜于未来消费的倾向 |
| 效率 | 以最小成本获得最大产出，消除资源浪费 |
| 公平 | 社会资源的分配是否符合某种公正标准（有多种定义）|
