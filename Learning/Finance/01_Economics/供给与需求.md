# 供给与需求

## 1. 主题定位

这篇讲的是经济学最基础的分析框架：供给与需求如何共同决定价格和数量。

它解决的问题是：价格为什么会变动？市场如何协调买卖双方？一个事件（政策、技术、自然灾害）会如何传导到价格和产量？

在整个知识体系中，供需框架是理解任何市场（商品、股票、债券、劳动力、外汇）价格变动的底层工具。行业分析、商业模式评估、大宗商品投资，都需要这个框架。

学它之前最好先懂：机会成本与边际思维（理解需求和供给背后的边际逻辑）。

学完后可以继续学：市场效率与市场失灵、行业研究方法、商品市场分析。

## 2. 核心概念

**需求（Demand）**
在其他条件不变的情况下，消费者在不同价格下愿意且能够购买的数量。

核心规律：**需求定律**——价格越高，需求量越低（大多数商品适用）。

直觉：苹果涨价，你买少一点；苹果降价，你买多一点。

**供给（Supply）**
在其他条件不变的情况下，生产者在不同价格下愿意且能够出售的数量。

核心规律：**供给定律**——价格越高，供给量越大（大多数情况下适用）。

直觉：苹果价格高，果农愿意多种苹果；价格低，有些果农放弃种植。

**均衡（Equilibrium）**
供给量等于需求量时的价格和数量。这是市场自发达到的稳定状态。

直觉：价格高于均衡 → 供过于求 → 价格下跌；价格低于均衡 → 供不应求 → 价格上涨。市场压力推动价格趋向均衡。

**需求曲线的移动 vs. 沿曲线移动**
价格变化 → 沿同一条需求曲线移动（需求量变化）
其他因素变化（收入、替代品价格、偏好）→ 整条需求曲线移动（需求变化）

这个区分非常重要，混淆它会导致分析出错。

## 3. 核心机制

**价格的信号功能**

价格是市场传递稀缺信息的机制：
- 价格上涨 → 信号"这里短缺，多来生产"→ 供给增加、需求减少 → 价格趋于平衡
- 价格下跌 → 信号"这里过剩，减少生产"→ 供给减少、需求增加 → 价格趋于平衡

这个自我纠正机制，是市场经济效率的核心。政府若人为压制价格（如价格管制），会破坏这个信号，导致持续短缺或过剩。

**需求曲线移动的驱动因素**
- 消费者收入变化（收入上升 → 正常商品需求增加）
- 替代品价格变化（替代品涨价 → 本品需求增加）
- 互补品价格变化（互补品涨价 → 本品需求减少）
- 消费者偏好变化
- 预期变化（预期未来涨价 → 今天多买）

**供给曲线移动的驱动因素**
- 生产成本变化（原料涨价 → 供给减少）
- 技术进步（效率提升 → 供给增加）
- 生产者数量变化
- 政策变化（补贴 → 供给增加；税收 → 供给减少）
- 预期变化

**价格弹性（Price Elasticity）**
需求量对价格变化的敏感程度。
- 弹性大（富有弹性）：价格小幅变化 → 需求量大幅变化（如奢侈品、非必需品）
- 弹性小（缺乏弹性）：价格大幅变化 → 需求量变化不大（如医药品、必需品）

弹性的投资含义：生产缺乏弹性商品（如石油、粮食）的企业，在供给收缩时定价能力强，利润空间大。

## 4. 分析框架

**用供需框架分析任何价格变动的步骤：**

第一步：确认是需求变化还是供给变化（还是两者都有）
事件影响的是买方还是卖方行为？

第二步：判断变化方向（增加 or 减少）

第三步：画出新均衡
- 需求增加（需求曲线右移）→ 价格上升、数量增加
- 需求减少（需求曲线左移）→ 价格下降、数量减少
- 供给增加（供给曲线右移）→ 价格下降、数量增加
- 供给减少（供给曲线左移）→ 价格上升、数量减少

第四步：考虑短期和长期的差异
短期内供给往往弹性较小（新增产能需要时间），因此供给冲击在短期内对价格影响更大；长期内供给会调整。

## 5. 关键指标或关键变量

**价格弹性系数**
反映什么：需求量对价格变化的敏感度，= 需求量变化% / 价格变化%
容易误判：弹性系数会随价格水平变化，不是常数。需在特定价格区间讨论。

**供需缺口**
反映什么：某时点上供给量与需求量的差值。持续的供需缺口会推动价格持续变动。
容易误判：供需缺口数据往往有滞后，且受库存调节，价格不一定立即反应。

**库存水平**
反映什么：库存是短期供需缺口的缓冲器。高库存压低价格；低库存放大价格波动。
容易误判：库存数据也有统计滞后，且部分行业的"战略库存"行为会影响解读。

## 6. 实际例子

**例 1：技术进步带来的供给冲击**
2010 年代，页岩油技术突破大幅提升了美国石油供给能力。供给曲线右移 → 均衡价格下降 → 国际油价大幅下跌（2014-2016 年油价从 100 美元以上跌至 30 美元以下）。

这个例子展示了供给侧变化对价格的重大影响，以及价格信号如何吸引更多供给的正反馈。

**例 2：需求侧冲击——疫情对旅游业**
2020 年新冠疫情造成旅行需求骤降。需求曲线大幅左移 → 均衡价格下跌、数量大幅减少 → 航空公司、酒店收入崩溃。这是纯粹的需求侧冲击导致行业困境的例子。

**例 3：弹性差异的行业影响**
粮食是缺乏弹性的商品（涨价时你还是得吃）。当粮食供给因旱灾减少时，价格大幅上涨，但需求量减少幅度小。结果是：生产者总收入反而上升（价格上涨幅度大于数量减少幅度）。这解释了为什么农业生产的丰收有时反而让农民收入下降（产量增加但价格暴跌，总收入减少）。

## 7. 常见误区

**误区 1：混淆"需求量变化"和"需求变化"**
价格上涨导致需求量下降（沿曲线移动），不是需求曲线本身移动。只有非价格因素变化才导致整条需求曲线移动。这个区分在分析中非常重要。

**误区 2：忽视时间维度**
供给对价格的响应有时滞。短期内供给弹性小（新产能无法立即建成），长期弹性大。所以同一个需求冲击，短期和长期的价格效应截然不同。

**误区 3：认为均衡是立刻达到的**
市场向均衡调整需要时间，有时会超调（价格超过均衡）再回调。特别是流动性差的市场，价格发现过程可能很慢。

**误区 4：忽视市场外的力量**
政策管制、卡特尔（如 OPEC）、网络效应等会阻止市场达到自由竞争均衡。在这类市场中，供需分析需要修正。

## 9. 最小可执行练习

**练习 1（框架应用）**
用供需框架分析：如果政府对某商品征收消费税，均衡价格和数量会如何变化？税收负担在买卖双方之间如何分担？（提示：弹性决定分担比例）

**练习 2（需求变化识别）**
以下事件各会如何影响电动汽车的供需？分类为供给变化 / 需求变化，并判断对价格的影响方向：
- 电池原材料价格上涨
- 政府发放购车补贴
- 消费者对气候变化的担忧增加
- 新的电池生产技术突破

**练习 3（弹性判断）**
以下商品的需求弹性更大还是更小？给出理由：胰岛素、名牌手包、自来水、高端餐厅服务。弹性差异对这些行业的定价权意味着什么？

**练习 4（行业分析应用）**
选择一个你感兴趣的行业（如半导体、猪肉、外卖），分析其供给端和需求端的主要驱动因素，判断当前供需格局，并推断价格压力的方向。

**练习 5（长短期对比）**
油价上涨后，短期内对石油公司利润的影响 vs. 长期影响有什么不同？（提示：短期供给弹性小，长期供给会响应）

## 10. 一页总结

**核心规律**：需求定律（价格↑→需求量↓），供给定律（价格↑→供给量↑），均衡是两者的交汇点。

**价格的功能**：传递稀缺信息，协调买卖双方行为，引导资源流向高价值用途。

**分析步骤**：判断是供给还是需求变化 → 判断移动方向 → 推断新均衡价格和数量 → 区分短期和长期。

**关键概念**：价格弹性（商品必需性影响定价权）；供需缺口（库存是缓冲）；短期 vs. 长期调整（时间改变弹性）。

**核心陷阱**：混淆需求量变化和需求变化；忽视时间维度；忽视政策和市场结构对自由均衡的干预。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 需求（Demand） | 消费者在各价格水平下愿意且能够购买的数量关系 |
| 供给（Supply） | 生产者在各价格水平下愿意且能够出售的数量关系 |
| 均衡（Equilibrium） | 供给量等于需求量时的价格与数量组合 |
| 需求定律 | 其他条件不变时，价格越高，需求量越低 |
| 价格弹性 | 需求量对价格变化的敏感程度，= 需求量变化% / 价格变化% |
| 富有弹性 | 价格小变化 → 需求量大变化（如奢侈品） |
| 缺乏弹性 | 价格大变化 → 需求量小变化（如必需品、医药） |
| 供需缺口 | 某时点供给量与需求量之差，是价格压力的来源 |
