# 经济周期

## 1. 主题定位

这篇讲的是经济为什么会周期性地扩张与收缩，以及这个周期的结构、驱动因素和对投资的含义。

它解决的问题是：经济不是匀速增长的，它有高峰和低谷，周期性波动是正常现象。理解经济周期，是做宏观判断、配置资产的基础框架之一。

在整个知识体系中，经济周期是连接宏观经济学与资产配置的桥梁。理解周期在哪个阶段，能帮助理解为什么某类资产当前表现好或差。

学它之前最好先懂：货币与信用（信用周期是经济周期的重要驱动）、供给与需求（基础市场逻辑）。

学完后可以继续学：货币政策、财政政策、通货膨胀、资产配置框架（不同周期阶段配置不同资产）、量化中的宏观因子。

---

## 2. 核心概念

**经济周期（Business Cycle）**
经济总产出（GDP）围绕长期趋势的周期性波动。经济不是直线增长的，而是在扩张与收缩之间交替。

直觉：把 GDP 想象成一条往右上方延伸的趋势线，但实际 GDP 会在这条线上下波动——有时超过趋势（繁荣），有时低于趋势（衰退）。经济周期就是这个波动的规律。

**经济周期的四个阶段**

| 阶段 | 特征 |
|------|------|
| 复苏（Recovery） | GDP 从低点回升，失业率开始下降，产能利用率低 |
| 扩张（Expansion） | GDP 持续增长，就业充分，消费和投资旺盛 |
| 过热（Peak/Overheating） | 增长速度开始放缓，通胀压力上升，产能接近极限 |
| 衰退（Recession / Contraction） | GDP 持续下降，失业上升，消费投资萎缩 |

容易混淆的点：经济"放缓"≠ 衰退。放缓是增速下降（但仍为正增长）；衰退通常指 GDP 连续两个季度负增长。

**衰退（Recession）**
通俗定义：GDP 连续两个季度负增长。
更准确的定义：GDP、就业、生产、收入多个指标同时出现显著和持续的下降（美国经济研究局 NBER 使用此定义）。

**产出缺口（Output Gap）**
实际 GDP 与潜在 GDP 之差。正的产出缺口（实际 > 潜在）意味着经济过热，通胀压力大；负的产出缺口（实际 < 潜在）意味着经济低迷，存在闲置产能。

## 3. 核心机制

**为什么经济会有周期？**

经济周期没有单一成因，而是多种力量共同作用的结果：

**1. 信用周期的驱动**
信用扩张时，投资和消费被放大；信用收缩时，支出骤减。信用本身的自我强化特性（见《货币与信用》）是短期经济周期波动的重要驱动力。

**2. 投资的波动性**
企业投资（建厂、买设备）对经济预期极度敏感。预期好时大量投资 → 需求上升 → 经济扩张；预期变差时投资迅速削减 → 需求骤降 → 经济收缩。投资的这种"超调"（过度扩张和过度收缩）放大了周期波动。

**3. 库存调整**
企业维持库存存在滞后性。需求下降时，企业先动用库存，之后才削减生产。需求回升时，先去化库存，之后才增加生产。这种库存调整制造了短期的产出波动（通常 2-4 年的"库存周期"）。

**4. 政策的顺周期性**
历史上，货币政策往往在繁荣期加息太晚（助长泡沫），在衰退期降息太晚（加深衰退）。政策滞后本身会放大周期波动。

**周期的嵌套结构**

经济周期并非单一时间尺度，而是多个周期相互嵌套：

```
库存周期（Kitchin Cycle）：约 3-4 年，由库存调整驱动
设备投资周期（Juglar Cycle）：约 7-11 年，由资本设备投资驱动
房地产周期（Kuznets Cycle）：约 15-25 年，由建设投资驱动
长波周期（Kondratiev Cycle）：约 45-60 年，由技术革命驱动
```

短期内主导感受的通常是库存周期和设备投资周期的叠加。

`量化视角：宏观周期因子是量化研究中重要的系统性因子，库存周期指标（PMI、工业产出）常被用于宏观择时模型中，但其预测效果存在较大争议。`

## 4. 分析框架

**判断经济周期位置的框架（领先、同步、滞后指标体系）**

经济指标分三类，对应不同的预测功能：

**领先指标（Leading Indicators）**——先于经济转折点变化
- 制造业 PMI（采购经理人指数）：PMI > 50 为扩张，< 50 为收缩
- 新屋开工数
- 股票市场（有时）
- 利率期限结构（长短期利率之差，即收益率曲线斜率）
- 信贷标准松紧

**同步指标（Coincident Indicators）**——与经济同步变化
- GDP
- 工业产出
- 就业人数
- 零售销售额

**滞后指标（Lagging Indicators）**——滞后于经济转折点变化
- 失业率（失业率往往在经济已经复苏后才开始下降）
- 通货膨胀率（价格变化往往在需求变化后才反映出来）
- 企业贷款余额

**实操框架：判断当前周期位置的五步法**

1. 看 GDP 增速趋势（加速 / 减速 / 负增长）
2. 看 PMI 绝对水平和变化方向
3. 看就业市场（就业是否充分？工资是否上涨？）
4. 看通胀水平（通胀上行意味着接近过热）
5. 看政策方向（央行在加息还是降息？财政在扩张还是收缩？）

## 5. 关键指标或关键变量

**GDP 增速**
反映什么：经济总产出的增长速度，是判断扩张 / 收缩最直接的指标。
容易误判：GDP 是季度数据，发布有滞后，且会被修订。用 GDP 判断当前状态存在时滞，需配合领先指标。

**PMI（采购经理人指数）**
反映什么：制造业或服务业的扩张与收缩信号，月度发布，是最常用的领先指标之一。
容易误判：PMI 反映的是"变化方向"，不是绝对水平。即使 PMI 从 45 升到 48，仍低于 50，代表仍在收缩，只是收缩速度在放慢。

**收益率曲线斜率（长短利率之差）**
反映什么：长期与短期利率之差。曲线倒挂（短期利率 > 长期利率）历史上是衰退的领先信号。
容易误判：收益率曲线倒挂预测衰退的成功率较高，但时间提前量不稳定（可能提前 6 个月至 2 年），不能作为精确择时工具。

**失业率**
反映什么：劳动市场的松紧，是最重要的滞后指标。
容易误判：失业率是滞后指标，经济已经在复苏时，失业率可能还在上升。在经济周期转折点附近，失业率的判断方向往往与实际情况相反。

**通货膨胀率（CPI/PPI）**
反映什么：价格水平的变化，反映需求强弱和成本压力。
容易误判：通胀是滞后指标，且受到供给侧冲击（如大宗商品价格）的影响，不能单独用通胀判断周期位置。

`中国语境提醒：中国使用官方 PMI（国家统计局）和财新 PMI 两套数据，覆盖样本略有不同，有时方向不一致。社会融资规模、工业增加值、货运量等中国特有指标也常被用于判断经济周期位置。`

## 6. 实际例子

**例 1：2008-2009 年全球金融危机——衰退的典型路径**
信用危机（银行收紧放贷）→ 企业投资崩溃 → 消费下降 → GDP 大幅负增长 → 大规模裁员 → 失业率飙升（滞后反应）→ 消费进一步萎缩（正反馈恶化）→ 央行降息至零 + 量化宽松 → 2009 年中开始复苏。

这个例子展示了从过热到衰退、再到复苏的完整路径，也展示了各指标的顺序：PMI 先跌、GDP 随后负增长、失业率最后才达到峰值。

**例 2：PMI 的领先作用**
PMI 通常在 GDP 转折点之前 1-3 个月发出信号。2015 年底-2016 年初，中国制造业 PMI 连续数月低于 50，此后工业产出和 GDP 增速相继放缓，印证了 PMI 的领先性。

这个例子说明：PMI 不是完美的预测工具，但它的月度高频发布使它在领先指标中具有较高的实用价值。

**例 3：收益率曲线倒挂预测衰退**
美国历史上，10年期国债收益率与2年期国债收益率倒挂（前者低于后者），在过去数十年多次先于衰退出现，被广泛视为衰退的"预警信号"。但倒挂出现后，到实际衰退发生的时间差异较大，不能作为精确择时依据。

## 7. 常见误区

**误区 1：认为经济周期是固定时长的**
经济周期的长度高度不规则，受政策、技术冲击、外部事件等多重因素影响。不存在"每 X 年一次衰退"的机械规律。

**误区 2：认为可以精确判断周期位置**
经济数据有滞后，且经常被修订。即便是专业机构，在周期转折点附近的判断也经常出错。周期分析提供的是概率框架，不是确定性预测。

**误区 3：把"经济放缓"理解为"熊市开始"**
经济放缓不等于衰退，也不等于市场立即下跌。资产市场通常会领先于经济数据反应，在经济还在扩张时市场可能已在下跌，在衰退已经发生时市场可能已在反弹。

**误区 4：只关注本国经济周期，忽视全球联动**
在高度全球化的今天，主要经济体（美国、欧洲、中国）的周期相互影响。单看本国数据可能错过来自外部的重要信号。

**误区 5：认为政策可以消除经济周期**
政策可以减缓衰退深度、加速复苏，但历史上没有任何政策体系实现了完全平滑的经济增长。过于激进的刺激政策往往推迟了必要的调整，积累了更大的下一次风险。

## 9. 最小可执行练习

**练习 1（框架应用）**
查找你所在经济体最新一期的 PMI 数据、GDP 增速和失业率。根据这三个数据，判断当前大约处于经济周期的哪个阶段，并写出你的判断依据。

**练习 2（指标分类）**
将以下指标分类为"领先、同步、滞后"：工业产出、新屋开工、失业率、零售销售额、企业债违约率、PMI。给出每个分类的理由。

**练习 3（机制推导）**
解释"库存周期"的完整运作机制：从需求下降开始，企业如何调整库存，这如何导致产出的短期波动，最后如何恢复均衡？用自己的语言写出这个因果链。

**练习 4（历史案例）**
选择任意一次你了解的经济衰退（如 2008 年金融危机、2020 年疫情冲击），找出当时领先指标、同步指标、滞后指标各自在什么时间发出了什么信号。对照本篇的理论框架，哪些地方吻合，哪些地方有出入？

**练习 5（误区识别）**
一位投资者说："现在 GDP 还在增长，所以不用担心熊市。" 请指出这个逻辑的错误，并解释为什么资产市场可能领先于经济数据转向。

## 10. 一页总结

**核心事实**：经济围绕长期趋势周期性波动，分为复苏、扩张、过热、衰退四个阶段，长度不固定。

**核心驱动**：信用周期、投资波动、库存调整、政策滞后——四者叠加，无单一原因。

**分析框架**：领先指标（PMI、收益率曲线）→ 同步指标（GDP、就业）→ 滞后指标（失业率、通胀），三类指标结合使用。

**关键规律**：
- PMI < 50 预示收缩
- 收益率曲线倒挂是历史上较可靠的衰退领先信号
- 失业率是滞后指标，在复苏初期仍会上升
- 市场通常领先经济数据 6-12 个月反应

**核心陷阱**：周期长度不规律、精确择时极难、经济放缓 ≠ 熊市开始、单一指标判断不可靠。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 经济周期（Business Cycle） | 经济总产出围绕长期趋势的周期性扩张与收缩 |
| 衰退（Recession） | GDP 连续两个季度负增长（通俗定义），或多项经济指标同时显著下降 |
| 产出缺口（Output Gap） | 实际 GDP 与潜在 GDP 之差，正值为过热，负值为低迷 |
| PMI（采购经理人指数） | 衡量制造业 / 服务业活动的月度指标，> 50 为扩张，< 50 为收缩 |
| 领先指标 | 先于经济转折点变化的指标，如 PMI、新屋开工 |
| 同步指标 | 与经济同步变化的指标，如 GDP、就业人数 |
| 滞后指标 | 滞后于经济转折点变化的指标，如失业率、通胀 |
| 收益率曲线倒挂 | 短期利率高于长期利率的现象，历史上是衰退的领先信号 |
| 库存周期（Kitchin Cycle） | 约 3-4 年的短期经济波动，由企业库存调整驱动 |
| 潜在 GDP | 在充分就业和正常产能利用率下，经济能够持续生产的最大产出 |
