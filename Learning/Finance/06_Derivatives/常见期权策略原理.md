# 常见期权策略原理

## 1. 主题定位

这篇讲的是最常用的期权组合策略的原理、构建方式和适用场景，侧重机制理解，不侧重具体交易技术。

学它之前最好先懂：期权基础（四种基本头寸）、对冲与投机。

学完后可以继续学：衍生品风险（策略的风险特征）、可转债（含权债券类比）。

---

## 2. 核心概念

期权策略 = 组合多个期权（或期权 + 现货）头寸，构建特定的收益/损失图谱。

不同策略回答不同的市场判断：
- 方向性判断（涨/跌）→ 单腿策略
- 波动率判断（大波动/横盘）→ 跨式/蝶式等
- 风险控制 → 价差策略

---

## 3. 核心机制

**策略一：保护性认沽（Protective Put）**
构建：持有股票 + 买入 Put（同标的）
目的：持有上涨潜力的同时，给组合买"下跌保险"
特征：
- 上涨：股票涨，Put 归零（损失权利金）
- 大幅下跌：Put 盈利，弥补股票损失

**策略二：备兑期权（Covered Call）**
构建：持有股票 + 卖出 Call（同标的）
目的：在横盘或小幅上涨时，通过收取权利金增强收益
特征：
- 收取权利金，增加持股的被动收益
- 股价大涨时，Call 被行权，上涨收益被截断（上行受限）
- 本质：用"潜在超额上行"换取"确定的权利金收入"

**策略三：牛市价差（Bull Call Spread）**
构建：买入低行权价 Call + 卖出高行权价 Call（同到期日）
目的：降低纯买 Call 的成本，适合温和上涨预期
特征：
- 成本 < 直接买 Call（卖出的 Call 收入部分抵消）
- 上行空间有限（最大收益 = 高低行权价差 - 净权利金）
- 最大损失 = 净权利金

**策略四：熊市价差（Bear Put Spread）**
构建：买入高行权价 Put + 卖出低行权价 Put（同到期日）
目的：降低纯买 Put 成本，适合温和下跌预期
特征：对称于牛市价差，方向相反

**策略五：跨式策略（Straddle）**
构建：买入相同行权价的 Call + Put（同到期日）
目的：押注标的大幅波动（方向不确定）
特征：
- 标的大涨或大跌都能获利
- 横盘时两腿都可能亏损（需支付两份权利金）
- 适合在重大事件（财报、政策）前，预期结果大幅偏离市场预期

**策略六：宽跨式（Strangle）**
构建：买入虚值 Put + 买入虚值 Call（同到期日，行权价不同）
目的：与跨式类似，但权利金更低（虚值期权便宜）
特征：需要更大的价格移动才能盈利，但成本低于跨式

## 4. 分析框架

**根据市场判断选择策略：**

| 市场判断 | 推荐策略 |
|---------|---------|
| 大幅上涨，愿意付权利金 | Long Call |
| 温和上涨，降低成本 | Bull Call Spread |
| 大幅下跌，寻求保护 | Long Put / Protective Put |
| 横盘，想赚权利金 | Covered Call / Short Strangle |
| 大幅波动，方向不确定 | Long Straddle / Long Strangle |
| 横盘或小幅波动，做波动率卖方 | Short Straddle / Short Strangle（高风险）|

---

## 5. 关键现象与观察点

**卖出期权的风险非对称性**
跨式/宽跨式的卖方（Short Straddle/Strangle）：收取两份权利金，但一旦出现大幅波动，损失可以极大。这是"收敛策略"——小概率的极端事件可以在一次内损失远超历史所有积累的权利金。

**波动率压缩（Volatility Crush）**
在重大事件前（如财报），隐含波动率（IV）通常大幅上升（期权昂贵）。事件公布后，不确定性消除，IV 迅速下降——即使标的价格按预期方向移动，也可能导致期权价格下降（买方亏损）。这是"买预期、卖事实"中期权特有的风险。

## 6. 实际例子

**例 1：财报前的跨式策略**
某公司即将发布季报，历史上财报后经常出现 ±15% 的大幅波动。当前股价 100 元，买入：Call（行权价 100，权利金 5）+ Put（行权价 100，权利金 4），总成本 9 元。

盈亏平衡：股价需涨超 109 或跌破 91 才能盈利。若财报后股价变动 ±15%，涨到 115 或跌到 85，均可盈利。

风险：若财报平淡，股价不动，两份权利金全损。

**例 2：Covered Call 增强持股收益**
持有某蓝筹股，预期短期内不超过 10% 的上涨。卖出 Call（行权价 = 当前价 × 1.1），收取 2% 权利金。

若股票涨幅 < 10%：保留股票 + 额外收 2% 权利金。
若股票大涨超 10%：股票被对方行权买走，错失超过 10% 的上涨，但已赚到行权价 + 权利金。
若股票下跌：权利金作为缓冲（但不完全弥补下跌）。

## 7. 常见误区

**误区 1：期权组合策略可以消除所有风险**
任何期权策略都有其"盲区"——特定市场情景下会亏损。理解策略的收益图谱（最大收益、最大损失、盈亏平衡点）是使用策略的基础。

**误区 2：Straddle 买方一定赚，因为方向不重要**
Straddle 买方需要实际波动率超过隐含波动率（IV）才能盈利。如果市场已经把大幅波动的预期"定价进去"了（IV 很高），即使真的发生波动，也可能不够大。

**误区 3：卖出期权是低风险策略（因为收了权利金）**
卖出裸 Call 的风险是无限的，卖出裸 Put 的风险极大（标的跌至 0 时）。Short Straddle/Strangle 的最大风险也是极端市场下的巨额损失。"稳定收取权利金"的背后是隐性的尾部风险。

## 9. 最小可执行练习

**练习 1（收益图谱）**
构建 Bull Call Spread：买入行权价 100 的 Call（权利金 8 元），卖出行权价 110 的 Call（权利金 3 元）。计算：净成本、最大损失、最大收益、盈亏平衡点。在到期时标的分别为 90、100、105、110、120 时，各自的盈亏。

**练习 2（策略选择）**
根据以下市场判断，选择最合适的期权策略并解释：
① 预期大幅上涨，不确定幅度，愿意付出有限代价
② 持有某股票，预期短期内小幅震荡不涨，想额外增收
③ 某公司财报即将公布，预期结果将大幅偏离预期，但方向不确定

**练习 3（波动率压缩）**
财报前 1 周，某股票 IV = 60%（期权很贵），你买入跨式策略，总成本 12 元（行权价 = 股价 100 元）。财报公布后，股票按预期上涨 10% 到 110 元，但 IV 立刻下降到 20%。期权理论价值变化如何？你是否能盈利？

---

## 10. 一页总结

**六大策略定位：**
- Protective Put：持股 + 下行保险
- Covered Call：持股 + 收取权利金（限制上行）
- Bull/Bear Spread：降成本的方向性押注，有限收益和损失
- Long Straddle/Strangle：押注大幅波动（方向不确定），需超过 IV 的实际波动
- Short Straddle/Strangle：做波动率卖方，收权利金，有极端尾部风险

**核心判断逻辑**：先判断方向和波动率预期，再选匹配的策略；了解每种策略的最大损失和盈亏平衡点。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| Protective Put | 持有股票 + 买 Put，为下行风险提供保护 |
| Covered Call | 持有股票 + 卖 Call，收取权利金，上行空间受限 |
| Bull Call Spread | 买低行权价 Call + 卖高行权价 Call，温和看涨 |
| Long Straddle | 买同行权价 Call + Put，押注大幅波动 |
| Strangle | 买虚值 Call + Put，比 Straddle 便宜但需更大波动 |
| 波动率压缩（Volatility Crush）| 事件后 IV 骤降，即使方向正确期权也可能亏损 |
| 盈亏平衡点（Breakeven）| 期权策略开始盈利的标的价格点 |
