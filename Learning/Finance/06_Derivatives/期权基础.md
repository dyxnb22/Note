# 期权基础

## 1. 主题定位

这篇讲的是期权的基本结构、核心术语、收益图谱，以及期权与期货的本质区别。

学它之前最好先懂：期货基础（理解衍生品的基本逻辑）、时间价值与贴现。

学完后可以继续学：常见期权策略原理、衍生品风险、可转债（含权债券的期权属性）。

---

## 2. 核心概念

**期权（Option）**
赋予买方在特定期间内，以约定价格（行权价）买入或卖出标的资产的权利（不是义务）。

**期权与期货的最大区别**：
- 期货：双方都有义务（多头必须买，空头必须卖）
- 期权：买方有权利但无义务；卖方有义务无权利

**看涨期权（Call Option）**
赋予买方以约定价格购买标的物的权利。预期标的价格上涨时买入 Call。

**看跌期权（Put Option）**
赋予买方以约定价格卖出标的物的权利。预期标的价格下跌时买入 Put。

**行权价（Strike Price / Exercise Price）**
买方有权以此价格买入（Call）或卖出（Put）标的物。

**到期日（Expiration Date）**
期权权利的有效期限。到期后，期权自动失效（虚值）或自动行权（实值）。

**权利金（Premium）**
期权买方付给卖方的费用，是购买"权利"的成本，也是期权卖方的最大收益。

**内在价值 vs. 时间价值**

`期权价格 = 内在价值 + 时间价值`

- 内在价值：期权立即行权可获得的价值（Call：max(S-K, 0)；Put：max(K-S, 0)）
- 时间价值：剩余到期时间和波动率带来的额外价值

**实值、平值、虚值**
- 实值（In-the-Money）：立即行权有利（Call：S > K；Put：S < K）
- 平值（At-the-Money）：S ≈ K
- 虚值（Out-of-the-Money）：立即行权无利，但有时间价值

## 3. 核心机制

**期权的四种基本头寸**

| 头寸 | 权利/义务 | 最大收益 | 最大损失 | 适用场景 |
|------|---------|---------|---------|---------|
| 买入 Call（Long Call）| 权利 | 无限 | 权利金 | 预期大幅上涨 |
| 卖出 Call（Short Call）| 义务 | 权利金 | 无限 | 预期横盘或下跌 |
| 买入 Put（Long Put）| 权利 | 行权价 - 权利金 | 权利金 | 预期大幅下跌或对冲 |
| 卖出 Put（Short Put）| 义务 | 权利金 | 行权价 - 权利金 | 预期横盘或上涨 |

**期权的不对称性**

买方：最大损失 = 权利金（有限），最大收益理论上无限（Call）或至标的跌至 0（Put）。

卖方：最大收益 = 权利金（有限），最大损失可以非常大（甚至无限）。

这种不对称性是期权最重要的特征。

**期权的六大影响因素（Greeks 概念简介）**

1. 标的价格（S）：Call 正向，Put 负向（Delta）
2. 行权价（K）：Call 负向，Put 正向
3. 到期时间（T）：通常越长，期权越贵（Theta：时间价值随时间流逝衰减）
4. 波动率（σ）：波动率越高，期权越贵（Vega），因为有利方向的收益无上限
5. 无风险利率（r）：利率越高，Call 越贵，Put 越便宜
6. 股息：除息会降低标的价格，影响期权价值

## 4. 分析框架

**选择期权策略的基本思路：**

1. 判断方向（涨/跌/横盘）
2. 判断波动率预期（大幅波动 or 平稳）
3. 判断时间期限（短期 or 长期事件驱动）
4. 综合考虑：买方（有限损失）vs. 卖方（收权利金但风险大）

**简单规则：**
- 预期大幅上涨 + 愿意付权利金：Long Call
- 预期大幅下跌 + 愿意付权利金：Long Put
- 预期小幅上涨或横盘 + 愿意收权利金并承担风险：Short Put
- 持有股票，想有保护：Long Put（保护性认沽）

## 5. 关键指标或关键变量

**Delta（δ）**
反映什么：标的价格变化 1 元，期权价格的变化量。Call 的 Delta 在 0 到 1 之间，Put 的 Delta 在 -1 到 0 之间。
容易误判：Delta 不是固定的，会随标的价格变化（Gamma 效应）。

**隐含波动率（Implied Volatility, IV）**
反映什么：从当前期权市场价格反推的市场对未来波动率的预期，是衡量期权"贵贱"的核心指标。
容易误判：IV 高不一定意味着买期权亏（高 IV 可能低于实际波动率），需要与历史波动率比较。

**Theta（时间衰减）**
反映什么：每过一天，期权时间价值损失多少（对买方不利）。
容易误判：Theta 在临近到期时加速衰减，持有到期日前的期权最后几天时间价值快速消失。

## 6. 实际例子

**例 1：Long Call 的收益图谱**
标的价格 100 元，行权价 105 元，权利金 3 元。
- 到期时标的 < 105：期权不行权，损失全部权利金 3 元
- 到期时标的 = 108：行权，获利 108-105 = 3 元，刚好弥补权利金，盈亏平衡
- 到期时标的 = 120：行权，获利 120-105-3 = 12 元

最大损失：3 元（有限）；最大收益：理论无限。

**例 2：Long Put 作为对冲**
持有价值 100 万元的股票组合，担心下跌。购买 Put 期权（行权价 = 当前价格，权利金 2%）。
若股市下跌 20%：Put 收益约 18%（扣除权利金），对冲了大部分损失。
若股市上涨：Put 权利金损失 2%，但股票涨幅弥补。

这是"组合保险"策略，相当于给股票组合买了一份保险。

**例 3：波动率对期权定价的影响**
同一标的，行权价 100，到期 1 个月，无风险利率 3%：
- 历史波动率 10%：期权价格约 1.1 元
- 历史波动率 30%：期权价格约 3.3 元
波动率翻 3 倍，期权价格也大约翻 3 倍。高波动率时，期权是昂贵的工具。

## 7. 常见误区

**误区 1：期权买方亏损无限，卖方很安全**
完全相反：买方最大损失有限（权利金），卖方最大损失可以无限（裸卖 Call）。

**误区 2：期权接近到期时行权价临近最合算**
期权临近到期时，时间价值几乎全部消失（Theta 衰减），购买期权的成本大幅降低，但时机极难把握——期权可以快速归零。

**误区 3：购买 Call 不需要预判股票涨多少**
购买 Call 需要标的价格涨过行权价 + 权利金的总和才能盈利（盈亏平衡点）。预期小幅上涨不足以使买 Call 盈利。

## 9. 最小可执行练习

**练习 1（收益计算）**
买入 Call 期权：行权价 50 元，权利金 2 元，到期时标的分别为 45、50、52、55、60 元。计算每种情况的盈亏。画出收益图。

**练习 2（Put 对冲）**
持有某股票，现价 100 元。购买行权价 95 元的 Put，权利金 3 元。如果股票跌至 80 元，你的净损失是多少（股票损失 + Put 收益 - 权利金）？相比没有 Put 保护少损失多少？

**练习 3（买方 vs. 卖方）**
描述裸卖 Call（不持有标的）的最大风险场景：如果标的从 100 元涨到 200 元，卖出行权价 110 元 Call（权利金 5 元），你的损失是多少？

**练习 4（波动率判断）**
某股票即将发布重大财报，市场预期大幅波动。这时候 IV 通常较高（期权贵）还是较低（期权便宜）？发布后 IV 会如何变化？"波动率压缩"对期权买方意味着什么？

## 10. 一页总结

**Call vs. Put**：Call 是买入权利，预期涨；Put 是卖出权利，预期跌。

**买方 vs. 卖方**：
- 买方：付权利金，有权利无义务，最大损失 = 权利金，潜在收益大
- 卖方：收权利金，有义务无权利，最大收益 = 权利金，潜在损失大（甚至无限）

**期权价格组成**：内在价值（立即行权价值）+ 时间价值（随时间衰减）。

**关键驱动因素**：标的价格（Delta）+ 波动率（Vega）+ 时间（Theta）。

**核心陷阱**：买方最大损失有限、卖方风险大（非"平均赢家"）；时间衰减对买方不利；需要足够大的方向性移动才能使买 Call/Put 盈利。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 期权（Option）| 赋予买方特定权利（非义务）的金融合约 |
| Call（看涨期权）| 买入标的资产的权利 |
| Put（看跌期权）| 卖出标的资产的权利 |
| 行权价（Strike）| 期权可以买入/卖出标的的约定价格 |
| 权利金（Premium）| 买方支付给卖方的期权购买价格 |
| 内在价值 | 立即行权可获得的利润（负值取 0）|
| 时间价值 | 期权价格超过内在价值的部分，随到期临近衰减 |
| Delta | 标的价格变化 1 元，期权价格的变化量 |
| Theta | 每过一天，期权时间价值的损失量 |
| 隐含波动率（IV）| 从期权市价反推的市场预期波动率 |
