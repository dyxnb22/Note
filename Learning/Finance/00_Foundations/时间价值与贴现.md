# 时间价值与贴现

## 1. 主题定位

这篇讲的是金融学最基础的一个事实：同样一笔钱，现在拥有比未来拥有更有价值。

它解决的问题是：如何比较不同时间点上发生的现金流？如何把未来的钱换算成今天的价值？

在整个知识体系中，时间价值是几乎所有定价模型的地基。债券定价、股票估值（DCF）、期权定价、项目投资决策，背后都在用同一套时间价值逻辑。

学它之前最好先理解：基本的利率概念（知道"借钱要付利息"即可）。

学完后可以继续学：复利与对数收益率、债券基础（到期收益率）、DCF估值。

---

## 2. 核心概念

**时间价值（Time Value of Money, TVM）**
今天的 1 元钱比未来的 1 元钱更值钱，原因有三：
- 今天的钱可以投资，未来产生回报
- 通货膨胀会侵蚀购买力
- 未来的现金流存在不确定性（风险）

直觉：如果你今天借给朋友 100 元，一年后还你 100 元，你是亏的——因为这 100 元放在你手里本可以产生收益，或者一年后的 100 元购买力更低了。

**现值（Present Value, PV）**
未来某笔钱在"今天"等价多少钱。是把未来现金流"折回"现在。

**终值（Future Value, FV）**
今天的一笔钱在未来某个时间点会变成多少。是把现在的钱"推算"到未来。

**折现率（Discount Rate）**
把未来现金流折算回现值时使用的利率。它反映了你对这笔钱的"机会成本"或"要求回报率"。

容易混淆的点：折现率不等于无风险利率。折现率应该反映这笔钱对应的风险水平——风险越高，折现率越高，未来现金流的现值越低。

**贴现（Discounting）**
把未来现金流换算成现值的过程。与"复利增长"方向相反。

---

## 3. 核心机制

**从复利到贴现**

如果你今天投入 PV 元，年回报率为 r，n 年后变成：

```
FV = PV × (1 + r)^n
```

反过来，如果你知道 n 年后会收到 FV 元，那它今天值多少？

```
PV = FV / (1 + r)^n
```

这就是贴现。分母 `(1 + r)^n` 叫做**折现因子（Discount Factor）**，它永远小于 1。

**为什么折现率越高，现值越低？**

折现率高，意味着你对这笔钱要求更高的回报（或者说你有更好的投资机会，或者这笔钱的风险更高）。同样一笔未来现金流，用更高的折现率去折，分母更大，现值自然更低。

这个机制解释了一个重要规律：**当市场利率上升，所有资产（股票、债券、房产）的理论现值都会下降。**

**多期现金流的现值**

如果未来每年都有现金流（CF₁, CF₂, ... CFₙ），总现值是每一期单独折现后相加：

```
PV = CF₁/(1+r)¹ + CF₂/(1+r)² + ... + CFₙ/(1+r)ⁿ
```

这就是 DCF（Discounted Cash Flow）模型的核心计算逻辑。

**永续年金（Perpetuity）的特殊简化**

如果每年永远产生固定现金流 C，折现率为 r：

```
PV = C / r
```

这个公式极度简洁，但对 r 的选取极度敏感，使用时必须审慎。

## 4. 分析框架

使用时间价值时，核心判断步骤：

**第一步：确认现金流的时间序列**
每笔钱发生在什么时间？金额是多少？方向是流入还是流出？

**第二步：选择折现率**
折现率 = 无风险利率 + 风险溢价
- 无风险利率：通常参考国债收益率（具体期限视现金流期限而定）
- 风险溢价：视现金流的不确定性而定，风险越大溢价越高

**第三步：折现计算**
将每期现金流除以 `(1+r)^t`，求和得总现值。

**第四步：验证假设的敏感性**
折现率小幅变动会大幅改变结果。必须做敏感性分析：如果 r 从 8% 变成 10%，现值变化多大？

---

## 5. 关键指标或关键变量

**折现率 r**
反映什么：你对这笔钱的机会成本和风险要求。
容易误判的地方：很多人用同一个折现率折现所有资产，但不同资产风险不同，折现率本就应该不同。

**现金流发生的时间 t**
反映什么：时间越长，折现效果越强，不确定性越高。
容易误判的地方：很多人低估"时间"的影响。10 年后的 100 元，用 8% 折现率折回今天只值约 46 元——不到一半。

**现金流金额本身**
反映什么：被折现的基础。
容易误判的地方：未来现金流往往是预测值，不是确定值。预测的误差被折现率"放大"。

**折现因子 1/(1+r)^t**
反映什么：每一期折现的力度。
容易误判的地方：长期现金流的折现因子极小，对总现值贡献很低，因此长期现金流预测的精度对结果影响较小——这既是 DCF 的局限，也是它的特性。

## 6. 实际例子

**例 1：比较两个合同报价**
你签了一个合同，对方可以选择：今天支付 90 万元，或者 3 年后支付 110 万元。假设你能以 8% 的年化回报率投资。

3 年后 110 万的现值：`110 / (1.08)³ ≈ 87.3 万元`

结论：今天 90 万 > 87.3 万，选今天支付更合算。
这个计算告诉你：未来 110 万和今天 87.3 万在经济上等价，不是面值比大小。

**例 2：债券定价逻辑**
一张债券每年付息 5 元，3 年后还本 100 元，市场利率（折现率）为 6%。

```
PV = 5/1.06 + 5/1.06² + 105/1.06³
   ≈ 4.72 + 4.45 + 88.20
   = 97.37 元
```

债券面值 100 元，但市场利率高于票面利率，所以它的现值低于面值——这是"债券折价"的直观解释。

**例 3：利率上升对股票的冲击**
一家公司预计未来每年稳定盈利 10 元（简化为永续年金）。折现率从 8% 升到 10%：

- 折现率 8%：PV = 10 / 0.08 = 125 元
- 折现率 10%：PV = 10 / 0.10 = 100 元

利率上升 2 个百分点，理论价值下跌 20%。这是为什么"加息周期"对股票估值有压力的机制来源。

## 7. 常见误区

**误区 1：只比较名义金额，忽视时间**
"3 年后的 110 万肯定比今天的 90 万多"——这个判断忽略了折现。必须把两者折算到同一时间点再比。

**误区 2：认为折现率是一个固定常数**
折现率是你的假设，它随利率环境、风险水平、持有期不同而不同。用同一个折现率套用在所有情景下是危险的。

**误区 3：低估远期折现的力量**
30 年后的现金流，用 8% 折现，折现因子约为 0.099——也就是说，100 元只值约 10 元。很多人做长期预测时忘了远期现金流几乎可以忽略不计。

**误区 4：把贴现率当作预期回报率**
折现率是"要求回报率"（最低门槛），不是"预期实际回报率"。两者方向一致，但含义不同。

**误区 5：忽视现金流本身的不确定性**
贴现公式假设未来现金流已知，但现实中未来现金流是估计值。对折现率的过度精确计算，不能弥补现金流预测本身的粗糙。

## 9. 最小可执行练习

**练习 1（计算）**
你被承诺 5 年后收到 100 万元。假设你的机会成本（折现率）为 7%，请计算这 100 万元今天的现值。答案：约 71.3 万元。验证你的计算过程是否正确。

**练习 2（判断）**
一个项目需要今天投入 50 万元，预计 3 年后收回 65 万元。折现率为 10%。请判断这个项目是否值得投资。（提示：计算 65 万的现值与 50 万比较。）

**练习 3（分析）**
永续年金公式 PV = C/r，如果 r 从 5% 变成 4%，现值变化幅度是多少？再从 5% 变成 6%，现值变化幅度是多少？观察两者不对称性，思考为什么低利率环境下资产估值特别敏感。

**练习 4（概念应用）**
解释为什么"央行加息"会导致股票市场下跌。尝试用时间价值的语言写出完整的逻辑链，而不是仅说"加息利空股市"。

**练习 5（误区识别）**
你的朋友说："这个理财产品 5 年后给我 150 万，我现在只投 100 万，收益 50%，很划算。" 请用时间价值框架分析这个判断的问题所在。

## 10. 一页总结

**核心事实**：今天的钱比未来的钱更值钱，原因是投资机会、通胀和不确定性。

**核心公式**：
- 终值：`FV = PV × (1+r)^n`
- 现值：`PV = FV / (1+r)^n`
- 多期：`PV = Σ CFt / (1+r)^t`
- 永续：`PV = C / r`

**核心机制**：折现率越高 → 分母越大 → 现值越低。利率上升压低所有资产现值。

**核心判断**：比较不同时间点的钱，必须先折算到同一时间点，不能直接比面值。

**核心陷阱**：折现率的选取主观性很强；远期现金流折现后微乎其微；现金流本身的预测误差比折现率精度更重要。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 时间价值（TVM） | 同等金额的钱，现在拥有比未来拥有更有价值的概念 |
| 现值（PV） | 未来现金流折算到今天的等价金额 |
| 终值（FV） | 今天的钱在未来某时间点的等价金额 |
| 折现率（Discount Rate） | 将未来现金流折算为现值时使用的利率，反映机会成本和风险 |
| 贴现（Discounting） | 将未来现金流换算成现值的计算过程 |
| 折现因子 | `1/(1+r)^t`，将第 t 期现金流折回现值的乘数，恒小于 1 |
| 永续年金（Perpetuity） | 每年产生固定现金流且永不终止的资产，现值 = C/r |
| 机会成本 | 选择一件事而放弃另一件事的潜在收益，是折现率的直觉来源 |
