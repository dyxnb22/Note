# 风险与收益的基本关系

## 1. 主题定位

这篇讲的是金融学最核心的一条原则：更高的预期收益，对应更高的风险。

它解决的问题是：为什么不同资产的收益率不同？如何理解"收益"和"风险"这两个概念？如何评估一笔投资是否值得承担它的风险？

在整个知识体系中，风险与收益的关系是资产定价、组合构建、估值模型的底层逻辑。不理解这个关系，就无法正确解读任何收益率数据，也无法评价任何投资。

学它之前最好先懂：时间价值与贴现（理解折现率中包含风险溢价）。

学完后可以继续学：市场效率、资产定价模型（CAPM）、波动率与最大回撤、夏普比率、风险预算。

---

## 2. 核心概念

**收益（Return）**
投资的回报。通常用百分比表示，可以分为：
- 价格收益（资本利得）：资产价格上涨带来的回报
- 现金收益（分红、利息）：持有期间收到的现金

直觉：你今天花 100 元买了某资产，明年卖出 110 元并收到 5 元分红，总收益率 = (10 + 5) / 100 = 15%。

**风险（Risk）**
在金融语境中，风险不等于"一定会亏钱"，而是指**结果的不确定性**，包括上行不确定性和下行不确定性。

直觉：一笔投资可能赚 50% 也可能亏 30%，与另一笔稳赚 5% 的投资相比，前者风险更高——不是因为它一定亏，而是因为结果的波动幅度更大。

容易混淆的点：风险 ≠ 亏损概率。低风险资产仍然可能亏损（如通胀环境下的现金）；高风险资产也可能长期盈利。风险是波动性，不是方向。

**预期收益（Expected Return）**
各种可能结果的加权平均，权重是每种结果的概率。它是"平均意义上"的回报，而非保证回报。

**风险溢价（Risk Premium）**
投资者因为承担额外风险而要求的额外回报，等于预期收益减去无风险利率。

无风险利率（Risk-Free Rate）：通常以短期国债收益率近似，是几乎没有违约风险的回报基准。

**风险与收益的权衡（Risk-Return Tradeoff）**
理性市场中，没有人会免费承担风险。因此，风险更高的资产，必须提供更高的预期收益，才能吸引投资者持有。

## 3. 核心机制

**为什么风险越高，要求收益越高？**

假设市场上有两种资产：
- 资产 A：100% 概率获得 5% 收益（国债）
- 资产 B：50% 概率获得 20% 收益，50% 概率亏损 10%（预期收益 = 5%）

如果资产 B 的预期收益也是 5%，没有人会选它——因为在同等预期回报下，没人愿意承担波动。所以资产 B 的价格会下跌，直到它的预期收益高于 5%，形成一个"风险溢价"，才能吸引投资者。

这就是风险溢价的来源：它是市场对"承担不确定性"的补偿。

**资产类别的风险收益谱系（从低到高）**

```
现金 < 短期国债 < 长期国债 < 投资级信用债 < 高收益债 < 大盘股 < 小盘股 < 新兴市场股 < 私募股权
```

这不是一条严格的规律，而是在长期历史数据中被反复观察到的大致顺序。

**分散化可以降低部分风险，但不能消除所有风险**

风险分为两类：
- 系统性风险（市场风险）：影响所有资产的风险，如经济衰退、利率变动。无法通过分散化消除。
- 非系统性风险（个股风险）：影响单个资产的风险，如某公司管理层丑闻。可以通过持有多个不相关资产分散掉。

这是为什么分散化有意义——它消除的是非系统性风险，但系统性风险依然存在。

**市场只补偿系统性风险**

在有效市场中，投资者持有大量个股可以轻易消除非系统性风险，因此市场不会为它付出溢价。只有系统性风险（无法分散）才能获得市场的风险溢价。这是 CAPM 模型的核心逻辑。

## 4. 分析框架

**评估一项投资的风险收益框架：**

第一步：识别这项投资的收益来源
- 它的收益从哪里来？利息、分红、价格上涨？
- 这些收益来源的稳定性如何？

第二步：识别这项投资的风险来源
- 市场风险（整体行情下跌）
- 信用风险（对方无法履约）
- 流动性风险（无法及时变现）
- 通胀风险（实际购买力被侵蚀）
- 汇率风险（跨境资产）
- 操作风险（人为错误、欺诈）

第三步：判断风险溢价是否合理
- 预期收益率 - 无风险利率 = 风险溢价
- 这个溢价是否与上述风险相匹配？
- 历史上类似风险水平的资产提供了多少溢价？

第四步：评估自身对风险的承受能力
- 这不是"这项投资对不对"，而是"这项投资对我对不对"
- 时间跨度、资金用途、心理承受力都影响合适的风险水平

## 5. 关键指标或关键变量

**波动率（Volatility）**
反映什么：收益率的标准差，衡量收益的波动幅度。
容易误判：波动率是双向的，向上的波动也被计入。高波动不一定是坏事，但意味着结果更难预测。

**最大回撤（Maximum Drawdown）**
反映什么：从历史最高点到最低点的最大跌幅。
容易误判：很多人关注平均收益，忽视最大回撤。但极端的下行风险才是实际持有中最难承受的部分。

**夏普比率（Sharpe Ratio）**
反映什么：每承担一单位风险获得的超额收益，公式为 `(收益率 - 无风险利率) / 波动率`。
容易误判：夏普比率高的策略不一定安全——它可能是通过承担尾部风险（偶发的极端亏损）获得的。

**Beta（β）**
反映什么：资产相对于市场整体的波动敏感度。Beta=1 表示与市场同步，Beta>1 表示放大市场波动。
容易误判：Beta 衡量的是相对波动，不是绝对风险。低 Beta 资产在熊市中跌得少，但并非没有风险。

**无风险利率**
反映什么：基准收益，任何投资的预期回报应该超过这个数字，否则不如持有无风险资产。
容易误判：无风险利率是动态变化的，折现率、风险溢价都随之变动。当无风险利率上升，所有资产的吸引力相对下降。

## 6. 实际例子

**例 1：国债 vs. 股票的长期风险收益**
历史上，发达国家股票市场的长期年化回报（含分红）大约比同期国债高 4-6 个百分点，这个差额被称为"股权风险溢价（Equity Risk Premium）"。代价是：股市在某些年份可能下跌 30%-50%，甚至连续数年负收益，而国债几乎不会出现如此剧烈的波动。

这个例子说明：股票的超额回报不是凭空而来，它是对波动性和持有期不确定性的补偿。

**例 2：高收益债（垃圾债）的风险溢价**
信用评级低的公司发行的债券（高收益债）需要提供比国债更高的利率，差额称为"信用利差"。信用利差的大小反映市场对违约风险的判断——经济不确定时，信用利差会显著扩大。

如果某高收益债的信用利差极小（接近投资级债券），意味着市场认为它的风险很低，但实际上信用质量可能并没有提升，只是市场情绪乐观。这时候承担的风险没有得到足够补偿。

**例 3：流动性风险溢价**
私募股权基金的锁定期通常为 7-10 年，期间资金无法赎回。因此，私募基金的历史平均回报通常高于公开市场股票，一部分原因是流动性风险溢价——投资者因为放弃了灵活性而获得补偿。

这说明：收益来源不只是市场风险，流动性不足也是一种可以被定价的风险。

## 7. 常见误区

**误区 1：把高收益直接等同于好投资**
高收益可能只是对应了更高风险，不一定是"超额"回报。判断一项投资好不好，要看它的收益是否与它承担的风险匹配，即风险调整后的收益。

**误区 2：认为分散化可以消除所有风险**
分散化消除的只是非系统性风险。当整个市场下跌时（如金融危机），几乎所有资产同时下跌，分散化无济于事。

**误区 3：把历史收益当作未来收益的保证**
历史数据告诉我们某类资产"曾经"的风险收益表现，但未来的风险溢价不保证不变。过去高收益可能是因为当时估值便宜；如果现在估值已高，未来同样回报的概率降低。

**误区 4：只关注绝对收益率，不关注风险来源**
一个产品宣称年化 20%，你需要问：这 20% 是怎么来的？如果来自于极高杠杆，或者承担了低频但极端的尾部风险，那它的真实风险远高于表面上看起来。

**误区 5：将波动率等同于"危险"**
短期波动大，不代表长期一定亏损。对于长期投资者，短期波动带来的是机会（可以在低点加仓），而不仅仅是威胁。把波动率等同于风险的框架，对短期投资者适用，对长期投资者会造成误判。

## 9. 最小可执行练习

**练习 1（概念辨析）**
列出以下资产，从低到高排列它们的风险水平，并写出你判断的理由：短期国债、房产、比特币、投资级公司债、大盘指数基金。

**练习 2（计算）**
资产 A 的历史年收益率序列：+15%, -5%, +20%, +8%, -3%。计算它的平均年收益率和收益率标准差（波动率）。（可用计算器）

**练习 3（框架应用）**
某理财产品宣传"年化 12%，低风险"。请用风险收益框架分析这个说法：它的风险来源可能有哪些？为什么 12% 的收益率和"低风险"很难同时成立？

**练习 4（机制推导）**
解释为什么在一个高度有效的市场中，非系统性风险不会获得风险溢价。（提示：从"分散化可以免费消除非系统性风险"出发）

**练习 5（误区识别）**
一位朋友说："这只基金过去 5 年年化 18%，风险很低，因为它从来没有亏损过。" 请指出这个判断中至少 2 个逻辑问题。

## 10. 一页总结

**核心原则**：更高的预期收益对应更高的风险，这是理性市场的均衡结果，不是偶然。

**风险的两种类型**：
- 系统性风险：无法分散，市场给予补偿（风险溢价）
- 非系统性风险：可以通过分散化消除，市场不给额外补偿

**评估框架**：
1. 收益从哪里来？
2. 风险来源有哪些？
3. 风险溢价是否合理？
4. 这个风险水平是否适合我？

**核心指标**：波动率（收益的标准差）、最大回撤、夏普比率（风险调整后收益）、Beta（相对市场的波动敏感度）。

**核心陷阱**：高收益 ≠ 好投资；分散化 ≠ 零风险；历史收益 ≠ 未来保证；波动率 ≠ 必然亏损。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 收益率（Return） | 投资回报，通常以百分比表示，含价格变动和现金分红 |
| 风险（Risk） | 结果的不确定性，不等于亏损，包含上行和下行两个方向 |
| 预期收益（Expected Return） | 各种可能结果的概率加权平均，不是保证收益 |
| 风险溢价（Risk Premium） | 因承担额外风险而要求的超额回报，= 预期收益 - 无风险利率 |
| 无风险利率（Risk-Free Rate） | 几乎不含违约风险的回报基准，通常用短期国债收益率近似 |
| 系统性风险 | 影响整个市场的风险，无法通过分散化消除 |
| 非系统性风险 | 影响单一资产的风险，可通过持有多个不相关资产分散 |
| 波动率（Volatility） | 收益率的标准差，衡量收益的波动幅度 |
| 最大回撤 | 从历史最高点到最低点的最大跌幅 |
| 夏普比率（Sharpe Ratio） | 每单位风险获得的超额收益 = (收益率 - 无风险利率) / 波动率 |
| Beta（β） | 资产相对于市场整体的波动敏感度 |
| 股权风险溢价（ERP） | 股票相对无风险资产的长期超额回报 |
