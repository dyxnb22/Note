# 相关性与回归分析

## 1. 主题定位

这篇讲投资分析中最常用的两个统计工具：相关性分析（衡量两个变量的线性关联程度）和线性回归（估计变量之间的数量关系）。Beta 的估算、因子分析、Alpha 计算都依赖这两个工具。

学它之前最好先懂：期望方差与协方差、收益率的计算与转换。

学完后可以继续学：统计检验基础（判断回归结果是否显著）、Alpha与Beta（Beta 的统计估计）、因子投资框架（多因子回归）。

---

## 2. 核心概念

**皮尔逊相关系数（Pearson Correlation）**

```
ρ = Cov(X,Y) / (σₓ × σᵧ) ∈ [-1, 1]
```

衡量两个变量线性关联的强度和方向：+1 完全正相关，-1 完全负相关，0 线性无关。

**斯皮尔曼相关系数（Spearman Rank Correlation）**
用秩次（排名）而非原始值计算相关系数，对异常值更鲁棒。量化因子有效性评估（IC）通常使用 Spearman 相关系数。

**普通最小二乘回归（OLS，Ordinary Least Squares）**

```
Y = α + βX + ε
```

找到使残差平方和最小的 α（截距）和 β（斜率）。

---

## 3. 核心机制

**OLS 估计量**

```
β̂ = Cov(X,Y) / Var(X) = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)²

α̂ = ȳ - β̂x̄
```

Beta 系数 = X 每变动一单位，Y 的平均变动量。

**R²（决定系数）**

```
R² = 1 - SS_residual / SS_total = ESS / TSS
```

X 能解释 Y 方差的比例。R²=0.6 表示 X 解释了 Y 60% 的变化。

注意：R² 越高不代表模型越好——过多变量总能提高 R²（使用调整后 R² 惩罚过多参数）。

**回归的投资应用**

**1. Beta 估算（CAPM 回归）**
```
Rᵢ - Rf = α + β(Rm - Rf) + ε

用历史数据回归，β̂ 即股票的市场 Beta，α̂ 即 Jensen's Alpha
```

**2. 多因子归因**
```
Rᵢ = α + β₁×MKT + β₂×SMB + β₃×HML + β₄×MOM + ε
```

各 β 即对相应因子的暴露程度，α 是控制所有因子后的残差 Alpha。

**3. 因子有效性检验（截面回归）**
```
Rᵢ,t+1 = α + β × 因子得分ᵢ,t + εᵢ,t
```

β 显著为正说明因子有正向预测能力。

**相关性与因果性的区别**

```
相关性：X 和 Y 同时变动（不说明方向）
因果性：X 的变化导致 Y 的变化

两者差异来源：
  - 遗漏变量（Z 同时影响 X 和 Y，形成虚假相关）
  - 反向因果（Y 影响 X，而非 X 影响 Y）
  - 偶然巧合（数据量小时假阳性）
```

金融研究中，相关性很常见，因果性很难证明（缺乏随机实验）。

---

## 4. 分析框架

**评估一个回归结果的完整清单**

1. **系数方向**：β 的符号是否符合经济学预期？
2. **统计显著性**：t 统计量 > 2（p < 0.05）？置信区间是否不含 0？
3. **经济显著性**：效应大小是否有实际意义（β 数值是否足够大）？
4. **R²**：模型解释力如何？（单因子模型 R² 低是正常的）
5. **残差检验**：残差是否随机（无明显模式）？是否有自相关（时序数据）？
6. **稳健性**：在不同时间段、不同样本中，结论是否稳定？

---

## 5. 关键指标或关键变量

**t 统计量**

```
t = β̂ / SE(β̂)
```

衡量系数估计值相对于其标准误的倍数。|t| > 2（约对应 p < 0.05）通常认为统计显著。

**标准误（Standard Error）**
β̂ 估计量的标准差。样本量越大、X 的变动越大、残差方差越小，标准误越小，估计越精确。

**调整后 R²**

```
R²_adj = 1 - (1-R²)(n-1)/(n-k-1)
```

对参数数量 k 进行惩罚，避免因增加无用变量而虚增 R²。

## 6. 实际例子

**例 1：Beta 估算**
用过去 3 年月度数据对某股票进行 CAPM 回归：
- β̂ = 1.35（t=8.2，显著）：股票对市场波动敏感度高于平均
- α̂ = 0.002（t=0.8，不显著）：月度 Alpha 统计上与 0 无差异
- R² = 0.62：市场因子解释了该股票 62% 的收益方差

解读：这只股票是高 Beta 股，但没有显著的 Alpha，持有它只是在承担市场风险。

**例 2：虚假相关的经典案例**
研究发现"某国奶酪消费量"与"被床单缠绕死亡人数"历史相关系数高达 0.95——这是典型的遗漏变量/巧合，两者都与人口和时间趋势有关，没有任何因果关系。金融中类似的"数据挖掘"到的虚假相关比比皆是。

**例 3：多因子回归揭示 Alpha 来源**
某基金经理 10 年年化 Alpha（相对市场）= 4.2%，t=2.1，看起来显著。但加入规模因子（SMB）和价值因子（HML）后，Alpha 降至 0.8%，t=0.4，不再显著。原来的"Alpha"是规模和价值因子的 Beta 贡献，不是真实的选股能力。

---

## 7. 常见误区

**误区 1：高相关 = 一定有关系**
样本量小时（如 30 个数据点），相关系数 0.35 可能在 5% 水平下显著——但可能是统计巧合。相关性需要结合样本量和经济逻辑一起判断。

**误区 2：R² 低 = 模型没用**
单因子回归 R² = 0.05（因子只解释 5% 的方差）在量化因子研究中很正常——即使解释力很弱的因子，应用于大量股票时，统计平均后也能提供有价值的 Alpha。

**误区 3：OLS 回归的结果在时序数据中一定可靠**
OLS 假设残差独立同分布。金融时序数据中，收益率往往有自相关（动量效应）和异方差性（波动聚类），违反 OLS 假设，需要使用 Newey-West 标准误等稳健估计。

**误区 4：相关性是稳定的**
相关系数随时间变化。资产之间的相关性在市场正常时期较低，在危机期间急剧上升。历史相关系数不应被视为未来的固定参数。

## 9. 最小可执行练习

**练习 1（Beta 计算）**
某股票 5 个月收益率（超额）：+3%、-2%、+5%、-1%、+4%
市场 5 个月超额收益：+2%、-1.5%、+3%、-0.5%、+2.5%
手动计算（或用 Excel）：Beta 和截距（Alpha）。

**练习 2（因果性判断）**
以下相关关系，哪些更可能有因果性，哪些更可能是虚假相关？
① 利率上升 → 债券价格下跌 ② 冰淇淋销量 → 溺水事故增加 ③ 公司 ROE → 股票长期表现 ④ 基金规模 → 基金业绩

**练习 3（R² 解读）**
三个因子回归结果（均统计显著）：R²=0.02、R²=0.35、R²=0.70。
分别说明：① 每个 R² 在金融研究中的典型意义 ② R²=0.70 的因子是否一定比 R²=0.02 的更有用？

---

## 10. 一页总结

**相关系数**：衡量线性联动（-1 到 +1），不代表因果；危机时相关性会剧增。

**OLS 回归**：β̂ = Cov(X,Y)/Var(X)，最小化残差平方和；Alpha = 截距；Beta = 斜率。

**关键输出**：β（效应大小）+ t 统计量（显著性）+ R²（解释力）。

**核心应用**：Beta 估算（CAPM 回归）、因子归因（多因子回归）、因子有效性检验（截面回归）。

**核心陷阱**：相关性≠因果性；小样本虚假显著；金融时序数据违反 OLS 基本假设。

---

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 皮尔逊相关系数 | 两变量线性关联强度，范围 [-1,1] |
| OLS（普通最小二乘）| 最小化残差平方和的参数估计方法 |
| R²（决定系数）| 回归模型能解释的因变量方差比例 |
| t 统计量 | 系数估计值 / 标准误，衡量统计显著性 |
| 残差（ε）| 实际观测值与模型预测值的差 |
| 截距（α̂）| 自变量为 0 时因变量的预测值，多因子模型中对应 Alpha |
| 斜率（β̂）| X 每变动一单位，Y 的平均变动量 |
