# 统计检验基础

## 1. 主题定位

这篇讲假设检验的核心逻辑：p 值、显著性、置信区间，以及在金融研究中如何正确解读和避免误用统计检验。量化研究和学术论文中几乎所有结论都需要经过统计检验。

学它之前最好先懂：期望方差与协方差、相关性与回归分析。

学完后可以继续学：回测偏差（多重检验问题）、因子投资框架（因子有效性的显著性检验）。

---

## 2. 核心概念

**假设检验的框架**

```
原假设（H₀）：默认无效果（如 Alpha = 0，相关系数 = 0）
备择假设（H₁）：要证明的命题（如 Alpha ≠ 0，相关系数 > 0）

逻辑：假设 H₀ 为真，若观测数据极不可能在 H₀ 下出现，则拒绝 H₀
```

**p 值（p-value）**
在原假设为真的情况下，观测到当前结果或更极端结果的概率。

```
p < 0.05：通常认为统计显著（5% 显著性水平）
p < 0.01：更严格的显著性标准
```

注意：p 值不是"H₀ 为真的概率"，也不是"结论错误的概率"——这是最常见的误解。

**置信区间（Confidence Interval）**
95% 置信区间：若反复实验，100 次中约 95 次该区间会包含真实参数值。

```
CI = β̂ ± t_{α/2} × SE(β̂)
```

置信区间比 p 值提供更多信息（效应大小 + 不确定性范围）。

**两类错误**

| | H₀ 实际为真 | H₀ 实际为假 |
|---|------------|------------|
| 拒绝 H₀ | **一类错误（假阳性）**，概率 = α | 正确拒绝（功效）|
| 未拒绝 H₀ | 正确（置信水平）| **二类错误（假阴性）**，概率 = β |

显著性水平 α（通常 5%）= 一类错误概率。统计功效 = 1 - β = 正确发现真实效果的概率。

## 3. 核心机制

**t 检验的逻辑**

```
t 统计量 = (估计值 - 假设值) / 标准误
         = (β̂ - 0) / SE(β̂)

|t| > 2（样本量足够时）→ 近似对应 p < 0.05
|t| > 2.6 → 近似对应 p < 0.01
```

**样本量对显著性的影响**

```
SE(β̂) ∝ 1/√n

样本量 n 越大 → 标准误越小 → t 统计量越大 → 更容易显著
```

这意味着：足够大的样本下，任何微小的效应都可以达到统计显著性——即使效应小到没有实际意义。**统计显著性 ≠ 经济显著性**。

**多重检验问题（Multiple Testing Problem）**

```
单次检验：在 H₀ 为真时，错误拒绝的概率 = 5%
100 次独立检验：至少有 1 次错误拒绝的概率 ≈ 1-(0.95)^100 ≈ 99.4%
```

测试越多策略/因子，"偶然发现显著结果"的概率越高。Bonferroni 校正：将显著性阈值从 5% 降至 5%/N（N = 检验次数）。

**功效分析（Power Analysis）**
在设计实验时，确定需要多少样本量才能以足够高的概率（如 80%）发现真实效果。

```
Alpha 研究中的粗略规则：
  要发现年化 Alpha = 2%（信噪比低），需要约 10-15 年数据
  要发现年化 Alpha = 5%，需要约 3-5 年数据
```

## 4. 分析框架

**解读一个统计结果的完整框架**

1. **效应大小**：系数（β̂）是多少？在经济上有意义吗？
2. **统计显著性**：p 值或 t 统计量是多少？
3. **置信区间**：区间的宽窄如何？区间是否排除了 0？
4. **样本量**：结果是否来自足够大的样本？
5. **多重检验**：这是第几次尝试才得到的显著结果？
6. **样本外**：结果在独立样本中是否可复现？
7. **经济逻辑**：结果有合理的经济学解释吗？

---

## 5. 关键指标或关键变量

**t 统计量门槛**（投资研究中 Harvey et al. 2016 建议）
- 传统标准：|t| > 2.0（p < 0.05）
- 考虑多重检验后的建议：|t| > 3.0（p < 0.003）
- 原因：金融领域普遍存在大量测试，传统 5% 阈值太宽松

**最小可检测效应量**

```
最小可检测效应 ∝ σ / √n

用实际数据：
  月度策略，3年数据（36个月），月度Alpha标准误约 = 年化Alpha_std / √12
  需要 t > 2 才能发现，所以最小可发现的年化Alpha约 = 2 × 年化波动率 / √n
```

## 6. 实际例子

**例 1：p 值的正确解读**
某因子研究：IC 均值 = 0.04，p = 0.03。
正确解读：假设该因子完全无效（IC_真实=0），观测到 IC 均值 ≥ 0.04 的概率为 3%，这足够低，所以拒绝"因子无效"的假设。
错误解读："因子有效的概率是 97%"（错：p 值不是这个含义）。

**例 2：样本量效应**
小样本（30 只股票，3 年）：某因子 IC = 0.08，SE = 0.06，t = 1.33（不显著）。
大样本（500 只股票，10 年）：同样 IC = 0.08，SE = 0.015，t = 5.3（高度显著）。
同样的效应大小，仅因为样本量不同，结论完全不同。

**例 3：统计显著但经济不显著**
某高频策略月度 Alpha = 0.001%（0.01 基点），t = 2.5（显著）。但扣除交易成本后，Alpha = -0.02%（负）。统计上显著，但经济上毫无价值（甚至是负的）。

## 7. 常见误区

**误区 1：p < 0.05 意味着结论"很可能是真的"**
p 值是在原假设为真时观测数据的概率，不是原假设为假的概率。此外，在多重检验环境中，即使 p < 0.05，如果测试了 100 个因子，其中 5 个可能是假阳性。

**误区 2：p > 0.05 意味着"没有效果"**
不显著 ≠ 效果为零。可能是样本量不足以检测到真实存在但较小的效果（功效不足）。"无法拒绝 H₀"不等于"证明了 H₀ 成立"。

**误区 3：只看显著性，不看效应大小**
大样本下，任何微小效应都可以显著。要同时看效应大小（β）和显著性（t 值），两者都有意义才算真正有价值的发现。

**误区 4：统计检验适用于任何数据**
统计检验有前提假设（独立性、分布假设等）。金融时序数据通常存在自相关、异方差，需要调整标准误（如 Newey-West）才能使检验有效。

## 9. 最小可执行练习

**练习 1（p 值计算）**
某策略 36 个月的月度超额收益：均值 0.5%，标准差 2%。
计算：① t 统计量 ② 这是否在 5% 水平下显著？③ 这个效应大小在经济上有意义吗（年化多少）？

**练习 2（多重检验）**
研究员测试了 80 个因子，其中 7 个在 5% 水平下显著。
计算：① 如果所有因子都无效，预期有几个会"偶然"显著 ② 真正有效的因子可能有几个 ③ 应该用什么更严格的显著性标准？

**练习 3（样本量需求）**
某策略年化 Alpha = 3%，年化跟踪误差 = 8%（对应月度约 2.3%）。
估算：需要多少月的数据才能在 5% 水平下发现这个 Alpha？这在实践中意味着什么挑战？

---

## 10. 一页总结

**假设检验逻辑**：假设 H₀ 为真，若观测极不可能，则拒绝 H₀。p 值 = 极端结果在 H₀ 下的概率，不是"结论错误的概率"。

**t 统计量**：系数 / 标准误，|t| > 2 约对应 p < 0.05；金融研究中建议用 |t| > 3 的更严格标准。

**核心警示**：统计显著性 ≠ 经济显著性；大样本下微小效应也会显著；多重检验会导致假阳性爆炸。

**置信区间**比 p 值更有信息量——给出效应大小的范围，而非只给出二元的"显著/不显著"。

---

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 原假设（H₀）| 默认命题，通常是"无效果"（如 Alpha = 0）|
| p 值 | 在 H₀ 为真时，观测到当前或更极端结果的概率 |
| 统计显著性 | p < α（通常 5%），表示在该置信水平下拒绝 H₀ |
| 一类错误 | 错误拒绝真实的 H₀（假阳性），概率 = α |
| 二类错误 | 错误未能拒绝错误的 H₀（假阴性），概率 = β |
| 统计功效 | 1-β，正确发现真实效果的概率 |
| 多重检验问题 | 大量检验导致假阳性概率急剧上升 |
