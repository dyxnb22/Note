# 常用概率分布

## 1. 主题定位

这篇讲投资分析中常用的几个概率分布：正态分布、对数正态分布、厚尾分布，以及为什么正态分布假设在金融中会失效。理解这些分布是风险建模的基础。

学它之前最好先懂：期望方差与协方差、收益率的计算与转换。

学完后可以继续学：统计检验基础（检验方法依赖分布假设）、波动率与最大回撤（极端事件的分布建模）。

---

## 2. 核心概念

**正态分布（Normal Distribution）**

```
X ~ N(μ, σ²)
f(x) = (1/√(2πσ²)) × exp(-(x-μ)²/(2σ²))
```

由均值 μ 和标准差 σ 完全确定，关于均值对称，钟形曲线。

**对数正态分布（Lognormal Distribution）**
若 ln(X) ~ N(μ, σ²)，则 X 服从对数正态分布。价格通常被假设服从对数正态分布（价格不能为负，且百分比变动的波动相对稳定）。

**厚尾分布（Fat-Tailed Distribution）**
尾部概率远高于正态分布的分布。金融资产收益率在极端事件（大跌/大涨）的发生频率远高于正态分布预测。

**峰度（Kurtosis）**
描述分布尾部厚度的统计量。正态分布峰度 = 3（超额峰度 = 0）。金融收益率通常超额峰度 > 0（尖峰厚尾）。

---

## 3. 核心机制

**正态分布的关键性质**

```
68-95-99.7 规则：
  ±1σ 内：约 68% 的概率
  ±2σ 内：约 95% 的概率
  ±3σ 内：约 99.7% 的概率
  ±4σ 以外：约 0.006%（约 16000 交易日一遇，60 年一遇）
```

**正态分布在金融中的局限**

| 正态假设下的预测 | 现实观察 |
|---------------|---------|
| 3σ 事件：约 60 年一遇 | 实际：每几年就发生 |
| 5σ 事件：几乎不可能 | 1987、2008 等危机均超过 5σ |
| 尾部风险低 | 实际尾部风险被严重低估 |

金融收益率的实际分布特征：**尖峰厚尾（Leptokurtosis）**——中间比正态分布更集中（高峰），两端比正态分布更厚（肥尾）。

**四个描述分布形状的统计量**

| 统计量 | 公式 | 含义 |
|--------|------|------|
| 均值（μ）| E[X] | 中心位置 |
| 方差（σ²）| E[(X-μ)²] | 离散程度 |
| 偏度（Skewness）| E[(X-μ)³]/σ³ | 左/右偏对称性 |
| 峰度（Kurtosis）| E[(X-μ)⁴]/σ⁴ | 尾部厚薄程度 |

金融资产通常：**负偏度**（下跌幅度更大）+ **正超额峰度**（极端事件更频繁）。

**对数正态分布与股价建模**

```
假设股票每日对数收益率 r ~ N(μ, σ²)
则股价 P_t = P_0 × exp(μt + σ√t × Z)，Z ~ N(0,1)

含义：
  - 股价不会为负（exp > 0）
  - 百分比变动的方差稳定（对数收益率方差不随价格水平变化）
  - 这是 Black-Scholes 期权定价模型的基本假设
```

---

## 4. 分析框架

**选择分布假设时的决策框架**

| 情景 | 推荐分布 | 原因 |
|------|---------|------|
| 日常风险估算（VaR）| 正态 + 尾部调整 | 计算简单，适合常规风险 |
| 股价/资产价格建模 | 对数正态 | 价格非负，百分比波动稳定 |
| 压力测试/极端风险 | 厚尾分布（t 分布、历史模拟）| 正态低估极端事件 |
| 因子 IC 分析 | 非参数（Spearman 相关）| 不依赖正态假设 |

---

## 5. 关键指标或关键变量

**VaR（在险价值，Value at Risk）**

```
VaR(α) = 分位数 q_α，使得 P(损失 > VaR) = α

在正态分布假设下：
VaR(5%) = μ - 1.645σ（日损失超过此值的概率为 5%）
VaR(1%) = μ - 2.33σ
```

正态假设下的 VaR 低估了实际极端损失（因为正态分布尾部太薄）。

**CVaR / Expected Shortfall**

```
CVaR(α) = E[损失 | 损失 > VaR(α)]
```

超过 VaR 门槛时的平均损失。比 VaR 更能反映尾部风险（VaR 只告诉你临界值，CVaR 告诉你超过临界值后平均亏多少）。

**偏度与峰度的实际意义**

- 负偏度：下跌幅度大于上涨幅度（股票市场典型特征）
- 高峰度（> 3）：极端事件更频繁，正态分布会低估尾部风险
- "买保险"的策略（long tail）有正偏度；"卖保险"策略（short tail）有负偏度

## 6. 实际例子

**例 1：正态分布低估股市崩溃概率**
2008 年 10 月，标普 500 单日下跌 9%。如果日收益率是正态分布（年均 0%，年波动率 20% → 日波动率约 1.25%），9% 的跌幅 = 7.2σ 事件，正态分布预测概率约为 3×10⁻¹³（即每 3 万亿天才发生一次）。但实际上类似事件每隔几年就会发生一次，说明正态分布严重低估了尾部风险。

**例 2：对数正态分布的直觉**
若年对数收益率均值 10%，标准差 20%：
- 一年后价格分布的中位数 ≈ P₀ × e^0.10 ≈ 1.105 P₀
- 均值 ≈ P₀ × e^(0.10 + 0.5×0.04) ≈ 1.127 P₀
- 价格不会为负（exp > 0），且高价格情景的概率贡献拉高了均值

**例 3：VaR 的局限**
某投资组合正态分布假设下，1% VaR（日）= -2%。这说明 100 天中大约有 1 天损失超过 2%。但正态假设下，CVaR（条件期望损失）≈ -2.67%；而在实际厚尾分布下，CVaR 可能是 -5% 或更多。VaR 报告了门槛，没有说明超过门槛后有多惨。

## 7. 常见误区

**误区 1：金融收益率服从正态分布**
正态分布假设在金融中几乎总是错的（尖峰厚尾）。正态假设用于日常管理是合理的近似，但用于极端风险估算会严重低估尾部损失。

**误区 2：σ 完整描述了风险**
标准差只描述了分布的扩散程度，没有区分偏度和峰度。同样标准差的两个分布可能有截然不同的尾部风险。

**误区 3：历史波动率 = 未来波动率**
波动率具有波动聚类效应（GARCH 特征）——低波动时期之后往往仍然低波动，高波动之后仍然高波动。历史波动率对未来波动有一定预测力，但不是固定常数。

**误区 4：VaR 足以衡量风险**
VaR 只反映了"多大概率发生损失"，没有反映"一旦发生，损失有多大"。CVaR（条件在险价值）更完整，能反映极端情景下的平均损失。

## 9. 最小可执行练习

**练习 1（正态分布计算）**
某资产日收益率均值 0%，标准差 1%。在正态分布假设下：
① 单日损失超过 2% 的概率是多少？
② 250 个交易日中，预期有几天损失超过 2%？
③ 实际上 2% 以上的损失会比预测的更多还是更少？为什么？

**练习 2（偏度直觉）**
设计两个策略的收益分布特征（无需精确数字，描述形状即可）：
① 一个"稳赚小钱但偶尔大亏"的策略（如卖出期权）
② 一个"经常小亏但偶尔大赚"的策略（如买入期权）
分别描述它们的偏度和峰度方向。

**练习 3（VaR vs. CVaR）**
某组合的日收益率分布：正态分布假设下，1% VaR = -3%，CVaR = -3.8%；实际历史模拟显示，CVaR = -7.5%。两种方法差异显著，如何解释这个差异？对风险管理有什么含义？

## 10. 一页总结

**正态分布**：由 μ 和 σ 完全描述，数学上方便，但金融中尾部风险被严重低估。

**对数正态分布**：股价/财富的合理模型（非负、百分比波动稳定），是 Black-Scholes 的基础。

**实际金融分布**：尖峰厚尾（高峰度）+ 负偏度，极端事件频率远高于正态预测。

**风险度量**：VaR（发生概率的门槛）→ CVaR（超过门槛后的平均损失）→ 压力测试（极端情景）。

**核心认知**：σ 不是风险的完整描述；尾部风险在正态假设下被系统性低估。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 正态分布 | 关于均值对称的钟形分布，由 μ 和 σ 完全描述 |
| 对数正态分布 | ln(X) 服从正态分布，价格/财富的常用模型 |
| 尖峰厚尾（Leptokurtosis）| 分布中间比正态更高，尾部比正态更厚 |
| 偏度（Skewness）| 描述分布的不对称性（负偏 = 左尾更长）|
| 峰度（Kurtosis）| 描述尾部厚度，正态分布峰度 = 3 |
| VaR | 给定置信度下的最大预期损失门槛 |
| CVaR / Expected Shortfall | 超过 VaR 时的平均损失，更全面的尾部风险指标 |
