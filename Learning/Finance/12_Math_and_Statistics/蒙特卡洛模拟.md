# 蒙特卡洛模拟

## 1. 主题定位

这篇讲蒙特卡洛模拟的核心思想：用大量随机情景替代解析公式，解决金融中分布复杂、路径依赖、多变量联合的问题。是压力测试、期权定价、退休规划等场景的基础工具。

学它之前最好先懂：常用概率分布、期望方差与协方差、收益率的计算与转换。

学完后可以继续学：波动率与最大回撤（蒙特卡洛模拟最大回撤分布）、期权基础（路径依赖期权定价）、回测原则（模拟路径 vs. 历史路径的区别）。

---

## 2. 核心概念

**蒙特卡洛模拟（Monte Carlo Simulation）**
通过大量随机抽样，从模拟结果的分布中推断感兴趣的统计量（期望值、分位数、概率等）。名称来自摩纳哥蒙特卡洛赌场，代表"随机实验"。

**核心思想**

```
问题：复杂分布的某统计量无法解析求解
方法：
  ① 定义随机变量的概率分布和参数
  ② 抽样 N 次（通常 1000-100000 次）
  ③ 对每次样本计算目标量
  ④ 用 N 次结果的统计量（均值、分位数）近似真实分布

N 越大，模拟结果越精确（标准误 ∝ 1/√N）
```

**路径模拟 vs. 终点模拟**
- **终点模拟**：只模拟 T 期末的最终价格（期权定价常用）
- **路径模拟**：模拟每期的价格路径（最大回撤、路径依赖期权、退休规划常用）

## 3. 核心机制

**股价路径的蒙特卡洛模拟（几何布朗运动）**

```
假设：对数收益率每日独立同分布，服从 N(μ, σ²)
模拟单条路径（T 天）：

for t in 1..T:
  ε_t ~ N(0, 1)   # 标准正态随机数
  r_t = μ + σ × ε_t
  P_t = P_{t-1} × exp(r_t)

重复 N 次，得到 N 条路径
```

**从 N 条路径提取信息**

```
终值分布：取所有路径第 T 天的价格 → 计算均值、中位数、5% 分位（VaR）
最大回撤分布：对每条路径计算最大回撤 → 分布的 95% 分位 = "95% 置信度下的最大回撤"
破产概率：计算路径价格跌破某阈值的比例
```

**多资产联合模拟（使用相关矩阵）**

```
不能对每个资产独立抽样（会忽略相关性），需要：
1. 从相关矩阵 Σ 生成相关随机数向量（Cholesky 分解）
2. L = Cholesky(Σ)，使得 L × Lᵀ = Σ
3. z ~ N(0, I)（独立标准正态向量）
4. ε = L × z （产生具有正确相关结构的随机向量）
```

**蒙特卡洛 vs. 历史模拟 vs. 解析方法**

| 方法 | 适用场景 | 局限 |
|------|---------|------|
| 解析公式（如 BS 公式）| 简单结构、分布假设明确 | 对复杂结构失效 |
| 历史模拟 | 不依赖分布假设 | 受历史数据限制，无法覆盖历史未发生的情景 |
| 蒙特卡洛 | 复杂结构、路径依赖、多资产 | 依赖分布假设；计算量大；随机误差 |

## 4. 分析框架

**蒙特卡洛模拟的标准设计流程**

1. **定义模型**：资产收益率服从什么分布？参数如何估算（历史数据 / 预期值）？
2. **确定模拟路径数**：越多越精确，权衡计算成本（通常 10000 条起步）
3. **生成随机数**：对多资产用 Cholesky 分解保持相关结构
4. **计算目标量**：每条路径计算目标指标（最终价格、最大回撤、收益率等）
5. **统计分析**：汇总 N 条路径的目标量分布（均值、分位数、超出阈值概率）
6. **敏感性测试**：改变关键参数（波动率 ±20%、相关性 ±0.2），检查结论稳健性

## 5. 关键指标或关键变量

**模拟精度（标准误）**

```
模拟均值的标准误 = σ_结果 / √N

N=1000：标准误 = σ/31.6
N=10000：标准误 = σ/100
N=100000：标准误 = σ/316
```

精度提高 10 倍需要计算量增加 100 倍（边际收益递减）。通常 10000-50000 次对金融应用已经足够。

**退休规划中的破产概率**

蒙特卡洛最常见的实际应用之一：

```
模拟 30 年的资产路径（每年随机收益率）
  - 每年提取固定金额生活费
  - 统计资产归零（破产）的路径比例
  - "90% 置信度下能坚持 30 年" = 90% 的路径没有破产
```

## 6. 实际例子

**例 1：VaR 的蒙特卡洛计算**
某组合：3 只资产，历史年化收益率均值 8%，波动率 15%，相关矩阵已知。

蒙特卡洛模拟 10000 条 1 年路径：
- 取终值分布的 5% 分位：-18.3%（即 5% VaR = 18.3%）
- 正态分布解析结果：-16.7%（VaR）

差异原因：模拟使用了历史实际分布（含厚尾），比正态假设的解析结果更保守——更真实地反映了风险。

**例 2：最大回撤分布模拟**
某策略年化收益 10%，波动率 20%，模拟 5 年路径 10000 条。

最大回撤分布：
- 中位数最大回撤：32%
- 95% 分位最大回撤：58%

结论：即使长期预期收益不错，5 年内有 5% 的概率遇到超过 58% 的回撤——这是决定仓位和杠杆时必须考虑的信息。

**例 3：退休规划破产概率**
60 岁退休，资产 500 万，每年取 30 万，组合预期年化 6%，波动率 12%，模拟到 90 岁（30 年）：
- 破产路径比例：21%（即有 21% 的概率在 90 岁前耗尽资产）
- 将年取款降至 25 万：破产概率降至 9%
- 将股债比从 60/40 调至 50/50：破产概率 15%（波动降低但预期收益也降低）

蒙特卡洛帮助在不同参数选择下量化后果，支持更理性的退休规划决策。

## 7. 常见误区

**误区 1：模拟次数越多越好（无限制）**
N=10000 和 N=1000000 在实践中的精度差异对金融决策几乎无影响，但计算量差 100 倍。通常 N=10000-50000 在精度和效率上是合理的平衡点。

**误区 2：蒙特卡洛结果比历史回测更可信**
蒙特卡洛依赖分布假设（正态、参数估计），历史回测依赖历史数据。两者有不同的局限性——蒙特卡洛能模拟历史未发生的情景，但如果分布假设错误（如忽视厚尾），结论也会系统性偏差。

**误区 3：多资产模拟可以对每个资产独立抽样**
忽视相关性会低估组合风险——两个正相关资产同时大跌的概率，远高于独立假设下的计算。必须使用 Cholesky 分解或其他方法保持相关结构。

**误区 4：模拟结果对参数不敏感**
蒙特卡洛结果（尤其是尾部风险估计）对波动率和相关性假设极度敏感。使用蒙特卡洛时必须做敏感性分析：波动率 ±20%、相关性 ±0.2 时，结论如何变化。

## 9. 最小可执行练习

**练习 1（手动模拟）**
用 Excel 或 Python 模拟单资产 1 年的价格路径（日度，250 个交易日）：假设日均对数收益率 = 0.04%（年化约 10%），日波动率 = 1%（年化约 16%）。运行 1000 条路径，统计：① 年末价格分布的均值和中位数 ② 5% VaR ③ 最大回撤的中位数和 95% 分位。

**练习 2（参数敏感性）**
在练习 1 的基础上，将波动率分别改为 0.5%（低波）和 2%（高波），重新运行。观察：① VaR 如何变化？② 最大回撤分布如何变化？③ 结论对波动率假设有多敏感？

**练习 3（退休规划）**
简单版退休模拟：初始资产 100，每年取款 5，年化预期收益 6%，波动率 12%，模拟 20 年、1000 条路径。计算：① 20 年后仍有资产的路径比例 ② 若取款改为 7，比例如何变化？

## 10. 一页总结

**蒙特卡洛本质**：用大量随机情景替代解析公式，从模拟分布中推断风险和收益指标。

**核心步骤**：定义分布 → 抽样 N 次 → 每次计算目标量 → 汇总分布统计量。

**精度**：标准误 ∝ 1/√N；N=10000 通常足够金融应用。

**主要应用**：VaR/CVaR 计算（含厚尾）、最大回撤分布、路径依赖期权定价、退休规划破产概率。

**核心局限**：依赖分布假设；多资产需保持相关结构；结论对波动率参数敏感，必须做敏感性分析。

|------|------|

| 标准误（MC）| 蒙特卡洛估计量的随机误差，= σ/√N |
