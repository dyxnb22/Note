# 期望、方差与协方差

## 1. 主题定位

这篇讲投资组合数学的三个核心统计量：期望值（预期收益）、方差（风险度量）、协方差（资产间关联性）。这是现代投资组合理论的数学基础。

学它之前最好先懂：收益率的计算与转换、概率直觉与统计基础。

学完后可以继续学：相关性与回归分析（协方差的延伸应用）、因子投资框架（多因子风险模型的数学基础）、分散化与相关性（09_Portfolio）。

---

## 2. 核心概念

**期望值（Expected Value）**

```
E[X] = Σ pᵢ × xᵢ（离散）
     = ∫ x · f(x) dx（连续）
```

在投资中，期望收益率 = 各情景收益率的概率加权平均。

**方差（Variance）**

```
Var[X] = E[(X - E[X])²] = E[X²] - (E[X])²
σ² = Σ pᵢ × (xᵢ - μ)²
```

衡量随机变量围绕期望值的平均离散程度。标准差 σ = √Var[X] 与原变量单位相同，更常用。

**协方差（Covariance）**

```
Cov(X, Y) = E[(X - E[X])(Y - E[Y])]
           = E[XY] - E[X]·E[Y]
```

衡量两个随机变量同向或反向运动的程度。正值 = 同向，负值 = 反向，零 = 线性无关。

**相关系数（Correlation Coefficient）**

```
ρ(X,Y) = Cov(X,Y) / (σₓ × σᵧ)
```

将协方差标准化到 [-1, 1] 区间，便于比较不同量纲的变量。

## 3. 核心机制

**组合收益率的期望**

```
E[Rₚ] = Σ wᵢ × E[Rᵢ]
```

组合期望收益 = 各资产期望收益的权重加权平均。

**组合方差（两资产情形）**

```
σ²ₚ = w₁²σ₁² + w₂²σ₂² + 2w₁w₂Cov(R₁,R₂)
    = w₁²σ₁² + w₂²σ₂² + 2w₁w₂ρ₁₂σ₁σ₂
```

组合方差不是简单加权平均——相关系数 ρ₁₂ < 1 时，组合方差 < 各资产方差的加权平均，这是**分散化的数学来源**。

**N 资产组合方差的矩阵形式**

```
σ²ₚ = w'Σw

其中：
  w = 权重向量（N×1）
  Σ = 协方差矩阵（N×N）
  Σᵢⱼ = Cov(Rᵢ, Rⱼ)（对角线是方差，非对角线是协方差）
```

**相关系数对分散化效果的影响**

| ρ₁₂ | 含义 | 分散化收益 |
|-----|------|----------|
| +1.0 | 完全正相关 | 无分散化效果 |
| +0.5 | 正相关 | 有一定分散化 |
| 0 | 不相关 | 显著分散化 |
| -0.5 | 负相关 | 很强的分散化 |
| -1.0 | 完全负相关 | 理论上可以完全对冲 |

**方差的重要代数性质**

```
Var[aX] = a²Var[X]（缩放）
Var[X+Y] = Var[X] + Var[Y] + 2Cov(X,Y)（求和）
Var[X+Y] = Var[X] + Var[Y]（当 X, Y 独立时）
E[aX + bY] = aE[X] + bE[Y]（期望的线性性）
```

## 4. 分析框架

**从统计量到投资决策的映射**

| 统计量 | 投资含义 | 决策应用 |
|--------|---------|---------|
| E[Rᵢ] | 预期收益 | 组合的期望回报估计 |
| σᵢ² | 单资产波动风险 | 个别资产风险评估 |
| Cov(Rᵢ, Rⱼ) | 联动性 | 资产对组合整体风险的贡献 |
| ρᵢⱼ | 标准化联动程度 | 分散化潜力判断 |
| w'Σw | 组合总风险 | 组合优化的目标函数输入 |

**协方差矩阵的估算问题**

N 只股票需要估算 N(N+1)/2 个参数（N 个方差 + N(N-1)/2 个协方差）。

N=100 → 5050 个参数；N=500 → 125250 个参数。

参数估算误差会累积，这是大规模组合优化不稳定的根本原因。因子模型（将 N 只股票的协方差分解为少量因子）是解决这个问题的工程方案。

## 5. 关键指标或关键变量

**样本均值和样本方差**

```
样本均值：x̄ = (1/n) Σ xᵢ
样本方差：s² = (1/(n-1)) Σ (xᵢ - x̄)²（贝塞尔校正）
样本协方差：s(X,Y) = (1/(n-1)) Σ (xᵢ - x̄)(yᵢ - ȳ)
```

注意用 n-1（而非 n）是为了得到无偏估计量。

**协方差矩阵的正定性**
有效的协方差矩阵必须是**正定矩阵**（所有特征值 > 0），才能保证组合方差 w'Σw ≥ 0。样本数据不足时（N > 时间序列长度），样本协方差矩阵可能不正定，需要正则化处理。

## 6. 实际例子

**例 1：两资产组合方差计算**
资产 A：σ_A = 20%，资产 B：σ_B = 15%，相关系数 ρ = 0.3，各持 50%。

组合方差 = 0.5² × 20%² + 0.5² × 15%² + 2×0.5×0.5×0.3×20%×15%
= 0.25×0.04 + 0.25×0.0225 + 0.5×0.3×0.03
= 0.01 + 0.005625 + 0.0045
= 0.020125

组合标准差 = √0.020125 ≈ 14.2%（低于 A、B 的简单加权平均 17.5%）

**例 2：负相关资产的对冲效果**
某股票仓位 σ=20%，对冲工具（与股票相关系数 -0.8，σ=15%），各持 50%：

组合方差 = 0.25×0.04 + 0.25×0.0225 + 2×0.5×0.5×(-0.8)×0.20×0.15
= 0.01 + 0.005625 - 0.012
= 0.003625

组合标准差 = 6%（仅为股票单独持有时的 30%！）

**例 3：样本估算不稳定性**
用 2 年月度数据估算 200 只股票的协方差矩阵：24 个月，200 只股票，需要估算 20100 个参数，但只有 24×200=4800 个数据点。参数远多于观测，估算极度不稳定，必须使用收缩估计或因子模型降维。

## 7. 常见误区

**误区 1：协方差为 0 意味着两资产完全独立**
协方差（相关系数）为 0 只意味着线性无关，两资产仍然可以存在非线性依赖关系（如在市场崩溃时同时大跌）。

**误区 2：历史协方差等于未来协方差**
协方差矩阵是时变的——在市场压力时期，相关系数往往急剧上升（相关性收敛），历史协方差会低估真实风险。这是风险模型的主要失效来源。

**误区 3：分散化在危机时仍然有效**
正常市场中相关性低的资产，在极端危机时相关性趋近 1（"流动性危机" = 所有资产同时抛售）。2008 年金融危机时，大多数资产类别相关性飙升，分散化效果大幅消失。

**误区 4：用算术均值估算长期期望收益**
长期复利增长的正确预测应使用几何均值，而非算术均值（参见波动率拖累）。用算术均值预测长期财富会系统性高估。

## 9. 最小可执行练习

**练习 1（基本计算）**
三种情景下的年收益率（概率相等）：牛市+25%，平市+5%，熊市-20%。
计算：① 期望收益率 ② 方差 ③ 标准差

**练习 2（两资产组合）**
股票仓位 60%（σ=22%），债券仓位 40%（σ=7%），相关系数 -0.1。
计算：① 组合期望收益（假设股票期望 10%，债券 3%）② 组合标准差 ③ 与纯股票仓位相比，组合的夏普比率如何变化？

**练习 3（分散化直觉）**
设计一个思维实验：如果持有 N 只独立（相关系数=0）且方差相同的资产（各占 1/N）。证明：组合方差 = 单资产方差 / N。N 从 1 增加到 10、50、100，方差各降低多少？这说明分散化的边际收益如何变化？

## 10. 一页总结

**期望值**：概率加权平均，衡量中心趋势；**方差**：衡量离散程度（风险）；**协方差**：衡量两资产联动性。

**组合期望**：权重加权平均；**组合方差**：不是简单加权——协方差项决定分散化收益。

**核心公式**：σ²ₚ = w'Σw；分散化效果 = 相关系数越低，σ²ₚ 降幅越大。

**核心局限**：协方差在危机时不稳定；相关性≠完全依赖性；历史估算有误差。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 期望值 | 随机变量的概率加权平均，衡量中心趋势 |
| 方差（σ²）| 随机变量偏离期望值的平方的期望，衡量离散程度 |
| 标准差（σ）| 方差的平方根，与原变量同单位，更直观 |
| 协方差 | 两随机变量联合偏离其均值的乘积的期望 |
| 相关系数（ρ）| 协方差标准化到 [-1,1]，衡量线性联动强度 |
| 协方差矩阵（Σ）| 多资产协方差的矩阵表示，组合优化的核心输入 |
| 正定矩阵 | 所有特征值 > 0 的矩阵，有效协方差矩阵的必要条件 |
