# 收益率的计算与转换

## 1. 主题定位

这篇讲收益率的各种计算方式：算术收益率、几何收益率、对数收益率，以及它们之间的转换关系和各自的适用场景。这是量化分析中最基础的数学工具。

学它之前最好先懂：时间价值与贴现（理解复利概念）。

学完后可以继续学：期望方差与协方差（统计分析的前提是选对收益率定义）、复利与对数收益率（00_Foundations 中的直觉讲解）。

---

## 2. 核心概念

**算术收益率（Simple Return）**

```
R = (P₁ - P₀ + D) / P₀ = (P₁ + D) / P₀ - 1
```

其中 P₀ = 期初价格，P₁ = 期末价格，D = 期间分红。

**几何收益率（Geometric / Compound Return）**
多期累计收益的等效单期年化形式。N 期总收益 W，对应的年化几何收益率：

```
r_geo = (W)^(1/N) - 1
```

**对数收益率（Log Return / Continuously Compounded Return）**

```
r_log = ln(P₁/P₀) = ln(1 + R)
```

连续复利假设下的收益率，在数学上最方便处理（多期加总 = 单期对数收益率之和）。

---

## 3. 核心机制

**三种收益率的关系**

```
R（算术）= (P₁ - P₀) / P₀

r（对数）= ln(1 + R) ≈ R（当 R 很小时）

关系：r = ln(1 + R)，反向：R = eʳ - 1
```

**多期累计的关键差异**

| | 算术收益率 | 对数收益率 |
|---|-----------|-----------|
| 多期累计 | (1+R₁)(1+R₂)...(1+Rₙ) - 1 | r₁ + r₂ + ... + rₙ |
| 计算方式 | 连乘（麻烦）| 加总（简单）|
| 平均方式 | 几何平均 | 算术平均即可 |

这是对数收益率在量化分析中被广泛使用的根本原因：可加性让多期分析大幅简化。

**年化收益率的计算**

```
月度收益率 → 年化：
  算术年化 = 月均收益率 × 12（近似，不精确）
  几何年化 = (1 + 月均收益率)^12 - 1（正确方式）
  对数年化 = 月均对数收益率 × 12
```

**算术均值 vs. 几何均值的差异**

```
算术均值（AM）= (R₁ + R₂ + ... + Rₙ) / n
几何均值（GM）= [(1+R₁)(1+R₂)...(1+Rₙ)]^(1/n) - 1

关系：GM ≈ AM - σ²/2

其中 σ² 为收益率方差
```

波动率越高，几何均值越低于算术均值。这就是为什么频繁交易（高波动）会侵蚀长期复利收益。

**不同场景下选哪种收益率**

| 场景 | 推荐使用 | 原因 |
|------|---------|------|
| 截面比较（同一期间不同资产）| 算术收益率 | 跨资产比较更直观 |
| 时序统计（回归、相关性）| 对数收益率 | 加法可加性，统计性质更好 |
| 长期复利计算 | 几何收益率 | 反映实际财富增长路径 |
| 风险估算（VaR）| 对数收益率 | 近似正态分布，便于统计 |
| 向投资者报告 | 算术收益率 | 最直观，日常理解 |

## 4. 分析框架

**收益率计算中的常见决策点：**

1. 是否含分红/利息？（Total Return vs. Price Return）
2. 用日/月/年哪种频率？（影响统计检验的功效）
3. 用算术还是对数？（取决于后续分析目的）
4. 名义收益率还是实际收益率？（是否扣除通胀）
5. 本币还是换算到共同货币？（跨国比较时的汇率影响）

---

## 5. 关键指标或关键变量

**波动率拖累（Variance Drag）**
几何均值相对算术均值的损耗 ≈ σ²/2。

例：某资产月均算术收益 1%，月标准差 5%：
- 波动率拖累 = 5%² / 2 = 0.125%/月
- 几何均值 ≈ 1% - 0.125% = 0.875%/月
- 年化几何收益率 = (1.00875)^12 - 1 ≈ 11%（而非算术年化的 12%）

高波动策略的复利收益系统性低于简单算术外推，这是分散化和控制波动的数学依据之一。

---

## 6. 实际例子

**例 1：算术均值 vs. 几何均值的差异**
某股票两年收益率：第一年 +100%，第二年 -50%。
- 算术均值 = (100% + (-50%)) / 2 = 25%（看起来很好）
- 几何均值 = √(2 × 0.5) - 1 = 0%（实际财富没有增长）
- 100 元 → 200 元 → 100 元，净收益 0%

**例 2：对数收益率的加法性**
三日对数收益率：+2%、-1%、+1.5%
累计对数收益率 = 2% - 1% + 1.5% = 2.5%
换算为算术收益率：e^0.025 - 1 ≈ 2.53%

**例 3：年化收益率的正确计算**
某基金 3 年累计收益 45%：
- 算术年化（错误）= 45% / 3 = 15%
- 几何年化（正确）= (1.45)^(1/3) - 1 ≈ 13.2%

## 7. 常见误区

**误区 1：用算术均值预测长期复利**
算术均值总是高估实际复利收益（除非波动率为 0）。预测长期财富积累必须用几何均值，差距随波动率和时间拉长而扩大。

**误区 2：对数收益率和算术收益率可以互换**
对于小幅波动（月度收益在 ±5% 以内），两者近似相等；但对于大幅波动（如 ±30%），差异显著，不可互换使用。

**误区 3：年化收益率只需乘以 12**
月度收益率 × 12 只是线性近似，当月度收益率超过 2-3% 时误差显著。正确方法是用几何年化：(1 + 月度收益)^12 - 1。

**练习 1（基本计算）**
某股票：期初价格 50 元，期末价格 58 元，期间分红 1 元。计算：① 算术收益率 ② 对数收益率 ③ 两者差多少？为什么？

**练习 2（几何均值）**
某基金五年收益率：+20%、-15%、+25%、-10%、+30%。
计算：① 算术均值 ② 几何均值 ③ 实际 100 元变成多少？

**练习 3（年化对比）**
某策略月均对数收益率 0.8%，月标准差 4%。
计算：① 年化算术收益率近似值 ② 年化几何收益率（精确）③ 波动率拖累的年化损耗是多少？

## 10. 一页总结

**三种收益率**：算术（直观、截面比较）、对数（可加性、统计分析）、几何（长期复利的真实衡量）。

**关键关系**：r_log = ln(1 + R)；GM ≈ AM - σ²/2（波动率拖累）。

**多期累计**：算术连乘 vs. 对数加总（这是对数收益率的核心优势）。

**选用原则**：预测长期财富 → 几何均值；统计回归/相关性 → 对数收益率；日常报告 → 算术收益率。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 算术收益率 | (期末价格 - 期初价格) / 期初价格，最直观的单期收益衡量 |
| 对数收益率 | ln(期末/期初)，连续复利假设下的收益率，多期可加总 |
| 几何均值 | 多期复利等效的年化收益率，反映实际财富增长速度 |
| 波动率拖累 | 因波动性导致几何均值低于算术均值的损耗，约等于方差/2 |
| 年化收益率 | 将任意时期收益率换算为年度等效值 |
| Total Return | 含分红/利息再投资的完整收益率（vs. 仅价格变动）|
