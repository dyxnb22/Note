# 如何读一家公司

## 1. 主题定位

这篇讲系统性研究一家公司的方法：从哪里开始、阅读顺序、关注哪些问题、如何建立对公司的完整理解。是投资研究工作流的入口文档。

学它之前最好先懂：基本面分析框架、财报分析、护城河与商业模式、DCF 估值。

学完后可以继续学：投资备忘录写作（将研究结论结构化）、投资复盘方法（检验研究假设）。

---

## 2. 核心概念

**由外而内的研究顺序**
先理解行业和竞争格局（宏观视角），再深入单一公司（微观视角）。单独研究一家公司而不理解行业，容易被表面数字迷惑，忽视行业结构性问题。

**主要信息来源层级**

| 来源 | 可信度 | 局限性 |
|------|--------|--------|
| 公司年报 / 20-F / 招股说明书 | 高（有审计）| 公司自选性披露 |
| 管理层电话会议记录 | 中 | 倾向于正面表述 |
| 监管文件（SEC、证监会）| 高 | 不含解释性信息 |
| 行业研究报告 | 中 | 分析师可能有利益关系 |
| 竞争对手年报 | 中高 | 间接了解行业 |
| 供应商 / 客户反馈 | 高（一手）| 难以获取 |
| 卖方研究报告 | 低 | 通常偏乐观 |

**第一性问题**
研究任何公司前先问：**这家公司靠什么赚钱，为什么客户选择它而非竞争对手？** 如果两分钟内无法回答，说明理解还不够深。

## 3. 核心机制

**读公司的七步骤**

**步骤 1：行业定位（30 分钟）**
- 这是什么行业？行业规模多大？增长率如何？
- 主要玩家有哪些？行业集中度？
- 行业的盈利驱动因素是什么（规模经济、网络效应、转换成本）？

**步骤 2：商业模式理解（1 小时）**
- 公司如何获客？留客？如何收费？
- 成本结构是什么？固定成本 vs. 可变成本的比例？
- 收入的可重复性如何（一次性 vs. 订阅/重复购买）？

**步骤 3：竞争优势识别（1 小时）**
- 护城河来源是什么？（品牌、专利、网络效应、成本优势、转换成本）
- 护城河是在扩大还是收窄？
- 最大的竞争威胁是什么？

**步骤 4：财务健康检查（1 小时）**
- 收入/利润的增长趋势（至少看 5 年）
- 利润率走势（毛利率、营业利润率、净利润率）
- 资本回报率（ROIC、ROE）是否稳定或提升？
- 现金流 vs. 账面利润的质量（高质量利润 = FCF ≈ Net Income）
- 资产负债表的健康程度（债务/EBITDA、流动比率）

**步骤 5：管理层评估（30 分钟）**
- 管理层的资本配置历史（是否理性、是否对股东友好）？
- 过去的预期 vs. 实际表现（言行一致性）？
- 激励机制是否与股东利益对齐？
- 有无关联交易、大额应收款、频繁会计变更等警示信号？

**步骤 6：估值粗判（30 分钟）**
- 当前市场给了什么估值（PE、EV/EBITDA、PS）？
- 相对历史均值和可比公司，是贵还是便宜？
- 市场隐含了什么增长假设？这个假设是否合理？

**步骤 7：风险清单（30 分钟）**
- 主要下行情景是什么？（监管、竞争、宏观、财务）
- 什么情况会让当前论点失效？
- 是否有尾部风险（突然失去大客户、诉讼、技术颠覆）？

## 4. 分析框架

**三个核心问题（Peter Lynch 框架延伸）**

1. **这家公司做什么？**（商业模式）
2. **为什么它能赚钱？**（竞争优势）
3. **为什么现在是好的买入时机？**（催化剂 + 估值）

三个问题都有清晰答案，才值得深入研究；任何一个无法回答，都应该停下来。

**年报阅读优先级**

```
优先阅读：
1. CEO 致股东信（多年对比，观察一贯性）
2. MD&A（管理层讨论与分析，了解核心驱动因素）
3. 风险因素部分（重点识别不寻常的新增风险）
4. 关联交易披露
5. 审计意见（是否有保留意见？）

次要阅读：
6. 详细财务报表（结合上述背景理解数字）
7. 注释（关注收入确认方法、固定资产折旧等会计政策）
```

## 5. 关键指标或关键变量

**快速健康检查指标**

| 指标 | 健康信号 | 警告信号 |
|------|---------|---------|
| 毛利率趋势 | 稳定或提升 | 持续下滑 |
| 应收账款 / 收入 | 稳定 | 快于收入增长 |
| 经营现金流 / 净利润 | 接近或大于 1 | 持续远小于 1 |
| ROIC | > 资金成本（WACC）| 持续低于 WACC |
| 净债务 / EBITDA | < 3x | > 5x（重资产行业除外）|

**定性质量评分**
- 商业模式可理解性（1-5 分）
- 护城河强度（1-5 分）
- 管理层质量（1-5 分）
- 财务透明度（1-5 分）

## 6. 实际例子

**例 1：阅读顺序的重要性**
研究一家中国在线教育公司，如果直接看利润，2019 年利润丰厚。但先研究行业——监管政策风险极大（K12 课外培训受政策约束）——就会对利润的可持续性保持警惕。2021 年监管落地，行业利润近乎消失。先理解行业结构，再看财务数字。

**例 2：CEO 致股东信的多年对比**
对比某公司 2015-2020 年 CEO 致股东信：2015 年承诺"聚焦核心业务，不盲目扩张"；2017 年开始大量收购；2019 年承认"过去的收购有些失误，将出售非核心资产"。言行不一致是管理层质量的重要红旗。

**例 3：应收账款异常**
某公司收入 5 年增长 3 倍，但应收账款增长了 8 倍。进一步调查发现：应收账款大量来自关联方，且回款周期从 60 天延长至 240 天。这是收入质量极差的典型信号（收入可能存在虚报）。

## 7. 常见误区

**误区 1：先看 PE，再决定是否研究**
PE 是结论的一部分，不是研究的起点。先理解业务，再讨论估值是否合理。

**误区 2：只读当年年报**
单年快照无法判断趋势。至少需要 5 年数据才能看出利润率变化、资本配置模式、管理层承诺的兑现情况。

**误区 3：相信管理层的解释而不独立验证**
管理层有动机以最好的方式呈现结果。重要结论（客户增长、市场份额）需要从独立来源（竞争对手数据、行业调研）交叉验证。

**误区 4：把复杂性当深度**
真正理解一家公司是能够用简单语言解释它——"这家公司靠 X 赚钱，竞争优势是 Y，最大风险是 Z"。如果解释需要 5 分钟，通常说明没有真正理解，或者公司本身过于复杂不适合投资。

## 9. 最小可执行练习

**练习 1（七步框架）**
选一家熟悉的上市公司（如某零售商或科技公司），用七步骤框架做一次完整梳理。每步用 3-5 句话记录核心发现，重点是：商业模式是否清晰？护城河是否真实？财务是否健康？

**练习 2（年报精读）**
下载该公司最近三年的年报，只阅读：① CEO 致股东信 ② 风险因素 ③ 关联交易。对比三年，识别：管理层的描述是否一致？风险因素有什么新增内容？

**练习 3（一句话测试）**
用一句话（不超过 30 个字）描述：这家公司靠什么赚钱。如果无法完成，说明研究还不够深，或者公司太复杂不适合投资。

---

## 10. 一页总结

**研究顺序**：由外而内（行业 → 商业模式 → 竞争优势 → 财务 → 管理层 → 估值 → 风险）。

**三个核心问题**：做什么 → 为什么能赚钱 → 为什么现在是买入机会。

**年报重点**：CEO 致股东信（多年对比）、MD&A、风险因素、关联交易、审计意见。

**快速警告信号**：应收款增长超过收入、现金流持续低于利润、利润率持续下滑、管理层言行不一。

**核心原则**：无法用简单语言解释的公司，通常意味着不理解或不适合投资。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| MD&A | 管理层讨论与分析，年报中管理层对经营状况的解释 |
| 20-F | 外国私人发行人向 SEC 提交的年度报告（美股上市的外国公司）|
| 关联交易 | 公司与其关联方（大股东、管理层等）之间的交易 |
| ROIC | 投入资本回报率，衡量资本使用效率 |
| FCF | 自由现金流，经营现金流减去资本支出 |
| 审计意见 | 审计师对财报公允性的独立意见，保留意见是重要红旗 |
