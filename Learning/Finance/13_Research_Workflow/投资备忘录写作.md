# 投资备忘录写作

## 1. 主题定位

这篇讲如何写投资备忘录（Investment Memo）：一份结构化的文档，将对一家公司的研究结论转化为清晰的投资论点。好的备忘录既是思考工具，也是日后复盘的依据。

学它之前最好先懂：如何读一家公司、基本面分析框架、安全边际。

学完后可以继续学：投资复盘方法（用备忘录作为复盘的参照基准）。

---

## 2. 核心概念

**投资备忘录（Investment Memo）**
一份记录投资决策逻辑的结构化文件：为什么买、预期收益从何而来、关键假设是什么、什么情况会让论点失效。是投资纪律的核心工具。

**备忘录的双重功能**
1. **决策工具**：强迫自己将模糊的直觉转化为可验证的具体假设，暴露逻辑漏洞
2. **复盘工具**：记录买入时的预期，未来可以对照实际结果，区分"正确论点但错误时机"与"错误论点偶然赚钱"

**核心区别：论点 vs. 故事**
- 故事：这是一家很好的公司，管理层优秀，产品创新。
- 论点：公司在 X 市场的渗透率目前仅 15%，行业天花板足够大，如果能达到 30%，收入将翻倍；当前估值（10x EV/Revenue）低于可比公司（15-18x）；催化剂是明年新产品发布。

备忘录要写论点，不写故事。

---

## 3. 核心机制

**投资备忘录的标准结构**

```
1. 一句话论点（Executive Summary）
   ← 整个备忘录的核心，1-2 句话

2. 公司简介（Company Overview）
   ← 商业模式、行业、规模，3-5 句话

3. 投资论点（Investment Thesis）
   ← 为什么这是好投资？3-5 个关键理由，每个理由附数据支撑

4. 估值分析（Valuation）
   ← 当前估值是什么？基本/乐观/悲观情景下的目标价？上行/下行空间？

5. 关键假设（Key Assumptions）
   ← 论点成立依赖哪些假设？这些假设可验证吗？

6. 风险与催化剂（Risks & Catalysts）
   ← 什么可能让论点失效？什么事件可能推动价格实现？

7. 退出标准（Exit Criteria）
   ← 什么情况下卖出？（目标价达到/论点失效/更好机会出现）
```

**关键假设的可验证性要求**

好的关键假设是可以被事后证伪的：

| 不可验证（差）| 可验证（好）|
|-------------|------------|
| 管理层执行力强 | 2024 年营收增速达到 20%+ |
| 竞争格局稳定 | 市占率维持在 30% 以上 |
| 品牌护城河深厚 | 毛利率不低于 55% |
| 估值便宜 | EV/EBITDA 低于 10x，低于历史均值 15x |

可验证的假设让备忘录成为有意义的复盘工具——你可以在未来某个时间点检查每个假设是否实现。

**估值矩阵（Scenario Analysis）**

```
                悲观情景（30%概率）  基准情景（50%）  乐观情景（20%）
收入增长            5%              15%             25%
利润率              12%             18%             22%
目标 EV/EBITDA      8x              12x             16x
隐含股价            25 元           45 元           70 元
期望价值 = 0.3×25 + 0.5×45 + 0.2×70 = 7.5 + 22.5 + 14 = 44 元
当前价格 40 元，隐含上行空间 10%（偏低，可能不值得持有）
```

## 4. 分析框架

**一句话论点的写法**

格式：[公司名称] 在 [时间段] 内将带来 [预期收益] 的回报，核心驱动是 [1-2 个主要因素]，当前交易于 [X 倍估值] 相对于 [比较基准] 存在 [折价/溢价]。

示例："XX 软件在未来 2 年可能带来 40-60% 的回报，核心驱动是企业数字化渗透率提升（当前仅 20%）和产品定价能力改善，当前交易于 15x EV/ARR，相对全球可比公司（20-25x）存在 25% 折价。"

**备忘录长度原则**
- 初始备忘录：1-3 页（核心论点要清晰，不需要面面俱到）
- 深入研究报告：5-10 页（用于重仓或复杂情况）
- 过长 = 没有想清楚，或者在用篇幅掩盖论点的薄弱

## 5. 关键指标或关键变量

**备忘录质量自检清单**

- [ ] 能否用一句话说清楚投资论点？
- [ ] 每个关键假设是否可以被事后验证？
- [ ] 是否列出了让论点失效的主要情景？
- [ ] 估值是否给出了上行和下行两个方向？
- [ ] 退出标准是否明确（不依赖"感觉"）？
- [ ] 备忘录中有没有"管理层优秀"这类不可验证的表述作为主要论据？

## 6. 实际例子

**例 1：好论点 vs. 差论点的对比**

差的论点：
"这家消费品公司品牌知名度高，产品质量好，管理层有多年经验，行业前景广阔，股票在低位值得买入。"
→ 没有可量化的假设，无法事后验证，无法说明为什么"现在"是好时机。

好的论点：
"该公司高端产品线占比从 2020 年的 20% 增长至当前 35%，驱动毛利率从 35% 提升至 45%，这个趋势有望在 2025 年前持续到 55%。当前 PE = 15x，低于高端消费品行业均值 22x，主要因为市场对其仍然用大众消费品定价。催化剂是 2024 年 H1 财报，预计毛利率首次达到 50%。"

**例 2：退出标准的重要性**
某投资者 2019 年买入某教育股，论点是"K12 线下教育渗透率快速提升"。但没有写退出标准。2021 年"双减"政策出台，核心论点已经失效，但因为没有预先定义退出标准，犹豫不决，最终在 90% 跌幅时才卖出。如果当初写明"若 K12 线下监管收紧，立即退出"，可以在政策初现时就行动。

## 7. 常见误区

**误区 1：备忘录写完就放着不管**
备忘录的价值在于持续更新和最终复盘。每季度对照关键假设检查进展，才能真正从投资经历中学习。

**误区 2：用备忘录说服自己，而非挑战自己**
写备忘录容易变成"寻找支持论点的证据"（确认偏差）。更有效的方式是先写出"为什么我的论点是错的"，再写"为什么我认为这些反驳不成立"。

**误区 3：忘记写估值的下行情景**
只写上行情景的备忘录不完整。下行情景（论点失效时能亏多少）与上行情景同等重要——收益/风险比是决策的核心。

**误区 4：论点等于"好公司 + 便宜价格"**
"好公司 + 便宜"是充分必要条件，但论点需要具体回答：**为什么现在便宜？市场为什么错了？什么时候市场会纠正？** 没有这三个问题的答案，只是描述了一个静态状态，不是可执行的投资论点。

## 9. 最小可执行练习

**练习 1（一句话论点）**
为一家你了解的公司（可以是你曾买过或研究过的），写一句话投资论点。必须包含：① 核心驱动因素（1-2 个具体的）② 当前估值水平 ③ 相对基准的折/溢价 ④ 时间预期。

**练习 2（可验证假设）**
从你写的论点中，提取 3-5 个关键假设，把每个假设转化为"可在 12 个月内用数据验证"的具体指标。然后问：如果每个假设都无法被验证，论点还成立吗？

**练习 3（完整备忘录）**
用本文的 7 步结构，为一家公司写一份完整的投资备忘录（1-2 页）。完成后用质量自检清单过一遍，识别哪些部分需要加强。

## 10. 一页总结

**备忘录的本质**：将模糊直觉转化为可验证假设的工具，同时也是未来复盘的依据。

**核心结构**：一句话论点 → 商业模式简介 → 投资论点（含数据）→ 估值矩阵 → 关键假设（可验证）→ 风险与催化剂 → 退出标准。

**好论点 vs. 差论点**：好论点有具体可量化的假设；差论点是对公司质量的模糊赞美。

**核心原则**：先写"为什么我可能是错的"，再写"为什么我认为市场错了"；退出标准不能依赖感觉。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 投资备忘录 | 记录投资决策逻辑的结构化文件，包含论点、假设、估值和退出标准 |
| 投资论点（Thesis）| 清晰说明为何这是好投资、收益来源和时间预期的核心论述 |
| 情景分析 | 在不同假设（悲观/基准/乐观）下估算潜在结果 |
| 退出标准 | 预先定义的卖出条件，避免决策时的情绪干扰 |
| 催化剂 | 可能推动价格向目标价格靠近的具体事件或信息 |
| 可验证假设 | 能在预定时间内用具体数据证实或证伪的假设 |
