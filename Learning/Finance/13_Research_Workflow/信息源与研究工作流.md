# 信息源与研究工作流

## 1. 主题定位

这篇讲研究一家公司时的信息来源体系、信息可靠性判断，以及如何构建高效的研究工作流。正确使用信息源是研究质量的基础。

学它之前最好先懂：如何读一家公司。

学完后可以继续学：投资备忘录写作（将信息转化为结构化论点）、财务造假识别（识别信息中的欺骗成分）。

---

## 2. 核心概念

**信息层级**
一手信息（公司直接披露）> 二手信息（分析师解读）> 三手信息（媒体报道、市场传言）。层级越低，失真越多，需要越多的交叉验证。

**信息优势的来源**
1. **更早**：比市场更快获得公开信息（信息处理速度）
2. **更深**：对公开信息有更深入的分析和理解（解读能力）
3. **更不同**：注意到市场忽视的公开信息（独立视角）
4. **更广**：整合多个信息源形成其他人没有的综合视角

对个人投资者而言，来源 2 和 3 是最可行的优势来源。

---

## 3. 核心机制

**信息源分类与使用方法**

**一手信息源（最权威）**

| 来源 | 获取方式 | 主要用途 |
|------|---------|---------|
| 公司年报 / 半年报 / 季报 | 公司官网、监管网站 | 财务数据、战略方向 |
| 招股说明书 | 证监会官网 | 公司历史、股权结构、风险因素 |
| 管理层电话会议记录 | 公司官网、第三方平台 | 管理层态度、短期指引 |
| 公司公告（重大事项）| 交易所官网 | 及时了解重要变化 |
| 专利数据库 | 国家知识产权局、USPTO | 技术护城河验证 |

**行业与竞争信息**

| 来源 | 可信度 | 使用方式 |
|------|--------|---------|
| 竞争对手年报 | 高 | 交叉验证行业数据，理解竞争格局 |
| 行业协会报告 | 中高 | 行业规模、增长率的参考 |
| 政府统计数据 | 高 | 宏观行业趋势验证 |
| 行业渠道调研 | 高（一手）| 了解真实市场情况，但难以获取 |

**分析师研究**

| 来源 | 可信度 | 注意事项 |
|------|--------|---------|
| 买方研究（私募/公募内部）| 高 | 通常不对外公开 |
| 卖方研究报告 | 低-中 | 通常偏乐观，尤其是 IPO 承销行的研究 |
| 独立研究机构 | 中 | 质量差异大，需要评估来源的独立性 |
| 做空报告 | 中（有动机）| 做空机构有利益冲突，但通常有详细事实依据 |

**替代数据（Alternative Data）**
非传统数据来源，用于获得竞争优势：
- 卫星图片（停车场、工厂活动、船运）
- 消费者行为数据（信用卡消费、APP 使用数据）
- 招聘数据（岗位变化预示业务扩张/收缩）
- 社交媒体情绪

> 替代数据的使用涉及合规边界，需要确保不违反内幕交易规定。

**研究工作流的标准流程**

```
触发（为何关注这家公司）
    ↓
快速筛查（15 分钟，是否值得深入）
    ├─ 商业模式是否可理解？
    ├─ 财务质量是否及格？
    └─ 估值是否在合理区间内？
    ↓
初步研究（1-3 小时）
    ├─ 阅读最近年报 MD&A
    ├─ 了解竞争格局
    └─ 简单估值区间
    ↓
决策：是否继续深入？
    ↓（如果是）
深度研究（3-20 小时）
    ├─ 多年财报精读
    ├─ 竞争对手分析
    ├─ 行业研究
    └─ 管理层评估
    ↓
写备忘录 → 决策
    ↓
持仓期监控
    └─ 每季财报更新 → 关键假设是否还成立？
```

## 4. 分析框架

**信息可靠性评估框架**

1. **来源独立性**：信息来源是否与结论有利益关系？（卖方分析师推荐自己承销的股票）
2. **可验证性**：核心数据能否从独立来源交叉验证？
3. **时效性**：信息是否已过时？市场情况变化快时，6 个月前的研究可能失效
4. **完整性**：是否选择性引用？（只引用支持论点的数据）
5. **一手来源比例**：结论有多少基于一手信息？

---

## 5. 关键指标或关键变量

**研究效率的衡量**
- 快速筛查通过率：研究 10 家公司，多少家值得深入？（高效筛选可以节省大量时间）
- 深度研究转化率：深入研究 5 家，最终建仓多少家？
- 时间分配：深入研究 vs. 持仓监控 vs. 新想法来源，是否合理平衡？

**关注事件清单**
持仓公司的持续监控需要关注：
- 季度财报（关键假设是否实现）
- 公司公告（重大合同、高管变动、股权变化）
- 竞争对手动态（是否影响核心竞争优势）
- 监管政策变化（是否影响行业逻辑）

## 6. 实际例子

**例 1：竞争对手年报的价值**
研究某连锁餐饮公司时，直接看其年报，管理层声称"市占率持续提升，竞争格局稳定"。但阅读两家主要竞争对手的年报后发现：两家竞争对手都在加速开店，且均声称自己市占率在提升——逻辑上不可能都在涨，说明至少有一方在误导投资者。竞争对手年报提供了独立的交叉验证视角。

**例 2：招聘数据作为领先指标**
某科技公司官方表示"业务稳健增长"，但研究招聘数据时发现：该公司在核心业务线的技术岗位招聘数量同比下降 40%，而市场营销岗位增加。这与"稳健增长"的叙述不符，暗示公司可能在削减研发投入，是潜在的业务收缩信号。

**例 3：做空报告的使用**
某公司被做空机构发布报告质疑财务数据造假。做空报告有明显利益动机（做空机构持有空头），但其中列举的具体事实（应收账款异常、关联交易细节）值得认真核实。独立核实后发现部分指控有依据，最终证实财务数据存在问题。结论：做空报告要核实事实，但不能直接接受结论。

## 7. 常见误区

**误区 1：读的越多越好**
研究的深度比广度更重要。大量阅读二手分析而缺乏对一手资料的独立判断，只是在传递市场共识，不会产生信息优势。

**误区 2：卖方研究是客观的**
卖方分析师的激励机制（获取投行业务、维护客户关系）导致评级系统性偏乐观。历史数据显示，卖方"买入"评级远多于"卖出"。把卖方研究当作信息整理工具（了解背景）而非决策工具。

**误区 3：一手信息天然比二手可靠**
管理层的电话会议本身就是有选择性的信息披露——他们会强调好消息，淡化坏消息。一手信息仍然需要结合数据独立验证。

**误区 4：信息越新鲜越有价值**
"热点信息"（市场上大家都在讨论的信息）往往已经被充分定价。真正有价值的信息往往是"无聊的"——多年财报的趋势分析、竞争对手的细节、被市场忽视的公告。

## 9. 最小可执行练习

**练习 1（信息源地图）**
选一个行业，列出该行业的主要信息来源（年报来源、行业协会、替代数据来源）。评估每个来源的可靠性和获取难度，建立个人的"信息源地图"。

**练习 2（交叉验证）**
研究某公司时，提取 3 个关键指标（如市占率、毛利率趋势、新客户增长）。分别从公司年报、竞争对手年报、行业报告三个来源找数据。三个来源的结论是否一致？不一致时如何判断？

**练习 3（研究效率追踪）**
记录下一次研究公司的时间分配：快速筛查花了多长时间？深度研究花了多长时间？哪些环节耗时最多？最终是否建仓？建立个人的研究效率基线，逐步优化。

## 10. 一页总结

**信息层级**：一手（公司披露）> 二手（分析师解读）> 三手（媒体传言）。层级越低，失真越多。

**信息优势来源**：处理速度 + 解读深度 + 独立视角 + 多源整合（对个人投资者，后两者更可行）。

**工作流核心**：快速筛查（15 分钟）→ 初步研究（1-3 小时）→ 深度研究 → 写备忘录 → 持仓监控。

**卖方研究定位**：作为背景信息来源，不作为决策依据；交叉验证比单一来源更可靠。

**核心原则**：读的越多不等于判断越好；信息优势来自独立的深度解读，而非信息量的堆积。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 一手信息 | 直接来自公司或监管机构的原始披露文件 |
| 买方研究 | 基金公司内部用于投资决策的研究，通常不对外发布 |
| 卖方研究 | 券商分析师发布的对外研究报告，有利益冲突风险 |
| 替代数据 | 非传统数据来源（卫星、消费行为、招聘等）用于获取信息优势 |
| MD&A | 管理层讨论与分析，年报中解释经营状况的核心章节 |
| 做空报告 | 做空机构发布的质疑公司基本面的研究报告，有利益动机 |
