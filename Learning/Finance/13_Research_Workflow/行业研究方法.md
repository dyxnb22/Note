# 行业研究方法

## 1. 主题定位

这篇讲如何系统性研究一个行业：行业分析的框架、关键问题、竞争结构分析，以及在研究个股之前应该建立的行业层面认知。行业理解是公司研究的前提。

学它之前最好先懂：如何读一家公司、供给与需求、护城河与商业模式。

学完后可以继续学：投资备忘录写作（行业研究是 Thesis 的重要背景）、观察清单管理（在行业框架下筛选候选标的）。

---

## 2. 核心概念

**行业研究的目的**
在研究个股之前，先理解"赛道"本身的质量：行业结构是否有利于获得高利润？行业的长期增长动力是什么？竞争最终会不会把超额利润侵蚀掉？

**行业吸引力 vs. 公司竞争力**
- 好行业 + 一般公司 = 可能还行
- 差行业 + 优秀公司 = 通常很难
- 行业结构是天花板，公司优势是在天花板内的位置

**波特五力（Porter's Five Forces）**
分析行业竞争强度和盈利潜力的经典框架：买方议价力、供应商议价力、现有竞争者、替代品威胁、新进入者威胁。五力越弱，行业越有吸引力（越容易保留利润）。

## 3. 核心机制

**行业研究的五个核心问题**

**问题 1：行业的规模与增长**
```
关注点：
  - 当前市场规模（TAM）有多大？
  - 历史增速是多少？驱动增速的因素是什么？
  - 未来增速是加速、维持还是放缓？（行业生命周期）
  - 增长是真实需求驱动还是库存/信贷周期驱动？

信息来源：行业协会报告、国家统计局数据、头部公司年报
```

**问题 2：行业的盈利结构**
```
关注点：
  - 头部公司的毛利率/净利率水平是多少？
  - 行业利润率在历史上是稳定、提升还是下滑？
  - 利润主要分布在哪个环节（上游原材料/中游制造/下游品牌）？
  - 资本回报率（ROIC）长期能否超过资本成本？
```

**问题 3：竞争格局**
```
波特五力分析：
  买方议价力：客户集中吗？切换成本高吗？
  供应商议价力：原材料/关键资源被垄断吗？
  新进入者：规模经济、监管壁垒、品牌壁垒有多高？
  替代品：是否有功能相近但成本更低的替代品？
  内部竞争：行业集中度如何？价格战是否频繁？
```

**问题 4：行业周期性**
```
关注点：
  - 行业是强周期（大宗商品、汽车）、弱周期（消费品）还是反周期（医疗）？
  - 当前处于周期的什么位置？
  - 行业的库存周期、产能周期有多长？
  - 宏观环境变化如何传导到行业业绩？
```

**问题 5：结构性变化**
```
关注点：
  - 有没有技术颠覆（数字化、AI 替代）正在改变行业格局？
  - 监管政策是否在改变行业游戏规则？
  - 消费者偏好是否在迁移（从线下到线上、从大品牌到白牌）？
  - 这些变化是正在发生还是已经定价？
```

**行业吸引力评分框架**

| 维度 | 1 分（差）| 3 分（中）| 5 分（好）|
|------|----------|----------|----------|
| 行业增速 | 萎缩 | 与 GDP 同速 | 高于 GDP 2 倍以上 |
| 利润率水平 | 行业净利润率 < 5% | 5-15% | > 15% |
| 竞争集中度 | CR5 < 20%（分散）| CR5 20-50% | CR5 > 50% |
| 进入壁垒 | 无壁垒 | 中等 | 强壁垒（监管/资本/技术）|
| 周期性 | 强周期 | 中等 | 弱周期或反周期 |
| 结构性变化 | 受技术颠覆威胁 | 中性 | 受益于结构性趋势 |

## 4. 分析框架

**行业研究输出的三层结论**

```
第一层：行业本身值不值得投资？
  → 使用吸引力评分框架做整体判断

第二层：行业内的利润分布在哪里？
  → 产业链分析：哪个环节最赚钱，为什么？

第三层：哪类公司在行业内最受益？
  → 竞争位置：头部 vs. 二线、成本领先 vs. 差异化
```

**行业生命周期定位**

```
导入期 → 成长期 → 成熟期 → 衰退期

各阶段特征：
  导入期：市场小，不确定性高，利润率低或为负，少数先驱者
  成长期：市场快速扩张，竞争格局未定，利润率改善
  成熟期：增速放缓，竞争白热化，利润率趋稳或下降，行业集中度高
  衰退期：市场萎缩，产能过剩，价格战，弱者退出
```

处于不同阶段的行业需要不同的分析重点和估值方法。

## 5. 关键指标或关键变量

**CR5（前五名市场份额）**
= 行业前五家公司的收入之和 / 行业总收入

CR5 > 60%：高度集中，龙头定价权强；CR5 < 20%：高度分散，价格战激烈，利润率受压。

**行业 ROIC 的历史中枢**
长期 ROIC 能否维持在 WACC 以上，是判断行业是否能持续创造价值的根本标准。ROIC 长期低于 WACC 的行业（如部分航空、钢铁），即使个股选择再好，整体回报也难以可观。

**库存/产能周期位置**
通过行业库存数据、在建产能、开工率等指标判断周期位置——库存低 + 开工率高 + 新增产能少 → 可能处于上行阶段。

## 6. 实际例子

**例 1：白酒行业的行业分析**
- 规模与增长：高端白酒需求结构性增长，与 GDP 增速脱钩
- 盈利结构：头部毛利率超过 80%，净利润率超过 40%，远超大多数行业
- 竞争格局：CR5 极高，高端市场茅台一家独大，定价权极强
- 波特五力：买方分散（无议价力）、替代品弱（文化属性）、进入壁垒极高（品牌+年份+产地）
- 结论：行业吸引力极高，但估值也因此长期较高

**例 2：光伏组件行业**
- 竞争格局：CR5 较高，但技术路线迭代快
- 盈利结构：毛利率长期被压缩（下游逐渐成为大客户，议价力强）
- 结构性变化：成本下降驱动装机持续增长，但受益于产业链内部转移而非整体利润率提升
- 结论：成长行业，但激烈竞争导致利润率偏低，价值主要集中在技术壁垒高的环节

**例 3：零售百货行业**
- 结构性变化：电商替代实体零售，流量从线下迁移到线上
- 行业生命周期：成熟期 → 衰退期
- 结论：无论个别公司如何优秀，行业逆风很强；在行业研究层面先排除，节省公司研究时间

## 7. 常见误区

**误区 1：先看公司，后想行业**
对行业结构不了解的情况下研究公司，容易被公司的短期财务数字迷惑，而忽视行业的结构性压力。行业分析应先于公司分析。

**误区 2：高增速行业 = 好投资**
高增速吸引大量竞争者进入，可能反而导致利润率下降。成长不等于盈利，关键是竞争壁垒是否足够阻止利润被侵蚀。很多成长赛道回报率不如稳定的传统行业。

**误区 3：行业分析只需要做一次**
行业结构会随时间变化（技术颠覆、监管变化、消费者偏好迁移）。应该定期（每年）重新评估行业吸引力评分，确认原来的判断是否仍然成立。

**误区 4：同行业的公司都适合用同一套分析框架**
行业内的产业链位置差异极大（上游资源、中游制造、下游品牌）。分析白酒品牌公司和分析玻璃瓶供应商，即使都算"白酒产业链"，驱动因素完全不同。

## 9. 最小可执行练习

**练习 1（波特五力）**
选一个你熟悉的行业（如外卖、新能源汽车、连锁咖啡），对五力逐一评分（1-5 分，5 = 威胁最小）。总分越高，行业越有吸引力。哪一力是这个行业的主要挑战？

**练习 2（产业链利润分布）**
画出某行业的产业链（上游 → 中游 → 下游），并标注各环节的代表公司和大致毛利率水平。利润主要集中在哪个环节？为什么？

**练习 3（生命周期定位）**
选三个行业（如手机、外卖、汽车），判断它们各自处于什么生命周期阶段，并列出 2-3 个支撑理由。处于不同阶段的行业，最重要的分析指标分别是什么？

---

## 10. 一页总结

**行业研究的五个核心问题**：规模与增速 → 盈利结构 → 竞争格局（波特五力）→ 周期位置 → 结构性变化。

**行业吸引力判断**：ROIC 长期 > WACC；竞争集中度高；进入壁垒强；弱周期；受益于结构性趋势。

**行业先于公司**：行业是天花板，公司是在天花板内的位置。差行业中的好公司，通常也很难长期出色。

**三层输出**：行业整体值不值得投资 → 利润分布在哪个环节 → 哪类公司在行业内最受益。
