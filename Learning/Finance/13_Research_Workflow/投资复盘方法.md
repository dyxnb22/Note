# 投资复盘方法

## 1. 主题定位

这篇讲如何系统性复盘投资决策：区分运气与技能，从成功和失败中提取真正有价值的经验，避免常见的复盘误区。复盘是持续改进投资能力的核心机制。

学它之前最好先懂：投资备忘录写作（复盘的对照基准）、损失厌恶和过度自信（了解复盘中的认知偏差）。

学完后可以继续学：14_Fraud_and_Pitfalls（从他人的失败案例中学习）。

---

## 2. 核心概念

**结果 vs. 过程**
投资结果（赚了多少）和投资过程（决策是否正确）是两件不同的事。好的过程可能产生坏的短期结果（运气不佳），坏的过程也可能产生好的短期结果（运气好）。只复盘结果而不复盘过程，会强化错误的决策模式。

**结果矩阵**

```
              结果：赚钱          结果：亏钱
过程正确：   ✅ 理想情景         ⚠️ 糟糕运气（正确决策，等待）
过程错误：   ⚠️ 虚假成功（侥幸）  ❌ 应该避免的情景
```

复盘的核心任务：将"虚假成功"从"理想情景"中区分出来，避免将侥幸好运归因于技能。

**归因的两种偏差**
- **自我服务归因**：盈利时归功于自己的技能，亏损时归咎于外部运气
- **结果偏差**：用最终结果反推决策质量（赚了 = 决策正确，亏了 = 决策错误）

两种偏差都会阻碍真正的学习。

## 3. 核心机制

**复盘的四层深度**

**第一层：结果层（发生了什么）**
- 买入价格、卖出价格、持有时间、总收益
- 相对基准的超额收益（排除市场 Beta 影响）

**第二层：原因层（为什么会这样）**
- 原来的论点是什么？哪些假设实现了？哪些没有？
- 价格的变动是因为基本面变化，还是因为估值变化，还是市场情绪？

**第三层：决策质量层（决策当时的过程对不对）**
- 买入时的信息和逻辑是否合理？（用当时的信息评估，而非事后信息）
- 如果重新回到买入那一刻，同样的信息下，是否应该做同样的决策？
- 退出决策（卖出）是否遵守了原来的退出标准？

**第四层：系统性改进层（如何提升）**
- 这次的错误/成功是否有规律性？（是个例还是系统性问题）
- 需要调整哪个环节的研究方法？（信息收集、估值方法、仓位管理）
- 下一次类似情况，决策流程的哪一步需要改变？

**复盘的三种情景处理**

**情景 1：赚钱且论点正确（理想情景）**
- 确认哪些假设实现了，理解为何实现
- 提取可复制的要素，但注意避免过度自信（部分可能有运气成分）

**情景 2：亏钱但论点原本正确（糟糕运气）**
- 区分"外部事件打断了正确论点"vs."论点本身的漏洞"
- 如果是外部黑天鹅（无法预见），不要因此修改正确的决策框架
- 如果是可预见的风险没有充分考虑，需要改进风险清单

**情景 3：赚钱但论点错误（虚假成功）**
- 最危险的情景——容易强化错误的决策模式
- 明确说明：这次赚钱的真实原因是什么（行业 Beta、市场情绪、还是恰好）
- 避免把侥幸好运归因于研究能力

## 4. 分析框架

**复盘备忘录的标准结构**

```
1. 基本情况
   - 买入时间、价格、论点摘要
   - 卖出时间、价格、持有期收益

2. 论点实现情况
   - 列出当初的关键假设
   - 每个假设：实现 / 部分实现 / 未实现 / 无法判断
   - 总体论点是否正确？（独立于结果）

3. 决策质量评估
   - 买入决策（当时信息下是否合理）：1-5 分
   - 仓位管理（是否与信念程度匹配）：1-5 分
   - 退出决策（是否遵守退出标准）：1-5 分

4. 本次主要教训
   - 最重要的 1-2 个可操作的改进点
   - 需要在未来研究中增加的检查项

5. 归类标签
   - 分析错误 / 执行错误 / 无法预见的外部冲击 / 正确但运气差
```

## 5. 关键指标或关键变量

**复盘的统计追踪（长期）**

积累足够多的复盘记录后，可以统计：
- 论点正确率（独立于结果）：反映基本面分析能力
- 盈利时正确论点比例：区分技能与运气
- 退出标准遵守率：衡量投资纪律
- 亏损分类：分析错误 vs. 执行错误 vs. 不可预见冲击

**真实技能的判断**
如果"论点正确率"明显高于 50%，且"赚钱时论点正确比例"明显高于"亏钱时论点正确比例"，说明存在一定的真实分析优势。相反，如果赚钱主要来自论点错误但运气好，说明过去的收益依赖运气，不可持续。

## 6. 实际例子

**例 1：虚假成功的识别**
某投资者 2020 年买入某科技股，论点是"估值低"（PE 15x）。后续股票涨了 80%，主要原因是市场整体估值扩张（大盘涨了 60%），且该股票 PE 从 15x 扩张至 25x。复盘时认为"我的价值投资判断正确"——但实际上：① 超额收益很小（仅超出大盘 20%）② 收益来源是估值扩张而非盈利增长 ③ "估值低"的论点并不意味着会重新定价，论点本身并不完整。

**例 2：过程正确但结果不佳**
某投资者研究了某家公司，论点完整、假设可验证、估值合理，建立 5% 仓位。后来该公司遭遇了突如其来的行业监管（无法事先预见），股价大跌 50%。复盘结论：这是典型的"正确过程、坏运气"——投资逻辑没有问题，监管风险是黑天鹅事件。正确做法是维持方法论，而非因此放弃同类研究。

**例 3：系统性错误的识别**
某投资者回顾 20 笔交易，发现 80% 的亏损交易有一个共同特征：买入时有"这只股票已经跌了很多，不会再跌了"的心理——而非来自基本面分析。这是一个系统性的锚定偏差，用价格而非价值判断买入时机。识别这个模式后，可以针对性修改决策流程：每次有"跌了很多"的感觉时，强制回到基本面重新估算内在价值。

## 7. 常见误区

**误区 1：只复盘亏损，不复盘盈利**
赚钱但过程错误（虚假成功）比亏损更危险——因为它会强化错误模式，且不被察觉。每笔盈利都需要同等严格的过程检查。

**误区 2：用事后信息评估事前决策**
复盘要还原到决策当时的信息状态，用当时可以获得的信息判断决策是否合理。"如果当时知道 XXX，当然不应该买"——这是事后诸葛，没有学习价值。

**误区 3：把复盘变成自我批评**
复盘的目的是系统性改进，而非自我惩罚。过度的自我批评会导致过度保守或情绪化决策。复盘的语气应该是"下次如何做得更好"，而非"这次我有多差"。

**误区 4：教训过于笼统**
"要更谨慎"、"要更深入研究"——这类教训没有可操作性。好的教训是具体的："未来在监管密集型行业建仓时，需要在备忘录中专门增加一节分析监管风险，概率和影响各打分"。

## 9. 最小可执行练习

**练习 1（结果矩阵分析）**
列出最近 5-10 笔投资，将每笔归入结果矩阵的四个象限（过程正确/错误 × 结果盈/亏）。分布是否符合预期？虚假成功有多少笔？

**练习 2（论点复盘）**
选一笔已经结束的投资，找出当时的买入理由（如果有文字记录更好）。
① 列出当时的 3-5 个关键假设 ② 事后每个假设是否实现 ③ 如果重新回到当时，决策是否一样

**练习 3（系统性模式识别）**
回顾所有亏损的投资，尝试找出共同特征（是否集中在某类行业？某类估值水平？某种市场情绪下买入？某种新闻触发？）。识别出 1-2 个系统性弱点，并设计一个具体的检查步骤来应对。

## 10. 一页总结

**核心区分**：结果（运气+技能）≠ 过程质量（技能）。复盘要评估过程，而非只看结果。

**四层复盘**：结果层（发生了什么）→ 原因层（为何）→ 决策质量层（过程是否正确）→ 系统改进层（如何提升）。

**最危险的情景**：虚假成功（论点错误但赚钱），容易把运气当技能，强化错误模式。

**核心原则**：用当时的信息评估当时的决策；教训要具体可操作；既复盘亏损也复盘盈利。

**长期统计**：追踪论点正确率和真实超额收益来源，区分技能与运气。

| 术语 | 解释 |
|------|------|
| 过程评估 | 独立于结果，评估决策逻辑和信息处理质量 |
| 虚假成功 | 过程错误但偶然赚钱，最危险的需要识别的情景 |
| 结果偏差 | 用最终结果反推决策质量，忽视过程正确性 |
| 自我服务归因 | 盈利归功于技能，亏损归咎于外部的认知偏差 |
| 黑天鹅 | 无法事先预见、影响极大的小概率事件 |
| 系统性错误 | 在多次不同决策中重复出现的同类错误模式 |
