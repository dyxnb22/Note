# 信用风险

## 1. 主题定位

这篇讲的是债券投资中最重要的非利率风险——发行人无法按时、足额偿还利息或本金的风险。

它解决的问题是：如何判断一家公司或政府有没有能力还债？信用评级是什么？信用利差如何定价违约风险？

学它之前最好先懂：债券基础、到期收益率（信用利差与 YTM 的关系）。

学完后可以继续学：利率风险（信用风险 vs. 利率风险的组合）、国债信用债可转债（不同信用主体的差异）、财务造假识别（信用分析的防线）。

---

## 2. 核心概念

**信用风险（Credit Risk）**
债务人（发行人）无法按时、足额履行还款义务的风险。包括：
- 违约风险（Default Risk）：完全无法偿还
- 信用迁徙风险（Migration Risk）：信用评级下调，导致市场价格下跌（即使未违约）
- 回收风险（Recovery Risk）：违约后能收回多少（回收率）

**信用评级（Credit Rating）**
专业评级机构（穆迪、标准普尔、惠誉；国内有中诚信、大公等）对债券发行人信用质量的评估。

主要分类（以标普为例）：
- 投资级（Investment Grade）：AAA、AA、A、BBB
- 投机级/高收益（Speculative Grade / Junk）：BB、B、CCC、CC、C、D

`此部分可能随监管、评级机构规则变化，需以最新规则为准。`

**违约率（Default Rate）**
在特定时期内发生违约的债券占同等级债券总量的比例。历史数据：AAA 级债券 1 年违约率接近 0；B 级债券 1 年违约率约 3-5%；CCC 级可达 20%+。

**回收率（Recovery Rate）**
违约后，债权人最终能收回的金额占面值的比例。高级有担保债的回收率历史上约 60-80%；无担保债约 30-50%；次级债可低于 10%。

`损失 = 违约概率（PD）× 违约敞口（EAD）× 损失率（LGD）`
其中 LGD = 1 - 回收率（Recovery Rate）

## 3. 核心机制

**信用利差如何定价信用风险**

信用利差（Credit Spread）= 债券 YTM - 同期限无风险利率

理论上，信用利差应该等于：
`信用利差 ≈ 违约概率 × 损失率（LGD）`

实践中，信用利差还包含：
- 信用风险溢价（补偿承担不确定性本身，不只是期望损失）
- 流动性溢价（信用债流动性通常低于国债）
- 信息不对称溢价（分析信用质量需要专业能力）

**信用利差的周期性**

信用利差随经济周期大幅波动：
- 经济扩张：企业盈利好，违约风险低，信用利差收窄
- 经济收缩：企业盈利压力大，违约风险上升，信用利差扩大

信用利差扩大（即使还未违约）会导致信用债价格下跌，即使基础利率不变。

**信用分析的两个维度**

**1. 偿债能力（Capacity to Pay）**
- 利息覆盖倍数（EBIT / 利息）：越高越安全
- 债务 / EBITDA：越低越安全
- 自由现金流 / 债务：越高越安全

**2. 偿债意愿（Willingness to Pay）**
- 发行人的历史还款记录
- 管理层的财务策略
- 是否有法律机制强制还款
- 对于主权债务：政府是否愿意向外国债权人偿债（主权风险）

`中国语境提醒：中国信用债市场（特别是城投债/地方政府融资平台债）存在特殊的隐性政府信用背书问题。城投债历史上隐含政府支持，但法律上不是政府直接负债。随监管收紧，刚性兑付预期在改变，需以最新政策为准，不能简单套用历史"保本"经验。`

## 4. 分析框架

**信用分析的五步框架（"CAMELS" 框架简化版）：**

1. **行业地位**：公司在行业中的竞争力，护城河是否支撑长期偿债能力
2. **盈利质量**：历史盈利和现金流是否稳定且充裕
3. **财务杠杆**：债务规模是否合理（Debt/EBITDA、资产负债率）
4. **流动性**：短期内能否偿还到期债务（短债长待风险）
5. **治理结构**：管理层的财务策略和历史还款记录

## 5. 关键指标或关键变量

**信用利差（Credit Spread）**
反映什么：市场对特定发行人违约风险的综合定价。
容易误判：信用利差受流动性和市场情绪影响，可以短期大幅偏离基本面。利差宽 ≠ 违约概率高；利差窄 ≠ 安全。

**债务/EBITDA 倍数**
反映什么：用年化 EBITDA 偿还全部债务需要多少年。通常 < 3 倍较安全，> 5 倍风险较高。
容易误判：不同行业的合理区间差异很大；周期性行业需看穿越周期的 EBITDA，不能只看景气高峰。

**利息覆盖倍数（ICR）**
反映什么：公司盈利偿还利息的能力。通常 > 3 被认为安全，< 1.5 风险较高。
容易误判：EBIT 是会计利润，非现金流。若公司现金流差，即使 ICR 看起来合理也可能有问题。

**评级迁移**
反映什么：信用评级的历史变迁模式，衡量从当前评级下调一级或多级的历史概率。
容易误判：评级机构有滞后性，信用事件发生后评级才下调，不能完全依赖评级作为实时信号。

## 6. 实际例子

**例 1：信用利差的扩大与价格影响**
企业债 YTM = 5%（无风险利率 3%，信用利差 200 bps）。经济衰退来临，市场对该企业违约的担忧上升，信用利差扩大到 400 bps，YTM 升至 7%（假设无风险利率不变）。债券价格下跌幅度 ≈ 修正久期 × 2%。如果久期 = 5，价格约下跌 10%——即使公司实际上没有违约。

**例 2：违约与回收率**
某高收益债发行人破产，债权人最终能收回 45 元（面值 100 元），回收率 45%，损失率 LGD = 55%。若购买时信用利差只有 150 bps（过低），实际预期损失远超利差补偿，是典型的信用风险被低估。

**例 3：中国信用债市场的违约演进**
2014 年超日太阳债打破中国公募债券"零违约"的先例，是里程碑式事件。此后中国信用债市场违约逐渐常态化，市场开始真正定价信用风险，信用分析变得更重要。这说明：依赖"刚性兑付"的历史经验，可能在制度改变后失效。

## 7. 常见误区

**误区 1：高信用评级 = 绝对安全**
评级是历史和当前基本面的综合评估，不代表未来一定安全。评级机构也会犯错（2008 年金融危机中大量 AAA 级 MBS 实际违约）。评级是参考而非最终答案。

**误区 2：只要没有违约，信用债就是安全的**
即使不违约，信用利差扩大也会导致债券市场价值下跌（信用迁徙风险）。对于不持有至到期的投资者，这个市值波动同样是真实风险。

**误区 3：城投债有政府隐性担保所以没有风险**
历史上确实存在政府对城投债的隐性支持，但随着"打破刚兑"政策推进，这个假设已经不再可靠。需要基于自身基本面分析，而不是依赖隐性担保。

**误区 4：高收益债的高收益是免费的**
高收益债的高 YTM 是对应高违约概率和高损失率的风险补偿。如果违约率超出预期，实际总回报可能大幅为负。历史上高收益债在衰退期经历的损失可以超过其在扩张期积累的利差收益。

## 9. 最小可执行练习

**练习 1（信用分析）**
选择一家你知道曾经发行过债券的公司（或城投平台），查找其信用评级和当前信用利差。从五步分析框架出发，评估其信用质量。

**练习 2（损失计算）**
某投资者持有某公司债，面值 100 万，以 95 元买入（现值 95 万）。该公司违约，回收率 40%（收回 40 万元）。计算：① 违约前信用利差是 150 bps（年化），持有 2 年共收到息票多少？② 最终总损失多少？

**练习 3（信用利差分析）**
查找中国当前 5 年期 AAA 评级企业债、AA 评级城投债、AA 评级产业债的信用利差（相对国债）。三者信用利差的差异说明什么？哪个性价比可能更高？

**练习 4（评级迁移）**
某企业债原评级 AA，财务状况持续恶化，被下调至 A-，信用利差从 100 bps 扩大到 250 bps。若债券修正久期 = 4，价格变化约为多少？这是"信用迁徙风险"的典型体现，写出完整逻辑链。

## 10. 一页总结

**信用风险三要素**：违约概率（PD）+ 违约时损失敞口（EAD）+ 违约损失率（LGD = 1 - 回收率）。

**信用利差**：YTM 超过无风险利率的部分，理论上补偿 PD × LGD，实践中还含流动性溢价和风险溢价。

**信用分析五步**：行业地位 → 盈利质量 → 财务杠杆 → 流动性 → 治理结构。

**信用风险的双重来源**：违约（本金损失）+ 信用利差扩大（市值损失，即使未违约）。

**核心陷阱**：高评级 ≠ 绝对安全；城投隐性担保不可依赖；高收益利差 ≠ 免费收益；利差窄期买入高收益债，是在最坏时机积累最大风险。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| 信用风险 | 发行人无法按时足额还款的风险 |
| 违约率（PD）| 在特定时期内发生违约的概率 |
| 回收率 | 违约后债权人能收回的金额占面值的比例 |
| LGD（违约损失率）| 1 - 回收率，发生违约时的净损失比例 |
| 信用评级 | 评级机构对发行人信用质量的综合评估 |
| 投资级 | BBB 及以上评级，被机构投资者广泛接受 |
| 高收益债（垃圾债）| BB 及以下评级，信用风险高，收益率高 |
| 信用迁徙风险 | 评级下调导致债券市场价值下降的风险 |
| 城投债 | 中国地方政府融资平台发行的债券，历史上存在隐性政府信用 |
| 刚性兑付 | 金融产品到期不论盈亏必须按预期返本付息的非正式保证（政策在打破）|
