# 到期收益率

## 1. 主题定位

这篇深入讲解到期收益率（YTM）的含义、计算逻辑、与其他收益率指标的比较，以及 YTM 作为债券定价基准的重要性和局限。

它解决的问题是：YTM 是什么，为什么它是"债券的价格"？它的假设是什么？什么情况下 YTM 会误导投资者？

学它之前最好先懂：时间价值与贴现、债券基础。

学完后可以继续学：久期与凸性（利率敏感度如何用 YTM 来计算）、利率期限结构（不同期限的 YTM 关系）。

---

## 2. 核心概念

**到期收益率（YTM，Yield to Maturity）**
使债券所有未来现金流（息票 + 到期本金）的现值之和等于当前市场价格的折现率。

直觉：YTM 是"如果你现在以市场价格买入这只债券，并且持有到期，且所有息票都以相同利率再投资，你实际获得的年化回报率"。

规范定义：YTM 是债券的内部收益率（IRR）。

**YTM 与债券价格的关系**

`P = Σ C/(1+y)^t + F/(1+y)^n`

给定价格 P，反解出 y（YTM）。这是一个方程，通常需要试错法或数值计算。

**简便公式（近似）**

```
YTM ≈ (C + (F-P)/n) / ((F+P)/2)
```
其中 C = 年息票，F = 面值，P = 当前价格，n = 期限（年）

---

## 3. 核心机制

**YTM 的三个组成部分**

1. 无风险利率（Risk-Free Rate）：等待时间的补偿
2. 期限溢价（Term Premium）：锁定资金更长时间的风险补偿
3. 信用溢价（Credit Premium）：承担违约风险的补偿

`YTM = 无风险利率 + 期限溢价 + 信用溢价`

对于同期限的国债（无信用风险），YTM = 无风险利率 + 期限溢价。

**YTM 的两个假设（也是两个局限）**

假设 1：持有至到期（不中途卖出）
如果在到期前卖出，实际回报 = 持有期回报（Holding Period Return），可能与 YTM 差异很大。

假设 2：息票可以按同一 YTM 再投资（再投资风险）
实际上，未来市场利率会变化，再投资的实际利率可能高于或低于 YTM。如果市场利率下降，再投资收益会低于预期，实际回报低于 YTM。

**YTM 与票面利率、当前收益率的比较**

| 指标 | 公式 | 反映 | 局限 |
|------|------|------|------|
| 票面利率 | 息票 / 面值 | 基于面值的名义利率 | 不考虑价格偏离 |
| 当前收益率 | 息票 / 市价 | 当前价格下的年化息票收益 | 不含资本利得 |
| YTM | 内部收益率 | 持有至到期的全部收益 | 再投资假设、需持有至期 |

折价债券：票面利率 < 当前收益率 < YTM（价格折扣带来资本利得）
溢价债券：票面利率 > 当前收益率 > YTM（价格溢价带来资本损失）
平价债券：三者相等

---

## 4. 分析框架

**使用 YTM 的正确场景：**

1. 比较相同信用风险、相同期限的债券（YTM 越高，相对价值越好）
2. 衡量债券市场整体利率水平（10 年期国债 YTM 是最重要的参考指标）
3. 计算久期（以 YTM 为折现率）

**不应直接用 YTM 的场景：**

1. 跨期限比较（YTM 含有不同期限溢价，不可直接比较）
2. 跨信用等级比较（YTM 差异可能是信用风险差异，不是价值差异）
3. 预期持有期远短于到期期限（持有期收益率更相关）

---

## 5. 关键指标或关键变量

**实际 YTM vs. 预期 YTM**
反映什么：实际 YTM 是市场当前的收益率；你预测未来市场 YTM 方向，判断价格走势。
容易误判：YTM 上升意味着债券价格下跌（两者反向），所以"YTM 很高是好事"——是相对于未来的机会，不代表当前持有者不亏损。

**YTM 的期限利差**
反映什么：不同期限国债的 YTM 差值，是利率期限结构的量化表达。
容易误判：期限利差收窄（平坦化）不一定代表长端跌，也可能是短端升（加息周期）。需分解短端和长端各自的驱动。

## 6. 实际例子

**例 1：YTM 计算（近似）**
面值 100，票面利率 5%（年息票 5 元），5 年期，当前价格 95 元。

近似 YTM ≈ (5 + (100-95)/5) / ((100+95)/2) = (5+1) / 97.5 ≈ 6.15%

解读：你以折价 95 买入，一方面收到每年 5 元息票，另一方面 5 年后收回 100 元面值（比你买入价多 5 元），综合年化约 6.15%。

**例 2：再投资风险的影响**
你买入一只 10 年期债券，YTM = 5%，意味着假设每年息票都能以 5% 再投资。但 3 年后市场利率降到 2%，剩余 7 年的息票只能以 2% 再投资。最终实际总回报将低于 5%——再投资风险实现了。

---

## 7. 常见误区

**误区 1：YTM 是"保证回报"**
YTM 只是一个计算数字，它假设：持有至到期 + 息票按同利率再投资。两个假设在现实中都很难完全满足，实际回报与 YTM 会有偏差。

**误区 2：不同期限的 YTM 可以直接比较**
10 年期国债 YTM = 4%，3 个月国债 YTM = 5%，不代表 3 个月国债更好——期限不同导致风险和流动性不同，YTM 必须在相同期限内比较才有意义。

**误区 3：YTM 高的债券一定比 YTM 低的好**
YTM 差异可能完全来自信用风险差异。高收益垃圾债的 YTM 可能远高于国债，但一旦发生违约，实际回报可能大幅为负。

## 9. 最小可执行练习

**练习 1（计算）**
面值 100，票面利率 6%，3 年期，当前价格 102 元。用近似公式计算 YTM，判断该债券是溢价、折价还是平价，验证 YTM 与票面利率的大小关系。

**练习 2（关系验证）**
对于溢价债券，证明：票面利率 > 当前收益率 > YTM。
对于折价债券，证明：票面利率 < 当前收益率 < YTM。
用具体数字举例验证。

**练习 3（再投资风险）**
你以 YTM = 8% 买入 5 年期债券（面值 100，票面利率 8%，平价买入）。如果未来每年息票只能以 4% 再投资，计算实际总回报（5 年后总收到多少钱，换算成年化回报）与 YTM 的差距。

**练习 4（信用利差）**
国债 YTM = 3%，企业 A 债 YTM = 5%，企业 B 债 YTM = 8%（三者同期限）。分别计算信用利差。市场认为 B 的信用风险是 A 的多少倍？这个倍数合理吗？

## 10. 一页总结

**YTM 的本质**：使债券未来所有现金流现值等于当前市价的折现率，是债券的"内部收益率"，是最重要的债券收益指标。

**三种收益率的关系**：
- 折价：票面利率 < 当前收益率 < YTM
- 溢价：票面利率 > 当前收益率 > YTM
- 平价：三者相等

**YTM 的两个假设**：持有至到期 + 息票按相同利率再投资（现实中通常不完全满足）。

**使用边界**：只在相同期限、相同信用级别的债券间比较 YTM 才有意义。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| YTM（到期收益率）| 持有至到期的年化内部收益率，最重要的债券收益指标 |
| 票面利率 | 基于面值的年利率，决定息票金额 |
| 当前收益率 | 年息票 / 当前市价，不含资本利得 |
| 内部收益率（IRR）| 使净现值为零的折现率，YTM 是债券的 IRR |
| 再投资风险 | 未来息票再投资利率不确定带来的收益不确定性 |
| 持有期回报（HPR）| 实际持有期间（非持有至到期）的实际回报率 |
| 信用溢价 | YTM 超出无风险利率的部分，反映违约风险定价 |
