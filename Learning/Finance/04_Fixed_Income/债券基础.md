# 债券基础

## 1. 主题定位

这篇讲的是债券的基本结构：什么是债券、主要要素是什么、价格如何形成、与股票有何本质差异。

它解决的问题是：债券不只是"固定利息的存款"，它是一种定价机制完全基于折现的市场化工具，价格随利率波动。

在整个知识体系中，这是 `04_Fixed_Income` 的入口篇，为后续的到期收益率、久期、信用风险提供基础。

学它之前最好先懂：时间价值与贴现（债券定价的核心工具）、利率机制（利率决定债券价格）。

学完后可以继续学：到期收益率、久期与凸性、信用风险、国债信用债可转债。

---

## 2. 核心概念

**债券（Bond）**
发行人（政府或企业）向投资者借款的凭证。发行人承诺在特定日期支付利息，并在到期日偿还本金。

**票面价值（Face Value / Par Value）**
债券的名义面值，通常为 100 元或 1000 元，是到期日偿还的金额。

**票面利率（Coupon Rate）**
基于面值计算的年利息率。票面利率 × 面值 = 年息票（Coupon Payment）。

**到期日（Maturity Date）**
本金到期偿还的日期。按期限分：短期（< 1 年）、中期（1-10 年）、长期（> 10 年）。

**当前收益率（Current Yield）**
= 年息票 / 当前价格。反映当前价格下的即期收益，不含资本利得/损失。

**到期收益率（Yield to Maturity, YTM）**
持有至到期获得的年化内部收益率，包含所有现金流（息票 + 本金）的折现。是最重要的债券收益指标。

## 3. 核心机制

**债券的定价原理**

债券价格 = 所有未来现金流的现值之和：

```
P = C/(1+y)¹ + C/(1+y)² + ... + (C+F)/(1+y)ⁿ
```
其中 C = 息票，F = 面值，y = 到期收益率，n = 剩余期数。

**债券价格与利率（YTM）的反向关系**

这是债券分析最核心的事实：
- 市场利率上升 → 折现率上升 → 债券现值下降 → 债券价格下跌
- 市场利率下降 → 折现率下降 → 债券现值上升 → 债券价格上涨

直觉：如果旧债券票面利率 3%，市场新债券利率 5%，旧债券必须价格下跌，使其实际收益率也达到 5%，才能吸引投资者。

**溢价、折价、平价**

- 溢价（Premium）：票面利率 > 市场利率 → 价格 > 面值（市价高于 100）
- 折价（Discount）：票面利率 < 市场利率 → 价格 < 面值（市价低于 100）
- 平价（Par）：票面利率 = 市场利率 → 价格 = 面值

随着到期日临近，债券价格趋向面值（"拉向平价效应"），无论之前是溢价还是折价。

**债券与股票的核心区别**

| 维度 | 债券 | 股票 |
|------|------|------|
| 索偿顺序 | 优先于股权 | 最后 |
| 现金流确定性 | 高（固定息票 + 本金）| 低（分红不确定，股价波动）|
| 最大收益 | 有上限（票面利率）| 无上限 |
| 违约风险 | 有，但低于股权 | 公司破产损失全部 |
| 利率敏感性 | 高（价格与利率反向）| 间接影响（折现率）|

## 4. 分析框架

**分析一只债券的关键问题：**

1. 发行人是谁？信用质量如何？（信用风险）
2. 到期收益率是多少？相对于无风险利率的利差是多少？（收益判断）
3. 期限多长？利率风险有多大？（久期风险）
4. 有没有提前赎回条款？（嵌入期权）
5. 当前价格是溢价还是折价？（价格与面值的关系）

---

## 5. 关键指标或关键变量

**到期收益率（YTM）**
反映什么：持有到期的年化总回报，是债券的"综合定价"指标。
容易误判：YTM 假设所有息票可以按同一利率再投资（再投资风险），实际回报可能偏差。

**信用利差（Credit Spread）**
反映什么：债券 YTM 减去同期限无风险利率，是违约风险的市场定价。
容易误判：信用利差会随市场情绪大幅波动，利差扩大不一定意味着实际违约风险上升，也可能是流动性恐慌。

**久期（Duration）**
反映什么：债券价格对利率变化的敏感度（详见"久期与凸性"篇）。
容易误判：久期越长，利率风险越大；短期债券久期短，利率风险小但再投资风险大。

## 6. 实际例子

**例 1：债券定价计算**
面值 100 元，票面利率 5%，3 年到期，市场利率（YTM）= 6%：

```
P = 5/1.06 + 5/1.06² + 105/1.06³
  = 4.72 + 4.45 + 88.20
  = 97.37 元
```

市场利率高于票面利率 → 折价（97.37 < 100）。这是债券价格与利率反向关系的直观验证。

**例 2：利率上升对债券组合的冲击**
某基金持有 10 年期国债，均价 100 元，YTM = 3%。市场利率突然上升 1%，债券 YTM 变为 4%。价格将下跌约 8-9%（10 年期债券久期约 8-9 年）。这解释了为什么 2022 年利率快速上升期，长期债券基金净值大幅下跌。

## 7. 常见误区

**误区 1：债券"保本"，一定安全**
债券不保本。信用债可能违约（本金损失）；即使不违约，价格也随利率波动（市场价值损失）。只有持有至到期且不违约，才能收回面值本金。

**误区 2：票面利率 = 投资回报率**
票面利率是基于面值计算的，如果你以溢价（超过面值）买入，实际到期收益率（YTM）低于票面利率；折价买入则高于票面利率。判断回报用 YTM，不用票面利率。

**误区 3：债券利率越高越好**
高利率往往对应高信用风险（高收益债券）。在收益率差异能被违约风险吃掉的情况下，选择了高收益债是承担了未被充分补偿的风险。

## 9. 最小可执行练习

**练习 1（价格计算）**
面值 100 元，票面利率 4%，2 年到期，市场 YTM = 6%。计算债券价格，判断是溢价还是折价。

**练习 2（价格与利率关系）**
同一只 10 年期国债（面值 100，票面利率 3%），分别在 YTM = 2%、3%、4%、5% 时，价格是多少？制作一个价格-收益率对照表，观察非线性关系。

**练习 3（与股票比较）**
某公司同时有股票和债券在市场交易。如果公司发生财务困难，哪个先受损？哪个先受益（如果公司好转）？解释索偿顺序的含义。

**练习 4（信用利差解读）**
10 年期国债 YTM = 3%，某企业债 YTM = 6%，信用利差 = 3%。这意味着什么？如果信用利差突然扩大到 5%，企业债价格如何变化？

## 10. 一页总结

**债券要素**：发行人 + 面值 + 票面利率 + 到期日。承诺固定现金流（息票 + 到期本金）。

**核心定价**：所有现金流按 YTM 折现。利率上升 → 价格下跌；利率下降 → 价格上涨。反向关系是债券分析的核心。

**三种价格状态**：溢价（票面利率 > YTM）、折价（票面利率 < YTM）、平价（相等）。随到期日临近，价格趋向面值。

**核心陷阱**：债券不保本（信用风险 + 市场价值波动）；票面利率 ≠ 实际回报（用 YTM）；高收益率 = 高风险，不是免费的午餐。
