# Learning

学习内容现在分成 10 类：

- `CS`：算法、数据结构、网络、操作系统
- `Backend`：再分成 `Java / Data / Architecture / Delivery`
- `Python Agent`：再分成 `Python / Agent`
- `AI`：偏模型理解、研究基础、AI 工具认知
- `Finance`：金融和投资学习
- `History_and_Society`：科技史、经济史、制度演化、组织与社会
- `Writing_and_Expression`：结构化写作、技术表达、汇报与说服
- `Health_and_Energy`：睡眠、运动、饮食、专注、压力管理
- `Web3`：区块链、智能合约、DeFi、安全、治理与研究流程
- `TypeScript`：类型系统、Node、Web3 与 AI 应用的工程实践

这样分的原则很简单：

- 偏底层原理，放 `CS`
- 偏传统后端实现，放 `Backend`
- 偏 Python 智能体开发，放 `Python Agent`
- 偏模型和理论认知，放 `AI`
- 偏市场和投资，放 `Finance`
- 偏历史背景、制度环境和组织观察，放 `History_and_Society`
- 偏输出能力和表达能力，放 `Writing_and_Expression`
- 偏长期状态管理和精力建设，放 `Health_and_Energy`
- 偏链上系统、协议机制、Token、合约安全和 Web3 工程实践，放 `Web3`
- 偏 TypeScript 类型建模、接口契约和跨端工程实践，放 `TypeScript`
