## 这个方向解决什么问题

- 为什么脑子里明白，写出来却很散
- 为什么做了很多事，却讲不出价值
- 为什么方案本身没问题，但推进不动
- 为什么面试、汇报、复盘时容易“想到哪说到哪”

`表达的底层原则` → `结构化写作` → `技术说明与方案表达`
→ `工作汇报与会议表达` → `论证、说服与异议处理`
