## 统一阅读抓手

README → 构建入口 → 核心链路 → 协议/错误/资源边界 → 测试/发布/许可证。生成类项目额外看模板、数据校验和批量失败；终端/代码执行类项目额外看权限、取消、审计和沙箱。
