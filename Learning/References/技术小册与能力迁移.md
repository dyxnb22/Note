# 技术小册与能力迁移

四套小册的价值在于把底层、设计、算法和面试表达连接到工程案例。

| 小册方向 | 核心内容 | 工程连接 |
|---|---|---|
| 字节码编程 | ASM、Javassist、Byte Buddy、Java Agent | 监控、插件、非侵入式增强和运行时诊断 |
| Java 设计模式 | 原则、创建/结构/行为模式 | 规则链、策略装配、组件扩展和领域服务 |
| Java 面经 | 数据结构、算法、并发和回答组织 | 面试表达、缓存、调度、路由和并发控制 |
| 数据结构与算法 | 线性结构、树、图、排序、查找 | 查询、索引、缓存、检索和任务调度 |

## 学习方法

每个知识点都落到一个案例：说明它解决的问题、替代方案、复杂度、边界和验证方式。设计模式不是类名清单，算法也不是脱离数据规模的题解。
