# 08 研究方法与论文复现

如果目标是申请全奖 PhD，仅仅“会用 LLM”不够。你需要证明自己有研究潜力：能读论文、找问题、复现实验、做 ablation、写出清楚的 technical report。

## 你需要补的能力

### 1. 研究阅读能力

读论文不要只看结论，要按下面结构拆：

- Problem：作者解决什么问题？
- Motivation：为什么这个问题重要？
- Method：方法真正的新点是什么？
- Experiment：实验是否支撑结论？
- Baseline：对比对象是否公平？
- Limitation：哪些地方还没解决？
- Extension：你能怎么改进？

### 2. 复现能力

复现不是只跑官方代码，而是：

- 用自己的话重写方法。
- 复现核心指标或趋势。
- 做 1-3 个 ablation。
- 解释失败原因。
- 写复现报告。

### 3. 研究问题意识

好问题通常来自：

- 现有方法太贵。
- 评测不可靠。
- 数据污染或偏差。
- 长上下文效率差。
- Agent 成本、安全、鲁棒性不足。
- 推理模型的 CoT 可监控性、长度和真实性问题。

## 阶段产出

至少完成：

- 10 篇论文精读笔记。
- 3 篇论文复现报告。
- 1 个可公开 GitHub repo。
- 1 篇 4-6 页英文 mini paper。
- 1 个明确 PhD research statement 方向。

## 项目说明

`project/paper_review_generator.py` 会生成标准论文阅读模板，帮助你把每篇论文读成“可研究”的形式，而不是读完就忘。

运行：

```bash
cd 08_research_methods_and_reproduction/project
python3 paper_review_generator.py "Attention Is All You Need"
```
