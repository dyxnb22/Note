# Report Template

Use this template for each module report.

## Title

Module name and experiment focus.

## Objective

What question was the module trying to answer?

## References

Link to the external reference chapters or papers used for study.

## Implementation Scope

- What was reproduced?
- What was implemented from scratch?
- What was intentionally skipped?

## Experimental Setup

- Dataset
- Model or baseline
- Metrics
- Key hyperparameters

## Results

- Main observations
- Curves, tables, or visualizations
- Failure cases

## Ablations and Diagnostics

- What changed?
- What stayed stable?
- What was surprising?

## Takeaways

- What is now understood better?
- What remains confusing?
- What should be tried next?
