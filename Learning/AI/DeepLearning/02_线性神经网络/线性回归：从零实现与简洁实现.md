# 线性回归：从零实现与简洁实现

## 模型

线性回归假设 `ŷ = Xw + b`，用于预测连续值。以平方损失 `L = (ŷ-y)^2 / 2` 衡量误差，用小批量 SGD 更新参数。

## 从零实现要理解的四件事

1. 用已知 `w,b` 合成带噪声数据，确认模型有可学习规律。
2. 初始化可训练参数，并使其 `requires_grad=True`。
3. 依次完成前向计算、loss、`backward`、SGD 更新和清空梯度。
4. 比较学到的参数与真实参数，而不是只看 loss。

```python
def linreg(X, w, b): return X @ w + b
def squared_loss(y_hat, y): return (y_hat - y.reshape(y_hat.shape)) ** 2 / 2
```

## 简洁实现

`nn.Linear` 封装参数，`nn.MSELoss` 封装损失，`torch.optim.SGD` 封装更新。简洁实现不替代从零实现：前者用于工作效率，后者用于知道出错点。

## 诊断

loss 不降：先检查标签 shape、学习率和是否调用 `step()`。训练 loss 很低但验证差：检查特征泄漏、数据分布与正则化。线性模型学不好也可能是问题本身非线性，不应立刻无脑加层。

## 最小练习

生成 `y=2x1-3x2+4+噪声` 的数据，分别从零和用 `nn.Linear` 训练；写下二者参数和误差的差别。
