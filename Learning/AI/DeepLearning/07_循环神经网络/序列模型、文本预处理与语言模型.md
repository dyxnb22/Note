# 序列模型、文本预处理与语言模型

序列数据有顺序：下一项依赖历史。语言模型估计 `p(x_t | x_<t)`，可按概率逐 token 生成文本。文本预处理包括规范化、分词/子词切分、词表、编码、批量化与掩码。

固定词表会产生未知词；子词方法在词表大小与泛化间折中，也是现代 tokenization 的前身。训练语言模型时，输入通常是前 t 个 token，标签是右移一位后的 token；评估常用 perplexity，越低代表平均预测概率越高。

最小练习：将一段文本构造成 `(X, Y)` 的下一个字符预测样本，验证 Y 相对 X 右移一位。
