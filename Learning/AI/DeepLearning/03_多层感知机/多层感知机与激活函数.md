# 多层感知机与激活函数

## 为什么需要隐藏层

线性层无论叠多少层仍等价于一个线性变换。MLP 在层间加入非线性激活，才能表示 XOR 等非线性关系：`H=σ(XW1+b1)`，`Y=HW2+b2`。

## 常用激活函数

- ReLU：`max(0,x)`，简单高效，负半轴可能“死亡”。
- Sigmoid：输出在 0 到 1，适合二分类概率输出；深层隐藏层易梯度饱和。
- Tanh：零中心但同样可能饱和。
- GELU：Transformer 常用，平滑地保留部分负值。

隐藏层通常选 ReLU/GELU；输出层由任务决定：回归常不加激活，多分类输出 logits，二分类可输出一个 logit 并用 `BCEWithLogitsLoss`。

## 容量与归纳偏置

更宽/更深的网络表达能力更强，也更易记住噪声。训练集高分不是目标，泛化到新数据才是目标。先用小模型形成基线，再增加容量。

## 最小练习

用两层 MLP 分类 Fashion-MNIST，分别尝试 ReLU 与 sigmoid，比较训练速度和验证准确率。
