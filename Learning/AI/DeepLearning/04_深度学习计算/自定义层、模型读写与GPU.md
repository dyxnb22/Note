# 自定义层、模型读写与 GPU

自定义层适用于框架没有的变换；只要继承 `nn.Module` 并实现 `forward`。模型保存优先保存 `state_dict()`，加载时先创建同结构模型再 `load_state_dict()`；同时保存配置、标签映射和预处理方式，权重本身不足以复现服务。

GPU 的规则简单：模型、输入、标签在同一个 device；将数据移动放在训练循环内。显存不足时先减 batch size、图像尺寸或序列长度，再考虑梯度累积与混合精度；不要用不断调用 `empty_cache()` 代替定位泄漏。

最小练习：训练一个小模型，保存/重新加载后验证同一输入输出一致；故意让输入留在 CPU，观察并解释 device 错误。
