# 从 D2L 到 LLM：哪些概念仍然有效

D2L 的 token/embedding、交叉熵、teacher forcing、自注意力、位置编码、Transformer、Adam、预训练与微调，都是理解 LLM 的直接基础。规模变化没有取消这些机制，只改变了数据、算力、工程与能力边界。

需要额外补充的现代话题包括 tokenizer 细节、decoder-only、KV cache、指令微调、偏好优化、上下文工程、RAG、工具调用和评测。它们分别在 `Learning/AI/LLM基础.md` 与 `Learning/Python Agent/Agent/` 中继续学习。

建议在完成 Transformer 后，自己画一张“D2L seq2seq → decoder-only LLM → Agent”概念迁移图。
