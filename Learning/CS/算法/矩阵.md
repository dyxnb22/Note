**LeetCode 54. 螺旋矩阵（顺时针螺旋顺序** ，返回矩阵中的所有元素，输出一个数组**）**
```java
    public List<Integer> spiralOrder(int[][] matrix) {
        if (matrix.length == 0)
            return new ArrayList<Integer>();
        int l = 0, r = matrix[0].length - 1, t = 0, b = matrix.length - 1, x = 0;
        Integer[] res = new Integer[(r + 1) * (b + 1)];
        while (true) {
            for (int i = l; i <= r; i++) res[x++] = matrix[t][i]; // left to right
            if (++t > b) break;
            for (int i = t; i <= b; i++) res[x++] = matrix[i][r]; // top to bottom
            if (l > --r) break;
            for (int i = r; i >= l; i--) res[x++] = matrix[b][i]; // right to left
            if (t > --b) break;
            for (int i = b; i >= t; i--) res[x++] = matrix[i][l]; // bottom to top
            if (++l > r) break;
        }
        return Arrays.asList(res);
    }
```

# LeetCode **59. 螺旋矩阵 II** {toggle="true"}
	给你一个正整数 `n` ，生成一个包含 `1` 到 `n^2` 所有元素，且元素按顺时针顺序螺旋排列的 `n x n` 正方形矩阵 `matrix` 。
	```java
public static int[][] generateMatrix(int n) {
    int startx = 0;
    int starty = 0;
    int count = n/2;
    int[][] matrix = new int[n][n];
    int num = 1;
    while(count > 0) {
        for(int i = startx; i < n-startx-1; i++) {
            matrix[starty][i] = num++;
        }

        for (int i = starty; i < n-starty-1; i++) {
            matrix[i][n-startx-1] = num++;
        }

        for (int i = n-startx-1; i > startx; i--) {
            matrix[n-starty-1][i] = num++;
        }

        for (int i = n- starty-1; i>starty; i--) {
            matrix[i][startx] = num++;
        }
        startx ++;
        starty ++;
        count --;
    }

    if (n%2 != 0) {
        matrix[n/2][n/2] = num;
    }
    return matrix;
}
	```
# LeetCode **48. 旋转图像** {toggle="true"}
	给定一个 *n *× *n* 的二维矩阵 `matrix` 表示一个图像。请你将图像顺时针旋转 90 度。
	你必须在[** 原地**](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95) 旋转图像，这意味着你需要直接修改输入的二维矩阵。**请不要 **使用另一个矩阵来旋转图像。
	```java
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        for (int i = 0; i < n / 2; i++) {
            for (int j = 0; j < (n + 1) / 2; j++) {
                int tmp = matrix[i][j];
                matrix[i][j] = matrix[n - 1 - j][i];
                matrix[n - 1 - j][i] = matrix[n - 1 - i][n - 1 - j];
                matrix[n - 1 - i][n - 1 - j] = matrix[j][n - 1 - i];
                matrix[j][n - 1 - i] = tmp;
            }
        }
    }
	```

# 区间和(前缀和) {toggle="true"}
	给定一个整数数组 $`Array`$，你需要频繁地查询该数组中某个指定区间 \[a, b\] 内所有元素的总和。<br>**【输入描述】**<br>1. 第一行输入一个整数 n，表示数组的长度。<br>2. 第二行输入 n 个整数，表示数组的具体数值。<br>3. 接下来的若干行（直到输入结束），每行包含两个整数 a 和 b ($`0 \le a \le b < n`$)，代表你要求和的起始下标和结束下标。
	```java
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] nums = new int[n];
        for(int i = 0; i < n; i++) {
            nums[i] = sc.nextInt();
        }
        int[] record = new int[n];
        int tmp = 0;
        for (int i = 0; i < n; i++) {
            tmp += nums[i];
            record[i] = tmp;
        }
        while(sc.hasNext()) {
            int a = sc.nextInt();
            int b = sc.nextInt();
            int result = a == 0 ? record[b] : record[b] - record[a - 1];
            System.out.println(result);
        }
    }
}
	```
# 开发商购买土地 {toggle="true"}
	**【题目背景】**<br>在一个城市中，有一块矩形的土地，被划分成了 $`n \times m`$ 个小正方形区域。每个区域内都有一定的价值。现在有一位开发商想要购买这块土地，但要求是：**只能水平或垂直地切一刀**，将土地分成两个矩形部分。<br>**【任务】**<br>开发商希望这两块土地的**总价值之差的绝对值最小**。请你计算出这个最小的差值。<br>**【输入描述】**<br>1. 第一行输入两个正整数 n 和 m，表示矩阵的行数和列数。<br>2. 接下来 n 行，每行包含 m 个整数，表示每个区域的土地价值。<br>**【输出描述】**<br>输出一个整数，表示切割后两块土地价值差的最小绝对值。
	```java
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int m = sc.nextInt();
        int[][] matrix = new int[n][m];
        int sum = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                matrix[i][j] = sc.nextInt();
                sum += matrix[i][j];
            }
        }
        int result = Integer.MAX_VALUE;
        int count = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j ++) {
                count += matrix[i][j];
                if (j == m - 1) {
                    // 两者的差值绝对值
                    result = Math.min(result, Math.abs(sum - 2 * count));
                }
            }
        }

        count = 0;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                count += matrix[j][i];
                if (j == n-1) {
                    result = Math.min(result, Math.abs(sum - 2 * count));
                }
            }
        }
        System.out.println(result);
    }
}
	```
