**LeetCode 136. 只出现一次的数字 **除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。异或（XOR）逻辑：result ^= num;

1. **恒等律：** $`a \oplus 0 = a`$（任何数与 0 异或，结果仍是它本身）。
2. **归零律：** $`a \oplus a = 0`$（任何数与自身异或，结果为 0）。

**LeetCode 31. 下一个排列** 指其整数的下一个字典序更大的排列 [1,2,3]、[1,3,2]、[3,1,2]、[2,3,1]

要让一个序列在字典序上“变大一点点”，我们需要：

1.  **改变尽可能靠右的数字**（因为左边的位权更高）。
2.  **用一个稍大一点的数替换掉当前的数**。
3.  **让替换后的右侧部分尽可能小**（即变成升序）

第一步：从右向左找到第一个小于右侧相邻数字的数 nums[i]。如果找到了，进入第二步；否则跳过第二步，反转整个数组

第二步：从右向左找到 nums[i] 右边最小的大于 nums[i] 的数 nums[j]

第三步：反转 [i+1, n-1]（如果上面跳过第二步，此时 i = -1）

**LeetCode 287. 寻找重复数** 给定一个包含 n + 1 个整数的数组 nums ，其数字都在 [1, n] 范围内（包括 1 和 n），可知至少存在一个重复的整数。 把数组的值当成下一个节点的下标，重复数字会形成环；先用快慢指针找到环内相遇点，再从起点和相遇点同步移动，最终相遇的位置就是重复数字
