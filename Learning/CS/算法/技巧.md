**LeetCode 75. 颜色分类**。红色、白色和蓝色的数组，按照红色、白色、蓝色顺序排列，不使用库内置的 sort 函数

```java
public void sortColors(int[] nums) {
    int n = nums.length;
    int ptr = 0;
    for (int i = 0; i < n; ++i) { // 使用 ptr 指针记录下一个“0”应该存放的位置。
        if (nums[i] == 0) {
            int temp = nums[i];
            nums[i] = nums[ptr];
            nums[ptr] = temp;
            ++ptr;
        }
    }
    for (int i = ptr; i < n; ++i) { // ptr 现在停留在第一个非 0 元素的位置。
        if (nums[i] == 1) {
            int temp = nums[i];
            nums[i] = nums[ptr];
            nums[ptr] = temp;
            ++ptr;
        }
    }
}
```
