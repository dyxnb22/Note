## **LeetCode 136. 只出现一次的数字**
给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。
异或（XOR）逻辑就像一个“开关”。它有几个非常神奇的数学性质，使其成为消除重复项的完美工具：
1. **恒等律：** $`a \oplus 0 = a`$（任何数与 0 异或，结果仍是它本身）。
2. **归零律：** $`a \oplus a = 0`$（任何数与自身异或，结果为 0）。
```java
public int singleNumber(int[] nums) {
    int result = 0;
    for (int num : nums) {
        result ^= num;
    }
    return result;
}
```

## LeetCode **75. 颜色分类**
给定一个包含红色、白色和蓝色、共 `n` 个元素的数组 `nums` ，[**原地**](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)** **对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。
我们使用整数 `0`、 `1` 和 `2` 分别表示红色、白色和蓝色。
必须在不使用库内置的 sort 函数的情况下解决这个问题。
```java
public void sortColors(int[] nums) {
    int n = nums.length;
    int ptr = 0;
    for (int i = 0; i < n; ++i) { // 使用 ptr 指针记录下一个“0”应该存放的位置。
        if (nums[i] == 0) {
            int temp = nums[i];
            nums[i] = nums[ptr];
            nums[ptr] = temp;
            ++ptr;
        }
    }
    for (int i = ptr; i < n; ++i) { // ptr 现在停留在第一个非 0 元素的位置。
        if (nums[i] == 1) {
            int temp = nums[i];
            nums[i] = nums[ptr];
            nums[ptr] = temp;
            ++ptr;
        }
    }
}
```
