# 回溯

**LeetCode 46. 全排列**
不含重复数字的数组，返回其所有可能的全排列。用一个 set 记录当前已遍历的值，全局 list 记录数，`list.size() == n` 时，`result.add(new ArrayList<>(tmp));`

**LeetCode 47. 全排列 II**
可包含重复数字。用一个数组记录下标是否被访问过。
`i > 0 && nums[i] == nums[i - 1] && used[i - 1] == false`
表示同一树层重复选择相同数字。

**LeetCode 78. 子集**
互不相同。返回该数组所有可能的子集（不重复）。遍历中加上坐标信息。

**LeetCode 17. 电话号码的字母组合**
给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。
`String[] numString = {"", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"};`
遍历中加上坐标信息。

**LeetCode 39. 组合总和**
无重复元素的整数数组和一个目标整数 `target`，目标和的组合，同一个数字可以无限制重复被选取。递归时传入 `j` 而不是 `j+1`。

**LeetCode 40. 组合总和 II**
每个数字在每个组合中只能使用一次。
`if (j > i && candidates[j] == candidates[j - 1]) { continue; }`

**LeetCode 216. 组合总和 III**
找出所有相加之和为 `n` 的 `k` 个数的组合，每个数字最多使用一次。
`if (n - i < 0 || tmpRecord.size() == k) { break; }`
