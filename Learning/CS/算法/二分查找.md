**LeetCode 34. 在排序数组中查找元素的第一个和最后一个位置** lowerBound 返回最小的满足 nums[i] >= target 的下标 i  `int start = lowerBound(nums, target);int end = lowerBound(nums, target + 1) - 1;`

~~~ java
if (nums[mid] >= target) {
    right = mid - 1; // 范围缩小到 [left, mid-1]
} else {
    left = mid + 1; // 范围缩小到 [mid+1, right]
}
// 循环结束后 left = right+1
// 此时 nums[left-1] < target 而 nums[left] = nums[right+1] >= target
// 所以 left 就是第一个 >= target 的元素下标
// nums[left-1] = nums[right] < target（left 前一个元素仍小于 target）；
//nums[left] = nums[right+1] >= target（left 所在位置元素大于等于 target）
~~~
