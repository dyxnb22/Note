# 链表

**LeetCode 160. 相交链表** 求两个单链表相交的起始节点。
当两个指针都走完自己的路并转而去走对方的路时，它们会在公共节点 `c` 的起始处相遇，因为此时它们走过的总步数完全相等（`a + c + b = b + c + a`）。`A = A != null ? A.next : headB;`

**LeetCode 206.反转链表。**设置一个空节点pre ，每次先保存 `next`，再让 `cur.next` 指向 `pre`，然后 `pre` 和 `cur` 一起往后走

```java
ListNode next = cur.next; cur.next = pre; pre = cur; cur = next;
```

**LeetCode 142. 环形链表 II** 返回链表开始入环的第一个节点。 快慢指针先判断是否有环；相遇后，一个从头走，一个从相遇点走，再次相遇的位置就是环入口。

**LeetCode 21. 合并两个有序链表** 两个升序链表合并为一个新的 升序 链表。谁小谁当头，剩下的继续递归合并
`l1.next = mergeTwoLists(l1.next, l2);`

**LeetCode 2.两数相加** 给你两个 非空 的链表，数字逆序存储，返回一个表示和的链表。 `ListNode addTwo(ListNode l1, ListNode l2, int carry)` 每一层递归负责加一位：算 `sum`，当前节点存 `sum % 10`，下一层带着 `sum / 10` 继续加

**LeetCode 19. 删除链表的倒数第 N 个结点**。 用两个指针 pre 和 next 拉开 n 个节点的距离。当 next 走到链表末尾时，pre 正好停在「要删除节点的前一个节点」。

**LeetCode 25. K 个一组翻转链表** `pre` 站在当前组前面，`end` 往后数 `k` 个节点找组尾；找到后，`start = pre.next` 保存组头，`next = end.next` 保存下一组头；先断开 `end.next`，再反转 `start` 开头的这一组；反转完成后，`pre.next` 接反转后的新头，`start.next` 接回 `next`；因为 `start` 已经从组头变成组尾，所以最后令 `pre = start，end = pre`，进入下一轮。

**LeetCode 138. 随机链表的复制** 长度为 n 的链表，每个节点包含一个额外增加的随机指针 random ，该指针可以指向链表中的任何节点或空节点。构造这个链表的 深拷贝
先用 Map 建立“原节点 → 新节点”的映射，再遍历原链表，用映射关系补上新节点的 `next` 和 `random`
`map.get(cur).next = map.get(cur.next);map.get(cur).random = map.get(cur.random);`
