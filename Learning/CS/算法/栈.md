# 栈

**LeetCode 20. 有效的括号**
括号是否能一一对应。左括号弹入栈，有括号匹配然后弹出。

**LeetCode 155. 最小栈** 设计一个支持 `push`，`pop`，`top` 操作，并能在常数时间内检索到最小元素的栈
`stack`：保存所有正常入栈的元素 `min_stack`：保存当前出现过的最小值。push: 不管 `x` 是多少，都要放进普通栈，最小栈看情况推入（单调栈）. pop: 然后判断被弹出的元素是否正好等于当前最小值

**LeetCode 394. 字符串解码**  `k[encoded_string]`，表示其中方括号内部的 `encoded_string` 正好重复 `k` 次。然后进行解码。 

res：当前层已经解析出的字符串
multi：当前读取到的重复次数
stack_multi：保存进入每层括号前的重复次数
stack_res：保存进入每层括号前，外层已经解析出的字符串

数字：累积计算 multi
字母：追加到当前 res
[：保存当前 res 和 multi，重置变量，进入下一层
]：弹出外层状态，将当前 res 重复 multi 次，再和外层字符串拼接

**LeetCode 739. 每日温度** 给定一个整数数组，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。
栈里存还没有找到更高温度的下标，并让对应温度保持单调不升。 遇到更高温度时，不断弹出比它低的下标，并用当前下标减去弹出下标。

**LeetCode 84. 柱状图中最大的矩形 **给定 *n* 个非负整数，用来表示柱状图中各个柱子的高度。求在该柱状图中，能够勾勒出来的矩形的最大面积。

left_i[i]: 柱子 i 左侧第一个矮于它的柱子的索引。right_i[i]: 柱子 i 右侧第一个矮于它的柱子的索引。并不是一步步向左回退，而是利用已经计算好的 left_i 进行跳跃 `while (tmp >= 0 && heights[tmp] >= heights[i]) tmp = left_i[tmp];`

**LeetCode 232. 用栈实现队列** 一个栈负责进队，一个栈负责出队。当出队栈为空时，把进队栈中的元素全部倒过去。 

**LeetCode 225. 用队列实现栈** 每次 `push` 时，把新元素放到队首，这样队头永远就是栈顶

**LeetCode 496. 下一个更大元素 I** 遍历 `nums2`，用单调递减栈找到每个元素右边第一个更大的数；如果这个元素属于 `nums1`，就把答案记录到 `res`

**LeetCode 503. 下一个更大元素 II** 循环数组 nums。遍历 2n 次，通过 i % n 模拟循环数组。
