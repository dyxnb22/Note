# 堆

**LeetCode 215. 数组中的第K个最大元素 **`PriorityQueue<Integer> pq = new PriorityQueue<>((a, b) -> a - b);`

**LeetCode 347. 前 K 个高频元素** 用 `HashMap` 统计每个数字出现次数。用一个大小最多为 `k` 的**小顶堆**，保存出现频率最高的 `k` 个数字。`PriorityQueue<int[]> pq = new PriorityQueue<>((pair1, pair2) -> pair1[1] - pair2[1]);`

**LeetCode 295. 数据流的中位数** 把所有数字分成左右两半，left ：保存较小的一半，最大堆。right ：保存较大的一半，最小堆。数量相同时，新数先进入右边，再把右边最小值转给左边；左边多一个时，新数先进入左边，再把左边最大值转给右边。
