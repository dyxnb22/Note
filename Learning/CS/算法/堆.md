**LeetCode 347. 前 K 个高频元素** 用 `HashMap` 统计每个数字出现次数。用一个大小最多为 `k` 的**小顶堆**，保存出现频率最高的 `k` 个数字。`PriorityQueue<int[]> pq = new PriorityQueue<>((pair1, pair2) -> pair1[1] - pair2[1]);`

**LeetCode 295. 数据流的中位数** 把所有数字分成左右两半，left ：保存较小的一半，最大堆。right ：保存较大的一半，最小堆

```java
class MedianFinder {
private final PriorityQueue<Integer> left = new PriorityQueue<>((a, b) -> b - a); // 最大堆
private final PriorityQueue<Integer> right = new PriorityQueue<>(); // 最小堆

public void addNum(int num) {
    if (left.size() == right.size()) {
        right.offer(num);
        left.offer(right.poll());
    } else {
        left.offer(num);
        right.offer(left.poll());
    }
}

public double findMedian() {
    if (left.size() > right.size()) {
        return left.peek();
    }
    return (left.peek() + right.peek()) / 2.0;
}
}
```
