# 矩阵

**LeetCode 73. 矩阵置零(如果一个元素为 **0 **，则将其所在行和列的所有元素都设为 **0** **)**
用两个一维数组跟踪行和列 `if (row[i] || col[j])`

**LeetCode 54. 螺旋矩阵（顺时针螺旋顺序** ，返回矩阵中的所有元素，输出一个数组**）**
用 `l、r、t、b` 四个边界框住当前还没遍历的矩阵区域，每走完一条边，就把对应边界向内收缩，直到边界交叉为止（类似 if (l > --r)）

**LeetCode 59. 螺旋矩阵 II**，生成一个包含 `1` 到 `n^2` 的螺旋矩阵。用 `startx` 和 `starty` 表示当前圈左上角的位置，每一圈按照“上、右、下、左”四条边依次填数字。每条边都采用左闭右开的规则，避免角落重复。填完一圈后，起点向右下移动一格。如果 `n` 是奇数，最后单独填中心点

**LeetCode 48. 旋转图像** 顺时针90度。四个位置一组进行轮换，每次选左上区域的一个点，然后找到它旋转后对应的另外三个点，把这四个点按顺时针方向交换

**LeetCode 240. 搜索二维矩阵 II（**每行的元素从左到右升序排列 每列的元素从上到下升序排列）
从右上角开始查，一直向下（更大）或向左（更小）

**区间和(前缀和)** 先用 record[i] 记录从 0 到 i 的前缀和，然后每次求 [a, b] 的区间和时，用 record[b] - record[a - 1] 快速得到结果。如果 a == 0，直接返回 record[b]

**开发商购买土地** 有一块矩形的土地，被划分成了 $n \times m$ 个小正方形区域。**只能水平或垂直地切一刀**，将土地分成两个矩形部分，希望二者之差的绝对值最小。
先求矩阵总和 `sum`，然后分别枚举每一种横切和竖切方式。每次用 `count` 表示其中一部分的和（每满一行/列 算一遍），另一部分就是 `sum - count`，差值就是 `|sum - 2 * count|`，最后取最小值。
