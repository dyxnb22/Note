**LeetCode 124. 二叉树中的最大路径和** 在二叉树中，一个节点的路径处理有两种情况：

1.  **作为“最高拐点”**：路径从左子树上来，经过当前节点，再下到右子树。这种路径**不能再向上延伸**给父节点了（因为二叉树路径不能有分叉）。
2.  **作为“贡献链”的一部分**：路径从当前节点出发，要么向左走，要么向右走，然后**向上延伸**给父节点，作为父节点路径的一部分。

-   **`ans`** 是在找**全局最优解**（可能是在树中间的某个“V”形）。
-   **`return`** 是在为**上一层递归**提供原材料（只能是单向的一条线）。

~~~java
ans = Math.max(ans, lVal + rVal + node.val); // 两条链拼成路径，尝试以当前节点为最高顶点（拐点），连接左右子树
return Math.max(Math.max(lVal, rVal) + node.val, 0); // 当前子树最大链和（注意这里和 0 取最大值了）
~~~
