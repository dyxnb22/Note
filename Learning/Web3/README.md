# Learning/Web3 知识库总览

## 关于本知识库

本知识库面向长期学习 `Web3`，目标不是追热点，也不是只会连钱包和看币价，而是建立从底层机制到工程实践、从协议理解到风险判断的完整框架。

写作原则：机制优先、系统优先、风险优先、长期有效优先。

参考了你现有 [web3](</Users/diaoyuxuan/web3/README.md>) 这套偏 TypeScript-first 的工程学习路线，但这里会整理成更适合复习、串联和沉淀的知识库结构。
