# 泛型、条件类型与映射类型

泛型让函数/类型在保留具体信息的同时复用，例如 `function first<T>(xs: T[]): T | undefined`。约束 `<T extends {id: string}>` 表示所需最小能力，而不是强迫调用者继承某类。

`keyof` 得到属性键，`typeof` 从值推类型，mapped type 批量变换属性，conditional type 根据关系分支。工具类型 `Partial`、`Pick`、`Omit`、`Record` 很实用，但过度复杂的类型体操会损害可读性。
