# 历史、制度与组织索引

这条主线不以记忆年代为目标，而是训练结构化解释：技术、资源、制度、组织和利益如何共同塑造长期结果。

## 共用分析方法

1. 明确时间、地区和比较对象，不把单一案例当成普遍规律。
2. 区分触发因素、必要条件、放大机制和长期结果。
3. 同时检查技术、资源、制度、组织和利益分配，避免单因解释。
4. 寻找反例：相似冲击为什么在另一地区或时期产生不同结果？
5. 将历史类比用于提出问题，不把类比本身当作预测证据。

主题速查只提供问题框架；涉及具体年份、规模、政策和因果判断时，应回到专题研究与原始资料核验。

## 分析主线

```text
技术与能源改变可行边界
  → 生产率、人口与贸易改变经济结构
  → 制度决定激励、执行和分配
  → 组织处理协作、信息和权力问题
```

## 阅读和判断方法

1. 先写清时间、地区和比较对象。
2. 区分触发事件、必要条件、放大机制和结果。
3. 至少同时检查技术、资源、制度、组织和利益分配。
4. 主动找反例，避免把一个成功案例写成普遍规律。
5. 历史类比只用于提出问题；预测仍需要当前数据和具体约束。
