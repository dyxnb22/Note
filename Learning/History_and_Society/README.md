## 这个方向解决什么问题

- 为什么同样的技术，在不同国家和时代效果完全不同
- 为什么资本市场、企业组织、监管制度会演化成今天的样子
- 为什么很多商业与金融现象，单靠课本模型解释不够
- 为什么个人在组织中会遇到信息不对称、激励错位和路径依赖
